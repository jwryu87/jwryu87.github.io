# ISO 22301
{: .fs-8 }

3.6 ISO 22301
{: .label .label-red }

> 암기: **조리계지/운성개**

---

## 핵심 키워드

`PDCA` `조직` `리더십` `계획수립` `지원` `운영` `성과평가` `개선`

---

## 정의/개념

기업의 비즈니스 연속성을 위해 **BCP 수립**에서 도입, 운영, 검토까지 지속적인 개선에 대한 요구사항을 규정한 **BCMS의 국제표준**

---

## 구성도/구성요소 (PDCA Cycle)

```
                        BCMS의 지속적 개선
                              │
              ┌───────────────┼───────────────┐
              │               │               │
        ┌─────┴─────┐   ┌─────┴─────┐   ┌─────┴─────┐
        │   이해    │   │  계획수립  │   │    이해   │
        │  관계자   │   │  (Plan)   │   │   관계자  │
        └───────────┘   └───────────┘   └───────────┘
              │               │               │
              ▼               ▼               ▼
        ┌───────────┐   ┌───────────┐   ┌───────────┐
        │ 유지 및 개선│   │ 실행 및 운영│   │  관리된   │
        │   (Act)   │   │   (Do)    │   │비즈니스연속│
        └───────────┘   └───────────┘   └───────────┘
              │               │               │
              └───────────────┼───────────────┘
                              │
                    ┌─────────┴─────────┐
                    │  모니터링 및 검토  │
                    │     (Check)      │
                    └──────────────────┘
              │                                 │
        비즈니스                           비즈니스
         연속성                             연속성
         요구사항                           연속성
```

### PDCA 단계

| 단계 | 설명 |
|:-----|:-----|
| **Plan (계획 수립)** | BCM 정책 및 전략 수립, 프로세스 절차 수립 |
| **Do (실행 및 운영)** | 수립된 BCM 정책, 프로세스 및 절차의 이행 및 운영 |
| **Check (모니터링 및 검토)** | BCM 운영결과 검토<br>내부감사 및 경영진 보고, 개선활동의 결정 |
| **Act (유지 및 개선)** | 경영진 검토, BCM 정책에 근거한 교정활동으로 BCM 유지관리 및 개선 |

---

## 주요 내용

| 구분 | 설명 |
|:-----|:-----|
| **조직상황** | BCM 관리자를 위한 조직의 역할<br>이해관계자와 요구사항 설정, BCM 범위 설정 |
| **리더십 (지도층)** | 경영진의 책무, 업무 연속성 방침<br>최고경영진 책임 및 권한 |
| **계획수립 (기획)** | 업무 연속성 목적 제시, BCM 관련 계획 수립<br>위험요소 식별, 위험평가, 영향분석, 재난위험요소에 대한 전략계획, 경감계획, 대응 계획 |
| **지원** | 자원관리, 수행능력 구축, 내/외부와의 커뮤니케이션 |
| **운영 (운용)** | 운영계획수립 및 통제관리, 업무영향분석 및 위험평가<br>복구 및 모의훈련을 통한 검증, 복구목표시간(RTO) 기준 |
| **성과관리 (성과평가)** | 감시, 측정, 분석 및 평가, 내부감사, 관리자 검토 |
| **개선** | 부적합 및 시정조치, 지속적 개선 |
