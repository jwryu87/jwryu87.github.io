# RA (Risk Assessment)
{: .fs-8 }

3.4 RA
{: .label .label-red }

> 암기: **식분우추**

---

## 핵심 키워드

`위험 식별` `위험 분석` `위험 우선 순위` `위험 추정`

---

## 정의/개념

**리스크 식별, 분석, 평가** 활동

---

## 주요요소/프로세스

| 구분 | 설명 |
|:-----|:-----|
| **정의** | 리스크를 식별, 분석 및 평가하고 처리 방법을 도출하는 활동 |

### 주요요소

| 주요요소 | 기법 |
|:---------|:-----|
| **위험 식별** | Check lists, Decision Driver Analysis, Assumption Analysis, Decomposition |
| **위험 분석** | Performance Model, Cost Model, Network Analysis, Decision Analysis, Quality-factor Analysis |
| **위험 우선순위** | Risk Exposure, Risk Leverage, Compound-risk Reduction |
| **위험 추정** | Risk value = Probability of Event * Cost of Event |

---

## 수행 프로세스

```
위험 구별        위험 측정        우선순위 도출       위험 분석
    │               │                │               │
    ▼               ▼                ▼               ▼
┌────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│계획 수립 │    │ 설문조사  │    │비즈니스   │    │ 위험 분류 │
│        │    │          │    │ 우선순서  │    │          │
│데이터   │ →  │발생 가능성│ →  │          │ →  │위험 카테고리│
│수집 및  │    │  측정    │    │ 위험 중요도│    │ 재 분류  │
│ 분석   │    │          │    │  결론 도출│    │          │
│        │    │ 영향도 측정│    │          │    │ 대응전략  │
│위험 결정 │    │          │    │          │    │          │
└────────┘    └──────────┘    └──────────┘    └──────────┘
        │                                          │
        └───────────────→ 위험 적용 ←───────────────┘
```

> 최종 선정된 리스크에 대한 경감, 예방 및 대응, 복구할 수 있는 방안을 마련해야 함.
