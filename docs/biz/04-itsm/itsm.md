---

## 구성요소

### 가. ITSM 서비스 지원 영역

| 구분 | 구성요소 | 설명 |
|:-----|:---------|:-----|
| **서비스 데스크** | SR(Service Request) | 인시던트, 이슈 해결위한 고객 요청 확인 |
| | VOC(Voice of Customer) | 콜센터, 고객 지원 센터 운영 통한 이슈 최소화 |
| **인시던트 관리** | BCP(Biz. 연속성 계획) | SLA, RTO, RPO 달성 및 체계적 운영 |
| | 인시던트 주요 지표 | 가동률, SR 적기 처리율, 사용자 만족도 |
| **문제 관리** | 문제원인 분석 | 식별/등록, 우선순위부여, 진단/해결, 사후검토 |
| | 문제의 근본원인 해결 | 문제제어, 사전문제관리, 오류제어 |
| **변경 관리** | CAB | 변경관리회의(Change Control Board), PMBOK |
| | CCB | 변경자문위원회(Change Advisory Board), ITIL |
| **릴리즈 관리** | 배포관리 프로세스 | 준비, 긴급배포 여부, 검토/승인/이행, 검증, 통보 |
| | 릴리즈 품질지표 | SW 배포 성공률, 배포장애발생율 |
| **형상관리** | 가시성, 추적성 확보 | Scope Creep, Gold Plating 방지 효과 |
| | 형상관리 프로세스 | 형상식별, 형상통제, 형상감사, 형상기록 절차 수행 |

> 고객에 대한 서비스 지원상 나타날 수 있는 장애, 문제를 최소화하고 근본원인을 제거함으로서 BCP 제공.

### 나. ITSM 서비스 제공 영역

| 구분 | 구성요소 | 설명 |
|:-----|:---------|:-----|
| **용량 관리** | IT 자원 성능 관리 | IT 리소스 모니터링 및 성능 평가 관리 |
| | IT 자원 용량 관리 | 가상화 환경 용량, VM, x86 자원 최적화 |
| **가용성 관리** | 가용수준 유지 관리 | 가용성, 신뢰성, 유지보수성, 보안성 통합 및 관리 |
| | 인프라 지원 조직 | 가용성의 모든 책임요소 모니터링 및 최적화 |
| **연속성 관리** | SLA | 변경관리 통제하에 서비스 수준 계약(SLA) 준수 |
| | 복구계획 | 중요 서비스 우선순위 목록, HW/SW 구성 정보 |
| **서비스 수준 관리** | 서비스 유형 | 기획, 고객관리, 서비스요청, 장애처리, 수정, 유지보수 |
| | 서비스 측정 항목 | 사용자 교육지원, 문서화, 피드백, 수정 요청처리 |
| **재무 관리** | IT 자산 관리 | HW, SW, 인건비, 설비, 외부서비스, 내부비용 |
| | ROI 측정 | IT 서비스 변경에 대한 비즈니스 측면 타당성 제시 |

> 서비스 제공 영역의 경우 **서비스수준관리(SLA)**를 기반으로 서비스 최적화에 대한 요소 다룸.
