## 정의/개념

기업이 고객에게 **가치를 제공**함에 있어서 부가가치 창출에 직/간접적으로 관련된 **주활동, 지원활동, 기능, 프로세스**를 연계하여 분석하는 기법

---

## 메커니즘 (Porter's Value Chain)

### 지원활동 (Support Activities)

| 구분 | 설명 |
|:-----|:-----|
| **기업전반관리** | 기획, 재무, 경리, 법무, 정보시스템 업무 등 |
| **인적자원관리** | 인재채용, 교육, 급여업무 등 |
| **연구개발** | 주활동과 관련된 신제품, 서비스개발, 각종 테스트 등 |
| **구매조달** | 주활동을 지원하는 물건이나 서비스의 구입 등 |

### 본원적활동 (Primary Activities)

| 구분 | 설명 |
|:-----|:-----|
| **물류투입활동** | 원재료나 부품의 구입과 배송 등 |
| **생산운영활동** | 구매한 재료의 조립과 가공 등 |
| **물류산출활동** | 제조한 제품의 창고 또는 소매점 배송 등 |
| **마케팅과 판매** | 제품의 영업, 선전이나 점포에서의 판매활동 등 |
| **서비스** | 판매 후의 문의대응이나 사후 A/S 등 |

---

## 액티비티 구조

### Support Activities

| 활동 영역 | 내용 |
|:----------|:-----|
| **Firm Infrastructure** | 일반관리 / 회계 / 재무 / 법무 / 전략기획 |
| **Human Resource** | 모집 / 훈련 / 배치 / 개발 / 평가 / 보상 |
| **Technology Development** | 기초연구 / 제품설계 / 생산설비 설계 |
| **Procurement** | 원자재·기계·기타 공급품 구입 |

### Primary Activities

| 활동 영역 | 내용 |
|:----------|:-----|
| **Inbound Logistics** | 원재료 구입, 수송, 조립, 재고 보관, 생산현장 출하 |
| **Operation** | 가공, 조립, 제품 재고관리, 생산설비 보수 |
| **Outbound Logistics** | 출하, 광고, 판매예산, 판매채널 관리, 수리 |
| **Marketing & Sales** | 고객관계 관리시스템 |
| **Services** | 제품조립, 시장조사 |
