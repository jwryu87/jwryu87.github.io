# AHP (Analytic Hierarchy Process)
{: .fs-8 }

1.4 AHP
{: .label .label-purple }

---

## 핵심 키워드

`계층` `쌍대비교`

---

## 정의/개념

의사결정과정에 상호 관련성을 갖는 요소를 **계층화**하고 이를 **쌍대비교**해서 도출된 객관적이고 정량적인 결과값을 통해 의사결정 지원기법

---

## 메커니즘

### 다기준 의사결정 기법 AHP 개요

| 구분 | 설명 |
|:-----|:-----|
| **개념** | 의사결정과정에 상호 관련성을 갖는 요소를 계층화하고 이를 쌍대비교에서 도출된 객관적이고 정량적인 결과값을 통해 의사결정을 지원하는 기법 |

### 3원칙

| 원칙 | 설명 |
|:-----|:-----|
| **Hierarchies** | 계층적인 구조의 설정 (복잡한 상황의 구조화) |
| **Priorities** | 대안간 상대적인 중요도의 설정 (비율척도를 통한 가중치 도출) |
| **Consistency** | 논리적인 일관성 유지 (통합 및 논리적 일관성 검증) |

### 장점

- 가능한 모든 비교를 통하여 여러가지 정보를 추출 가능
- 판단의 변화에 따른 전체 우선순위의 일관성 검토 가능
- 유일하게 우선순위의 두가지 측면인 우세와 열세를 반영

---

## AHP 분석 절차

```
1.계층(Hierarchy) → 2.쌍대비교(Pairwise comparison) → 3.부분 우선순위(local priority) → 4.일관성 검사(Consistency check) → 5.최종 우선순위(global priority) 도입
```

| 단계 | 활동 |
|:-----|:-----|
| **1단계** | 계층(Hierarchy) 설정 |
| **2단계** | 쌍대비교(Pairwise comparison) |
| **3단계** | 부분 우선순위(local priority) 계산 |
| **4단계** | 일관성 검사(Consistency check) |
| **5단계** | 최종 우선순위(global priority) 도출 |
