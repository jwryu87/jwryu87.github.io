# 균형성과기록표 (BSC, Balanced Score Card)
{: .fs-8 }

2.7 BSC
{: .label .label-blue }

> 암기: **재고내학**

---

## 핵심 키워드

`재무` `고객` `내부 프로세스` `학습과 성장`

---

## 정의/개념

재무, 고객, 내부 프로세스, 학습과 성장, **KPI**, 비재무적 관점 **성과관리** 기법

---

## 구성도

```
                    ┌────────────┐
                    │   재무관점  │
                    │주주만족을 위해│
                    │어떻게 해야 하나?│
                    └────────────┘
                          │
           ┌──────────────┴──────────────┐
           │                              │
    ┌──────┴─────┐                ┌───────┴──────┐
    │   고객관점   │      비전      │ 내부프로세스 관점 │
    │재무적으로 성공하기 위해│   전략    │프로세스와 서비스의 질을│
    │고객에게 어떻게 보여야 하나?│         │높이기 위하여 어떻게 하여야 하나?│
    └────────────┘                └──────────────┘
                          │
                    ┌─────┴─────┐
                    │학습과 성장 관점│
                    │조직의 역량을 어떻게│
                    │변화시키고 개선 시킬 수 있나?│
                    └───────────┘
```

---

## 구성요소

### BSC 4가지 관점별 KPI

| 관점 | KPI |
|:-----|:----|
| **재무적 관점** | 매출액, 영업이익률 |
| **고객관점** | 고객 만족도, 적시 공급률 |
| **내부 프로세스 관점** | 생산 Cycle Time, 단위원가 |
| **학습과 성장 관점** | 종업원 만족도, 자격증 취득건수 |
