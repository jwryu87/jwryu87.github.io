# LISS (Linearly Independent Spanning Set)
{: .fs-8 }

2.4 LISS
{: .label .label-blue }

> 암기: **파분제가/계분결**

---

## 핵심 키워드

`핵심 산출` `중복 제거`

---

## 정의/개념

문제를 종합했을 때 중복 없이 하위 분석 대상들의 **핵심만을 산출**해내는 전략적 분석 기법

---

## 구성도/구성요소

### LISS 분석

```
┌─────────────────────────────────────────┐
│                LISS 분석                │
│                                         │
│    ┌────┐      ┌───┐      ┌────────┐   │
│    │A(a1)│ →   │ A │  →   │ a1  a2 │   │
│    │A(a2)│     └───┘      └────────┘   │
│    │B(b1)│      ↓         중복없는      │
│    │B(b2)│     ┌───┐     핵심요소도출   │
│    └────┘  →   │ B │  →   │ b1  b2 │   │
│   문제집합      └───┘      └────────┘   │
│                 분석결과                │
│                                         │
│          ← 상하관계 →                   │
└─────────────────────────────────────────┘
```

---

## 절차

| 단계 | 내용 |
|:-----|:-----|
| **문제파악** | 문제의 핵심을 파악하고 분석 방안 도출 |
| **문제분해** | 로직트리를 이용해 문제를 하위요소로 분해 |
| **문제제거** | 중요하지 않은 이슈들을 배제하여 핵심요소도출 |
| **가설** | 이슈에 대한 기승전결 가설을 작성 |
| **계획수립** | 기간, 목표, 범위, 산출물 등의 계획작성 |
| **분석과 종합** | 가설을 검증하여 필요한 결과를 도출 |
| **결과도출** | 결정권자의 입장에서 결과 보고서 작성 |
