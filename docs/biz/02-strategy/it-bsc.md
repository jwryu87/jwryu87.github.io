## 구성요소

### IV. IT BSC의 4가지 관점 상세설명

| 구분 | 세부 요소 | 측정 지표 |
|:-----|:---------|:---------|
| **기업 공헌도 관점** | - IT 투자 비용관리 (단기적 재무성과 평가)<br>- 신규 사업 가치 창출 (장기적 재무성과 평가) | - ROI, NPV, IRR 재무적 평가지표<br>- IT 예산, 인력당 IT비용 |
| **사용자 관점** | - IT 사용자들의 평가(기업 내부/외부 사용자 관점)<br>- 사용자 파트너십 구축, 만족도 평가 | - 제공하는 Application 의 비율<br>- Application 의 사용자 만족도 |
| **운영 프로세스 관점** | - SW 개발 및 프로세스 효율성 측정<br>- 산업 표준 모니터링, 생산성 평가 | - 기능점수(Function Point)<br>- 문제해결 능력, IT 인력관리 |
| **미래 지향적 관점** | - IT 전문 교육 및 훈련<br>- IT 포트폴리오 및 신기술 연구 관리 | - IT 교육 일수, 총 교육비 비율<br>- IT 연구 예산 비율 |

> 효율적인 IT-BSC 수행을 위해서는 기본계획을 명확히 분석하여 IT-BSC KPI를 도출하여 성과분석을 실시
