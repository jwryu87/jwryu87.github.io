## 정량적 평가 기법

### II. IT 투자 평가의 정량적 및 정성적 평가 항목 설명

| 평가 항목 | 설명 |
|:---------|:-----|
| **EVA (Economic Value Added)** | 정보화로 인한 수익 증함 - 정보화에 투입된 자본 비용 총함<br>투자에 대한 장기적 가치 창출 관점에서 기업 성과 측정 |
| **TCO (Total Cost of Ownership)** | 기업의 IT 투자 직접 비용, 간접 비용을 통한 하여 평가<br>운영비, 기술 지원, 유지 보수 등 간접 비용 증가에 따라 도입 |
| **EVS (Economic Value Source)** | 기업 가치 창출 원천을 4가지로 한정 하고 원천별 효과 분석<br>시간과 위험의 가치를 명시적으로 고려, 측정 |
| **TEI (Total Economic Impact)** | 직접적인 비용, 효과 분석 및 미래의 잠재적 이익 추가 평가<br>비용, 이익, 유연성 3가지 핵심 영역 평가 및 위험 판단 |
| **TVO (Total Value of Opportunity)** | IT 조직, 경영진, 재무 조직 간 합의점 제공을 위한 평가 활목<br>비용/수익 분석, 불확실성, 조직 진단, 선진 사례 분석 포함 |
| **ROI (Return of Investment)** | ROI = (IT 총 성과비용 / IT 총 투자비용) = (순성과합게 / 총비용)<br>투자비용: 컨설팅 / 구축 / 유지·보수 등의 구축·운영을 위한 비용의 합 |
| **NPV (Net Present Value)** | $NPV = \sum_{t=0}^{N} \frac{C_t}{(1+r)^t}$<br>N = 사업전체의 기간, t = 현금흐름의 기간, r = 할인율<br>Ct = 시간 t에서의 순현금흐름<br>투자안에서 발생되는 모든 현금흐름 (편익)을 현재 시점 가치로 환산한 값과 비용을 현재 시점의 가치로 환산한 값의 차이 (NPV > 0 이면 회계상 이익) |
| **IRR (Internal Rate of Return)** | NPV를 0으로 만드는 할인율, 같은 조건이면 IRR이 높을수록 유리함 |
| **PP (Payback Period)** | Break Even Point로 비용과 성과가 0인 지점, 같은 조건이면 짧을 수록 좋음 |

## 정성적 평가 기법

### IV. IT 투자 평가의 정성적 평가 항목 및 내용

| 평가 항목 | 설명 |
|:---------|:-----|
| **IO (Information Orientation)** | 정보행동/가치, 정보 관리 실행 정보 기술 실행의 3역량 평가<br>3역량의 상호 작용을 통한 시너지 효과 창출 통한 성과 향상 |
| **IPM (IT Portfolio Management)** | 기업의 IT 포트폴리오 관련 모든 활동을 가치, 효익으로 구분<br>IT 자원, 투자, 프로젝트의 우선순위 관리 통한 효율성 확보 |
| **IE (Information Economics)** | IT와 비즈니스 관리자의 합의 통한 전략적 우선 순위 부여<br>양측 관리자간 우선 결정 요인 목록화 후 중요도, 위험 평가 |
| **BCG Matrix** | 시장의 성장률과 기업의 시장 점유율을 고려한 평가<br>Cash cow, Dog, Question mark, Star로 분류 하여 전략 수립<br>기업의 부가 가치 발생 활동을 주활동, 보조활동으로 분류 |
| **Value Chain** | 주활동(구매, 생산, 물류, 마케팅, A/S), 지원 활동(기획, 조달, 개발) 등으로 분류, 구성 |
