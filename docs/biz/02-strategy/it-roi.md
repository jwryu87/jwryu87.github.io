{: .label .label-blue }

> 암기: **현리전투상의**

---

## 핵심 키워드

`사전평가` `중간평가` `사후평가`

---

## 정의/개념

IT 투자 프로젝트 사전, 중간, 사후 평가 통한 **정량적 측정**, 유기적 연계, **타당성 관리** 모델

---

## 프로세스

### IT-ROI 평가 프로세스

```
                     IT 드 백
       ┌─────────────────────────────────┐
       │  사전평가  →  평가  →  중간평가  →  평가  →  사후평가  │
       └─────────────────────────────────┘
           ↓
    ┌──────────────────────────────────────────┐
    │ ① 현황분석   ② 투자비용    ③ 전략적   ④ 리스크   ⑤ 사업적   ⑥ 의사  │
    │   계획수립      분석/예측      가치평가      평가        가치평가     결정   │
    └──────────────────────────────────────────┘
```

### IT-ROI 7단계 프로세스

| # | 프로세스 |
|:--|:---------|
| 1 | 현황분석 및 계획수립 |
| 2 | 투자비용 분석 및 예측 |
| 3 | 전략적 가치평가 |
| 4 | 리스크 평가 |
| 5 | 사업적 가치평가 |
| 6 | 의사결정 |
| 7 | 피드백 |

---

## 평가방법론

### III. IT-ROI 측정 관련 평가방법론

| 유형 | 기법 | 특징 |
|:-----|:-----|:-----|
| **실물옵션** | 블랙숄즈 | IT 투자옵션의 가치변화 산출 |
| **가치평가** | 이항모형 | 경우의 수 기반 동적 반영 |
| **이윤** | 투자수익률 (ROI) | 투자대비 순이익 비율 |
| **편익분석** | 순현재가치 (NPV) | 순현재 가치 산출 |
| **다중** | 비즈니스 연계 | 핵심 역량별 복합평가 |
| **분석** | 우선도의 산출 | 측정결과 통한 우선도의 도출 |

> **유기적 연계통한** 일련된, 단년도, 지연적 **평가방식해적 "끝"**
