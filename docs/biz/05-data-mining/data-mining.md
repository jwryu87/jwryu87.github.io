# 데이터 마이닝 방법론
{: .fs-8 }

5.1 데이터 마이닝 방법론
{: .label .label-purple }

---

## 핵심 키워드

`KDD` `CRISP-DM` `SEMMA`

---

## 정의/개념

대용량 데이터에서 유용한 정보와 패턴을 발견하기 위한 체계적인 분석 방법론

---

## 개념 비교

### 2. 데이터 분석 방법론 개념 설명

| 구분 | KDD | CRISP-DM | SEMMA |
|:-----|:----|:---------|:------|
| **개념** | Knowledge(지식) 기반 즉, Database의 데이터 를 통해 통계적 패턴이나 지식을 찾기 위한 표준화된 처리 절차와 방법에 대한 데이터 분석 방법론 | 비즈니스의 이해 기반으로 데이터를 이해, 모델링, 평가 및 배포 등 데이터 분석 목적의 6단계로 진행되는 데이터 마이닝 및 분석 방법론 | SAS사 주도 통계적 방법 기반으로 데이터 샘플링, 탐색, 수정, 모델링, 검증 등의 기능을 제공하여 사용자가 쉽게 마이닝이 가능한 데이터 분석 방법론 |

---

## 상세 비교

### 3. 데이터 분석 방법론 상세 설명

| 구분 | KDD | CRISP-DM | SEMMA |
|:-----|:----|:---------|:------|
| **기반 기술** | 프로파일링 기술 기반 | 4단계 계층적 프로세스 모델 기반 | 통계 기술 기반 |
| **추출 대상** | Database | 도메인 지식 | 자사 기능 |
| **주요 특징** | 체계적 절차 | 비즈니스 이해 기반 | 통계적 방법<br>SAS사 기능 제공 |
| **주요 기술** | - Classification (분류)<br>- Clustering (군집)<br>- Predictive (예측) | - RACI 차트 (역할별 책임)<br>- Exploratory Data Analysis (EDA) | - Correlation Analysis<br>- Neural Network<br>- Logistic Regression |
| **활용 분야** | - 데이터 마이닝<br>- 데이터 패턴 해석<br>- ML/AI 응용 | - 비즈니스 데이터 분석<br>- 비즈 어플리케이션 적용 | 일반 데이터 분석 |
| **반복 여부** | 평가 결과 미 충족 시 반복 수행 | 단계별 반복 수행 (분석 품질 향상) | 목표 도달까지 반복 / 순환 |
| **분석 단계** | 5 단계 | 6 단계 | 5 단계 |
