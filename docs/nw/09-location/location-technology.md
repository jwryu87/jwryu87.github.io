## 정의/개념

이동체가 자신의 **위치, 속도, 경로 등을 알게 하는 위치결정 기술**

---

## 위치 측위 기술 개요

| 위치 측위 기법 | 실외 측위 기술 | 실내 측위 기술 |
|:-------------|:-------------|:-------------|
| Cell ID | 이동 통신 기반 | WLAN 기반 |
| ToA | GPS 기반 | Bluetooth 기반 |
| AoA | | UWB 기반 |

{: .note }
> 다양한 측위 기술을 융합하여 정확한 위치 정보를 파악하기 위한 기술 발전

---

## 1. 위치 측위 방법

| 구분 | 방법 | 설명 |
|:-----|:-----|:-----|
| **통신망 이용** | Cell ID | 통신용 전파 송수신 인프라(기지국, 중계기, AP 등)에 접속된 단말의 위치를 해당 AP의 위치로 매칭시키는 방법 |
| **거리 기반** | ToA | 위치가 알려진 다수의 AP와 단말 사이의 신호 전송 시간을 측정함으로써 단말의 위치를 계산하는 삼각측량법 |
| | TDoA | ToA 방식과 달리 AP들과 단말 사이의 시각동기를 필요로 하지 않으며 AP 들간의 시각동기만으로 측위가 가능한 방법 |
| | AoA | AP에서 전송된 신호의 각 정보를 사용하여 단말의 위치를 추정하는 방식 |
| **패턴 기반** | Fingerprint | 확률론적 모델링에 의한 위치 추정 방법으로 노이즈 및 주위 환경 정보를 위치추적을 위한 정보로 활용하는 방식<br>측위수행 전, 먼저 데이터베이스를 구성하는 training 단계를 수행<br>이외에도 클러스터링 기법, Hidden Markov Model, Bayesian Model 등의 다양한 기법을 사용 |

---

## 2. 실외 위치 추적 기술

| 구분 | 기술 | 설명 |
|:-----|:-----|:-----|
| **GPS 기반** | DGPS | GPS의 오차를 보정하여 위치를 추적 할 수 있는 기술(differential GPS)<br>SBAS(위성기반 보강시스템)와 GBAS(지상기반 보강시스템)로 구분 |
| | SBAS | 추가적인 위성방송메시지를 사용에서 광범위한 지역 또는 지역적인 보강을 지원하는 시스템 |
| | GBAS | 지상망을 통해 위치 보정정보를 제공하는 시스템 |
| **이동통신 기반** | - | 이동 통신 기지국을 이용하여 위치 정보 서비스 제공<br>통신 단말을 이용하며 쉽게 서비스 가능<br>오차를 보정하여 정확한 위치를 제공하기 위해 기술 발전 |

---

## 3. 실내 위치 추적 기술

### 가. 센서 및 주파수 기반의 실내 위치 추적 기술

| 구분 | 기술 | 설명 |
|:-----|:-----|:-----|
| **센서 기반** | 자기장센서 | 스마트 폰 자기장 센서를 이용해 미리 입력된 건물 내부 자기장 지도와 비교후 위치 추적 |
| | 관성센서 | 단말기 내 관성센서(가속도,방향각 등) 이용하여 측위<br>제자리 기준 측정값을 근간으로 상대적인 위치 측위 |
| | RFID | RFID Tag로 Bluetooth, WiFi등의 전용통을 이용해 서비스 제공<br>Tag의 위치 정보 수집 및 분석에 별도의 서버 필요 |
| **주파수 기반** | 적외선 | 실내 곳곳에 부착된 적외선 센서가 고유 ID 코드를 가진 적외선 장치를 인식하여 위치를 찾아내는 방식<br>디바이스 내부에 IrDA 컨트롤러 탑재필요 |
| | 초음파 | 스마트폰으로 초음파를 발사해 벽에 맞붙었다가 반사되는 것을 포착해 위치와 건물내부의 형태 파악 |

### 나. Network 기반의 실내 위치 추적 기술

| 구분 | 기술 | 설명 |
|:-----|:-----|:-----|
| **WLAN 기반** | WPS | 무선AP의 SSID와 MAC주소를 주요 파라미터로 활용<br>핑거프린팅방식으로 GPS가 지원되지 않는 영역에서 측위 |
| | RSSI | 센서로부터 전압레벨을 측정하여 실내위치 측위<br>무선주파수 내에서 송수신된 전력으로 거리 측정 |
| | UWB | UWB단말에 고유ID를 32bit로 송신<br>광대역주파수를 활용하여 뛰어난 통과력과 고속성 제공 |
| **WPAN 기반** | Zigbee | ZigBee 탑재단말이 AP통해 단말위치 전송<br>저비용성 제공 및 다수 노드와 클러스터링 용이 |
| | Bluetooth | Bluetooth 4.0, iBeacon 사용, 최대 50m까지 전송 가능<br>블루투스 AP 활용하여 지오펜싱 기반의 전파랩 생성 |

{: .highlight }
> 다양한 기술의 융합을 통해 보다 정확한 위치정보를 활용하며, 다양한 서비스 제공 가능
