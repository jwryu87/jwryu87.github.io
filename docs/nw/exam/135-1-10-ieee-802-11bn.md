### 🔑 핵심 키워드 3개

| 키워드 | 쉬운 설명 |
|:------|:---------|
| **UHR (Ultra High Reliability)** | 초고신뢰성 - 데이터가 끊기지 않고 확실히 도착 |
| **Wi-Fi 8** | 2028년 출시 예정인 차세대 무선랜 표준 |
| **100Gbps** | 현재 Wi-Fi 7보다 2배 이상 빠른 속도 |

---

### 📖 등장배경

> **문제**: 메타버스, AR/VR, 산업용 IoT, 자율주행 등 **초저지연·초고신뢰성**이 필요한 응용이 급증
> 
> **해결**: Wi-Fi 7(802.11be)의 후속으로, **신뢰성(Reliability)**에 초점을 맞춘 Wi-Fi 8(802.11bn) 표준 개발

---

### 📝 개념 정의

> **IEEE 802.11bn**은 Wi-Fi 8이라고 불리는 차세대 무선랜 표준으로,
> **UHR(Ultra High Reliability)**를 핵심 목표로 삼아
> 메타버스, 산업용 IoT, 로봇 공학 등 **높은 신뢰성을 요구하는 응용**을 지원합니다.
> 
> 🎯 비유: Wi-Fi 7이 "빠른 고속도로"라면, Wi-Fi 8은 "절대 막히지 않는 고속도로"

---

### 🏗️ 기술요소

#### PHY 계층 기술

| 기술 | 설명 |
|:-----|:-----|
| **DRU (Distributed RU)** | 분산 자원 유닛으로 전력 스펙트럼 밀도 제한 극복 |
| **Unequal Modulation** | 채널 환경에 따라 적절한 변조 방식 선택 |
| **New MCSs** | Wi-Fi 7 대비 다양한 MCS 조합 추가 |

#### MAC 계층 기술

| 기술 | 설명 |
|:-----|:-----|
| **Seamless Roaming** | 로밍 시 단절 없는 데이터 송수신 |
| **Multi-AP Coordination** | C-TDMA로 여러 AP 간 협력하여 저지연 전송 |
| **NPCA** | Non-Primary Channel Access로 채널 효율 향상 |
| **Dynamic Power Save** | 대기 전력 절감 기술 |

---

### ⭐ 차별점 키워드

> **UHR (Ultra High Reliability)**: Wi-Fi 8의 핵심 목표는 **속도**가 아닌 **신뢰성**!
> - Wi-Fi 7: EHT (Extremely High Throughput) - 극한 처리량
> - Wi-Fi 8: UHR (Ultra High Reliability) - 초고신뢰성

---

### 🏗️ 구성요소

#### IEEE 802.11bn 특징 3가지 `적연전`

| 특징 | 설명 | 상세 |
|:-----|:-----|:-----|
| **적**시 전송 신뢰성 | 데이터가 제 시간에 도착하는 신뢰성 확보 | MU-MIMO, QoS 통한 저지연 전송 |
| **연**결 상태 신뢰성 | 기존 채널 상태 파악 및 연결 품질 최적화 | 동적 전력 제어, 채널 피드백 기반 |
| **전**송 성공률 신뢰성 | MIMO와 공간 다중화로 동시 신호 전송 | 채널 부하 분산, 간섭 최소화 |

---

#### PHY 계층 후보 기술 `DRU-UM-MCS`

| 기술 | 설명 | 효과 |
|:-----|:-----|:-----|
| **D**RU (Distributed RU) | 분산 자원 유닛, 6GHz 대역 PSD 제한 극복 | 넓은 대역폭에 분포, 높은 전송 파워 |
| **U**nequal **M**odulation | 채널 환경 따라 적절한 변조 방식 선택 | 채널 적응형 변조 |
| New **MCS**s | Wi-Fi 7 대비 16개 MCS 조합 추가 | 다양한 환경 대응 |

---

#### MAC 계층 후보 기술 `SR-MAC-NPCA-DPS`

| 기술 | 설명 | 효과 |
|:-----|:-----|:-----|
| **S**eamless **R**oaming | 로밍 시 단절 없는 데이터 송수신 | 서비스 연속성 보장 |
| **M**ulti-**A**P **C**oordination | C-TDMA로 여러 AP가 TXOP 공유 | 저지연 트래픽 송수신 |
| **NPCA** | Non-Primary Channel Access | Primary 채널 점유 시 subchannel 사용 |
| **D**ynamic **P**ower **S**ave | 무선랜 단말 대기 전력 절감 | 제한된 성능 상태로 대기 전력 절감 |

### 📊 Wi-Fi 버전별 비교 `출최변공멀`

| 구분 | Wi-Fi 6 (802.11ax) | Wi-Fi 7 (802.11be) | Wi-Fi 8 (802.11bn) |
|:-----|:-------------------|:-------------------|:-------------------|
| **출**시연도 | 2019년 | 2024년 | 2028년 (예정) |
| **최**대 PHY 속도 | 9.6Gbps | 46Gbps | 100Gbps |
| **변**조 | 1024 QAM | 4096 QAM | 4096 QAM |
| **공**간 스트림 수 | 8 | 8 | 8 |
| **멀**티 링크 작동 | - | Yes | Yes |
| 멀티 AP 협력 | - | - | Yes |
| NPCA/dRU | - | - | Yes |
| 핵심 목표 | OFDMA, MU-MIMO | EHT (Extremely High Throughput) | UHR (Ultra High Reliability) |
