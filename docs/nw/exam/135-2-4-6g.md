### 🔑 핵심 키워드 3개
- **6G**: 2030년 상용화 목표의 차세대 이동통신
- **1Tbps**: 5G보다 50배 빠른 속도
- **테라헤르츠**: 6G에서 사용할 초고주파 대역

### 📖 등장배경

> 5G도 빠르지만, 홀로그램 회의, 원격 수술, 플라잉카 같은 미래 서비스를 위해서는 **더 빠르고 지연 없는 통신**이 필요해요! 그래서 2030년 상용화를 목표로 **6G**가 개발되고 있어요!

### 📝 개념 정의

> 6G는 **"5G보다 50배 빠른 1Tbps 속도로, 지상·공중·우주를 연결하는 초연결 통신 기술"**이에요.
> 
> 비유하자면, 5G가 고속도로라면 6G는 **우주까지 연결되는 초고속 하이퍼루프**와 같아요!

### 🏗️ 5G vs 6G 비교

| 구분 | 5G | 6G |
|:-----|:----|:----|
| 속도 | 20Gbps | 1Tbps |
| 지연 | 1ms | 0.1ms |
| 주파수 | ~100GHz | 100GHz~10THz |
| 상용화 | 2020년 | 2030년 |

│                                                                         │

|:-----|:-----------|:----------|:--------|
| **초성능** | 최대 전송속도 | 1Tbps | 20Gbps |
| | 체감 전송 속도 | 1Gbps | 100Mbps |
| | 광 액세스 | Tbps 급 | 최대20Gbps |
| **초대역** | 주파수 대역 | 100GHz 대역 이상 | 100GHz 대역이하 |
| | 대역폭 | 수십 GHz 대역폭 | 수GHz 대역폭 |
| **초정밀** | 무선 구간 지연 | 0.1ms | 1ms 이하 |
| | 종단간 지연 | 수ms | N/A |
| **초지능** | 지능화 네트워크 | 학습기반 이동통신(연결지능) | 해석적기반 이동통신 |

> ITU에서 IMT-2030 비전의 정식 발표를 기점으로 6G 이동통신기술 주파수 논의가 활발히 검토 중

### 다. 6G와 5G 서비스 비교

| 서비스 분야 | 5G | 6G |

| 실감콘텐츠 | 모바일 AR·VR 방송 | 원거리에서의 실시간 비대면 홀로그램 회의 |
| 자율주행차 | 차량-차량, 차량-인프라 간 초저지연통신 | 6G 위성으로 플라잉카, 드론과 초저지연 통신 |
| 스마트시티 | 우체국 드론택배: 산간지대 배송 6분 단축 | 디지털 트윈: 물류·교통 이동체에 대한 완전한 디지털 재현 및 관제 |
| 스마트공장 | 유선 기반 제조설비 라인의 무선화 | 산업현장 빅데이터 기반 안전하고 최적화된 설비 자동정밀제어 |

## Ⅳ. 6G 이동통신기술 주파수 동향

### 가. 주파수 대역별 특징

| 항목 | 주파수 대역 | 특징 |
|:-----|:----------|:-----|
| **테라헤르츠 대역** | 100GHz~10THz | 초고속 데이터 전송에 유리, 전파 감쇠가 심하고 도달 거리가 짧음 |
| **Sub-THz 대역** | 7~24GHz | 전파 도달 거리가 길고 건물 투과율이 높음 |
| **리미터파(mmWave) 대역** | 30~100GHz 대역 | 5G에서도 사용되는 대역으로 재활용 가능 |

### 나. 6G 주파수 동향 상세

| 주파수 대역 | 의제 | 주요 내용 |
|:----------|:-----|:---------|
| 50.4~51.4GHz | 항공/해상이동 주파수 공유 | IMT 일부 대역 공유, 수요예측 필요 |
| 37.5~42.5GHz | 고정위성망 기술/규정 검토 | IMT 일부 대역 공유, 수요예측 필요 |
| 4,400~4,800MHz | IMT 추가 주파수 지정 검토 | 1/2 지역 이동 1차 업무 |
| 7,125~8,400MHz | IMT 추가 주파수 지정 검토 | 2/3 지역 이동 1차 업무 |
| 7,125~7,250MHz | IMT 추가 주파수 지정 검토 | 1 지역 이동 1차 업무 |
| 14.8~15.35GHz | IMT 추가 주파수 지정 검토 | 글로벌 1차 업무 |
| 1.4~2GHz | MSS 주파수 분배 | 저속데이터 이동위성 서비스 |
| 698MHz~2.7GHz | MSS 주파수 분배 | IMT 단말 위성 접속 주파수 |
| 2GHz 위성업무용 | MSS 주파수 분배 | 지상/위성 공유주파수 |

> 국내에서도 6G 산업을 위해서 주파수 대역 분석 및 글로벌 주파수 확보를 주요 전략으로 추진

### 다. 6G 핵심 기술

| 구분 | 핵심 기술 |
|:-----|:---------|
| **초성능** | Tbps급무선통신기술, Tbps급광통신인프라기술 |
| **초대역** | 6G 이동통신지원RF 기술, 6G 주파수확보기술 |
| **초현실** | 초실감3차원공간미디어, 텐저블미디어서비스기술, 6G방송통신융합미디어 기반 전송 기술 |
| **초지능** | 지능형무선액세스네트워크기술, 초유연 6G 모바일코어, 멀티도메인/멀티스페이스6G 네트워크자동화 |
| **초정밀** | Tbps급무선통신기술, 6G 네트워크 종단간 초저지연 고정밀 고가용 네트워크 기술 |
| **초공간** | 3 차원 공간 이동체 브로드 밴드 무선 통신 기술, 천 음속 급 이동통신기술, 6G우주 인프라 및 액세스 |

> 6G 이동통신은 테라헤르츠(THz) 대역을 포함한 광대역 주파수를 활용할 것으로 예상
