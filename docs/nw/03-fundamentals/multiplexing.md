# 다중화 (Multiplexing)
{: .fs-8 }

3.1 네트워크 기본
{: .label .label-purple }

---

## 핵심 키워드

`주파수` `시분할` `파장` `코드` `FDM` `TDM` `WDM` `CDM`

---

## 정의/개념

다수 디바이스가 단일 데이터 링크를 공유하여 전송하는 효율적인 데이터 기법, 하위 계층의 신호를 모아서 상위 계층의 신호를 만드는 과정

---

## 다중화 유형

| 유형 | 방식 | 설명 |
|:-----|:-----|:-----|
| **FDM** | 주파수 분할 다중화 | 각각의 송신 신호를 서로 다른 송신 주파수로 변경하여 전송 |
| **TDM** | 시분할 다중화 | 다수의 송신 신호를 일정 시간 간격으로 각각을 순차적으로 전송 |
| **WDM** | 파장분할 다중화 | 다수의 송신 신호를 다른 빛의 파장으로 만든 후 결합하여 송신 |
| **CDM** | 코드분할 다중화 | 각각의 송신신호에 고유코드를 부여하여 여러 신호를 동시에 송신 |
| **SDM** | 공간분할 다중화 | 공간적으로 분리된 다수의 물리채널을 마치 하나의 채널인 것처럼 만들어진 논리적 채널로 전송 |

---

## 주파수 분할 다중화 (FDM)

| 구분 | 요소 | 설명 |
|:-----|:-----|:-----|
| **송신측** | 대역 | 일정 대역폭을 사전에 동수의 대역으로 나누어 신호를 변조하여 전송 |
| **전송** | 대역 | 식별가능 채널로 통신 중 무제한 사용 가능, 무 점유율 높아질 시포화 (1초 단위) |
| **보호대역** | - | 두 채널 간 상호 간섭 방지 |

{: .note }
> 간단한 구현으로 비용이 저렴하나 채널 이용율 저하 시 낭비

---

## 시분할 다중화 (TDM)

| 구분 | 요소 | 설명 |
|:-----|:-----|:-----|
| **동기식** | 방식 | 각 장치에 할당된 타임슬롯이 프레임내 위치가 항상 일정하게 정의 |
| **전송 탐색** | - | 데이터 미전송 시 타임슬롯도 낭비 발생 |
| **비동기** | 방식 | 각 장치에 할당된 타임슬롯이 미사용 내 위치가 가변되지 않게 전송 |
| | | 효율 재연 없음, 점유 오버헤드 발생 |
| **전송 단위** | - | 타임슬롯 크기가 bit, character 단위 구성 |

---

## 코드 분할 다중화 (CDM) 특징

| 특징 | 설명 |
|:-----|:-----|
| **내잡음성** | 난독 신호 스펙트럼으로 인해 다중경로 페이딩(Multipath Fading)에 유리 |
| **보안성** | 도청이어 감시에 강인함 |
| **유연성** | 수신만역서 전기대역에 사용되는 주파수 범위여역 외차가 적음 |
