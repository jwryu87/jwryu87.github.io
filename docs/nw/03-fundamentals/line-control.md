# 회선제어
{: .fs-8 }

3.1 네트워크 기본
{: .label .label-purple }

---

## 핵심 키워드

`ENQ/ACK` `Polling` `Select 모드` `Poll 모드`

---

## 정의/개념

반이중 전송 링크에서 두 개의 스테이션이 동시에 신호를 전송하는 경우, 발생하는 **신호 간 충돌 현상 방지 제어 방식**

---

## 회선제어 방식 비교

| 구분 | ENQ/ACK | Polling - Select 모드 | Polling - Poll 모드 |
|:-----|:--------|:----------------------|:--------------------|
| **개념** | 모든 노드는 동등한 권한으로 전송을 시작할 수 있으며, ENQ와 ACK 프레임으로 링크를 제어하는 회선제어 기법 | 주 스테이션에서 종속 스테이션에 전송할 데이터가 있을 때 SEL ACK 프레임으로 링크를 제어하는 회선제어 기법 | 종속 스테이션에서 주 스테이션으로 전송할 데이터가 있을 때 POL ACK 프레임으로 링크를 제어하는 회선제어 기법 |
| **특징** | 1:1 통신, 동등한 권한 | 주 스테이션에서 링크에 대한 제어 권한 보유 | 종속 스테이션에서 응답을 통해 링크를 제어 |

---

## 동작 원리

### ENQ/ACK 방식
```
단말A          단말B
  |---- ENQ ---->|
  |<--- ACK -----|
  |---- Data --->|
  |<--- ACK -----|
  |---- ... ---->|
  |---- EOT ---->|
```

### Polling - Select 모드
```
Station        Host    Host
  |---- SEL ---->|       |
  |<--- ACK -----|       |
  |---- Data --->|       |
  |<--- ACK -----|       |
```

### Polling - Poll 모드
```
Station        Host    Host
  |---- POLL --->|       |
  |<--- NAK -----|       |
  |              |       |
  |---- POLL ----------->|
  |<--- Data ------------|
  |<--- ACK -----|       |
```

{: .note }
> 반 이중(Half-Duplex) 모드에서 두 단말 또는 서버/클라이언트 간 동시에 신호를 발생하지 않도록 제어
