# 네트워크 오류제어
{: .fs-8 }

7.1 OSI 7 Layer
{: .label .label-purple }

---

## 핵심 키워드

`FEC` `BEC` `전진 오류 수정` `후진 오류 수정`

---

## 정의/개념

데이터링크 계층에서 발생 오류를 검출하고 수정 및 처리하는 방식

---

## 오류 발생 장소

데이터링크계층의 오류 발생 장소:

| 발생 장소 | 설명 |
|:---------|:-----|
| **DTE에서 발생하는 오류** | 데이터 단말장치 |
| **통신 제어 장치에서 발생하는 오류** | 통신 제어 장치 |
| **정보전송 회선에서 발생하는 오류** | 전송 회선 |
| **정보전송 장치에서 발생하는 오류** | 전송 장치 |
| **정보매체로 변화할 때 발생하는 오류** | 매체 변환 |
| **정보 발생지에서 발생하는 오류** | 발생지 |

{: .note }
> 데이터링크 계층에서 오류는 데이터단말장치나 통신회선, 전송매체 등의 기계적/구조적 원인에 따라 발생
> 오류가 발생하는 장소나 원인에 따라 적절한 오류제어방식을 사용하여 오류를 검출하고 수정 및 처리

---

## 오류 제어 유형

```
                    오류 제어(Error Control)
                           │
            ┌──────────────┴──────────────┐
            │                             │
      전진 오류 수정                  후진 오류 수정
         (FEC)                          (BEC)
            │                             │
    ┌───────┼───────┐            ┌────────┼────────┐
    │       │       │            │        │        │
  비블럭   블럭   콘볼루션      오류검출   수정 방식
   코드    코드    코드          방식
    │       │       │            │        │
  Turbo   BCH   해밍코드      Parity   Stop-and-
   코드          │           Check     Wait
                │             │          │
           Reed-Solomon    Block Sum  Go-Back-N
                          │          │
                         CRC     Selective-
                          │       Repeat
                        Check       │
                         Sum     Adaptive ARQ
                                    │
                                  H-ARQ
```
