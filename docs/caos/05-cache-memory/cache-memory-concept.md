## 핵심 키워드

`L1` `L2` `L3`

---

## 정의/개념

**캐시 메모리(Cache Memory)**는 메인 메모리와 CPU간의 데이터 속도 차이 따른 **병목현상 해결 위해** 중간 버퍼 역할 **메모리**입니다.

---

---

## 구성요소

| 유형 | 핵심 | 구성 | 데이터 호출 속도 |
|:-----|:-----|:-----|:---------------|
| **L1 (1차캐시)** | 고성능, 적은 용량, 코어 근접, 유형(인스트럭션, 데이터) | SRAM | 3클럭 |
| **L2 (2차캐시)** | 성능 저하, 용량추가 | DRAM | 8클럭 |
| **L3 (3차캐시)** | 캐시 일관성 구현 사용 | DRAM | 25클럭 |

---

## 캐시 메모리 특징

| 특징 | 설명 |
|:-----|:-----|
| **속도** | 주기억장치보다 빠른 접근 속도 |
| **용량** | 주기억장치보다 작은 용량 |
| **비용** | 주기억장치보다 비싼 단가 |
| **지역성 활용** | 시간/공간 지역성 기반 동작 |

---

## 캐시 성능 지표

| 지표 | 설명 |
|:-----|:-----|
| **적중률 (Hit Rate)** | 캐시에서 데이터를 찾은 비율 |
| **실패율 (Miss Rate)** | 캐시에서 데이터를 찾지 못한 비율 |
| **평균 접근 시간** | Hit Time + Miss Rate × Miss Penalty |
