## 정의/개념

**컴퓨터(Computer)**는 전자 회로를 이용한 고속 자동 계산기입니다. 수 계산 이외에 자동 제어, 데이터 처리, 사무 관리, 언어나 화상(畵像)의 정보 처리 등 광범위하게 이용되는 전자계산기입니다.

---

## 컴퓨터 구성의 분류

| 구분 | 구성요소 | 설명 |
|:-----|:---------|:-----|
| **하드웨어** | 중앙처리장치 | CPU (ALU + CU) |
| | 연산장치 | ALU (산술논리연산장치) |
| | 제어장치 | CU (제어장치) |
| | 기억장치 | 주기억장치, 보조기억장치 |
| | 입출력장치 | 입력장치, 출력장치 |
| **소프트웨어** | 시스템 소프트웨어 | OS, 컴파일러 등 |
| | 응용 소프트웨어 | 사용자 프로그램 |

## 하드웨어 기능별 분류

| 장치 | 구성요소 | 설명 |
|:-----|:---------|:-----|
| **중앙처리장치** | ALU | 산술논리연산장치 |
| | CU | 제어장치 |
| **기억장치** | 레지스터 | CPU 내부 고속 기억장치 |
| | 캐시 | CPU와 주기억장치 사이 |
| | 주기억장치 | RAM, ROM |
| | 보조기억장치 | HDD, SSD |
| **입력장치** | - | 마우스, 키보드 등 |
| **출력장치** | - | 모니터, 프린터 등 |
