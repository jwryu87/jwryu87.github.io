# PNM (Processing near Memory)
{: .fs-8 }

9.1 신기술
{: .label .label-green }

---

## 핵심 키워드

`근접 메모리 처리` `데이터 이동 최소화` `PIM`

---

## 정의/개념

**PNM(Processing near Memory)**은 메모리 근처에서 연산을 수행하여 **데이터 이동을 최소화**하는 컴퓨팅 패러다임입니다.

---

## PIM vs PNM

| 구분 | PIM | PNM |
|:-----|:----|:----|
| **위치** | 메모리 내부 | 메모리 근처 |
| **통합 수준** | 높음 | 중간 |
| **설계 복잡도** | 높음 | 상대적 낮음 |

---

## 장점

- 데이터 이동 감소
- 에너지 효율 향상
- 대역폭 병목 해소
- 지연 시간 감소
