{: .fs-8 }

9.1 신기술
{: .label .label-green }

---

## 핵심 키워드

`메모리 확장` `메모리 풀링` `PCIe 기반`

---

## 정의/개념

**CXL(Computer Express Link)**은 CPU와 가속기, 메모리 간의 고속 인터커넥트를 위한 **개방형 표준 인터페이스**입니다.

---

## CXL 프로토콜

| 프로토콜 | 설명 |
|:---------|:-----|
| **CXL.io** | PCIe와 호환되는 I/O 프로토콜 |
| **CXL.cache** | 가속기가 호스트 메모리 캐싱 |
| **CXL.mem** | 호스트가 장치 메모리 접근 |

---

## 활용 분야

- 메모리 풀링
- 메모리 확장
- 가속기 연결
- 데이터센터
