# 워치독 타이머 (Watchdog Timer)
{: .fs-8 }

14.1 기타
{: .label .label-yellow }

---

## 핵심 키워드

`시스템 안정성` `하드웨어 타이머` `리셋`

---

## 정의/개념

**워치독 타이머(Watchdog Timer)**는 시스템이 정상 동작하는지 감시하고, 이상 발생 시 시스템을 **자동으로 리셋**하는 하드웨어 타이머입니다.

---

## 동작 원리

1. 타이머 시작
2. 정상 동작 시 주기적으로 타이머 리셋 (Kick)
3. 타이머 만료 시 시스템 리셋

---

## 용도

- 임베디드 시스템 안정성
- 무한 루프 탐지
- 시스템 복구
