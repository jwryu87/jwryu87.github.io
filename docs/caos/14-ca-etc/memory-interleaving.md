# 메모리 인터리빙 (Memory Interleaving)
{: .fs-8 }

14.1 기타
{: .label .label-yellow }

---

## 핵심 키워드

`메모리 대역폭` `병렬 접근` `모듈 분산`

---

## 정의/개념

**메모리 인터리빙(Memory Interleaving)**은 연속된 메모리 주소를 여러 메모리 모듈에 분산시켜 **메모리 접근 속도를 향상**시키는 기법입니다.

---

## 인터리빙 방식

| 방식 | 설명 |
|:-----|:-----|
| **하위 주소 인터리빙** | 하위 비트로 모듈 선택 (연속 주소 분산) |
| **상위 주소 인터리빙** | 상위 비트로 모듈 선택 (블록 단위 분산) |

---

## 장점

- 메모리 대역폭 향상
- 병렬 접근 가능
- 지역성 활용

---

## 단점

- 모듈 수 제한
- 복잡한 주소 지정
