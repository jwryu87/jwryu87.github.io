## 정의/개념

다양한 연산 장치들의 특성과 용도 비교

---

## 프로세서 비교

| 구분 | CPU | GPU | FPGA | ASIC |
|:-----|:----|:----|:-----|:-----|
| **특성** | 범용 처리 | 병렬 처리 | 프로그래머블 | 특정 용도 |
| **유연성** | 높음 | 중간 | 높음 | 없음 |
| **성능** | 중간 | 높음 (병렬) | 중간 | 최고 |
| **전력** | 중간 | 높음 | 중간 | 낮음 |
| **비용** | 낮음 | 중간 | 높음 | 매우 높음 (개발) |

---

## 용도

| 프로세서 | 주요 용도 |
|:---------|:---------|
| **CPU** | 범용 컴퓨팅, 운영체제, 애플리케이션 |
| **GPU** | 그래픽 렌더링, AI/ML 학습, 병렬 연산 |
| **FPGA** | 프로토타이핑, 저지연 처리, 맞춤 로직 |
| **ASIC** | 암호화폐 채굴, 네트워크 장비, AI 추론 |
