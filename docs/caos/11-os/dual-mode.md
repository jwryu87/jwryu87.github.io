# 이중 모드 (Dual Mode)
{: .fs-8 }

11.1 운영체제
{: .label .label-purple }

---

## 핵심 키워드

`커널 모드` `사용자 모드` `특권 명령` `시스템 콜` `모드 비트`

---

## 정의/개념

**이중 모드(Dual Mode)**는 운영체제가 **커널 모드**와 **사용자 모드** 두 가지 실행 모드를 제공하여 시스템을 보호하는 방식입니다.

---

## 모드 비교

| 구분 | 커널 모드 | 사용자 모드 |
|:-----|:---------|:-----------|
| **권한** | 모든 명령어 실행 가능 | 제한된 명령어만 실행 |
| **접근** | 모든 메모리/하드웨어 접근 | 제한된 영역만 접근 |
| **용도** | OS 커널 코드 실행 | 응용 프로그램 실행 |
| **모드 비트** | 0 | 1 |

---

## 모드 전환

```
사용자 프로그램 (사용자 모드)
         │
         │ 시스템 콜
         ▼
    커널 (커널 모드)
         │
         │ 처리 완료
         ▼
사용자 프로그램 (사용자 모드)
```

---

## 특권 명령

- I/O 명령
- 타이머 설정
- 인터럽트 제어
- 메모리 관리
