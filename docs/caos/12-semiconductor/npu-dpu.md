# NPU/DPU
{: .fs-8 }

12.1 프로세서
{: .label .label-green }

---

## 핵심 키워드

`신경망 처리` `데이터 처리` `AI 가속` `네트워크 오프로드`

---

## 정의/개념

- **NPU (Neural Processing Unit)**: AI/ML 연산에 특화된 **신경망 처리 장치**
- **DPU (Data Processing Unit)**: 데이터 중심 워크로드를 처리하는 **데이터 처리 장치**

---

## NPU vs DPU 비교

| 구분 | NPU | DPU |
|:-----|:----|:----|
| **목적** | AI/ML 연산 가속 | 데이터 처리 오프로드 |
| **주요 기능** | 행렬 연산, 추론 | 네트워크, 스토리지, 보안 |
| **활용** | 스마트폰, 엣지 AI | 데이터센터, 클라우드 |

---

## NPU 특징

- 병렬 처리 최적화
- 저전력 설계
- 추론 가속

---

## DPU 특징

- SmartNIC 발전형
- CPU 오프로드
- 네트워크 가속
