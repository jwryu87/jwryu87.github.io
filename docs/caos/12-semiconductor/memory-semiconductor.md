# 메모리 반도체와 비메모리 반도체
{: .fs-8 }

12.2 메모리
{: .label .label-blue }

---

## 핵심 키워드

`DRAM` `NAND` `시스템 반도체` `로직 반도체`

---

## 정의/개념

- **메모리 반도체**: 데이터를 저장하는 반도체
- **비메모리 반도체 (시스템 반도체)**: 데이터를 처리하는 반도체

---

## 비교

| 구분 | 메모리 반도체 | 비메모리 반도체 |
|:-----|:-------------|:---------------|
| **기능** | 데이터 저장 | 데이터 처리 |
| **종류** | DRAM, NAND, SRAM | CPU, GPU, AP, ASIC |
| **특성** | 대량 생산, 범용 | 다품종 소량, 맞춤 설계 |
| **시장** | 메모리 시장 | 시스템 반도체 시장 |

---

## 메모리 반도체 종류

| 종류 | 설명 |
|:-----|:-----|
| **DRAM** | 휘발성, 고속 랜덤 접근 |
| **SRAM** | 휘발성, 캐시 메모리 |
| **NAND Flash** | 비휘발성, SSD |
| **NOR Flash** | 비휘발성, 코드 저장 |

---

## 비메모리 반도체 종류

| 종류 | 설명 |
|:-----|:-----|
| **CPU** | 범용 프로세서 |
| **GPU** | 그래픽/병렬 처리 |
| **AP** | 모바일 프로세서 |
| **ASIC** | 특정 용도 칩 |
