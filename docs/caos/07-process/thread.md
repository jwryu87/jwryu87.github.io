# 스레드 (Thread)
{: .fs-8 }

7.1 프로세스
{: .label .label-purple }

---

## 핵심 키워드

`경량 프로세스` `자원 공유` `TCB` `커널 스레드` `사용자 스레드`

---

## 정의/개념

**스레드(Thread)**는 프로세스 내에서 실행되는 **경량 프로세스**로, 같은 프로세스 내의 다른 스레드와 자원을 공유합니다.

---

## 프로세스 vs 스레드

| 구분 | 프로세스 | 스레드 |
|:-----|:---------|:-------|
| **자원 공유** | 독립적 메모리 공간 | 코드, 데이터, 힙 공유 |
| **생성 비용** | 높음 | 낮음 |
| **통신** | IPC 필요 | 직접 통신 가능 |
| **문맥교환** | 오버헤드 큼 | 오버헤드 작음 |

---

## 스레드 유형

| 유형 | 설명 |
|:-----|:-----|
| **커널 스레드** | OS 커널이 관리하는 스레드 |
| **사용자 스레드** | 사용자 라이브러리가 관리하는 스레드 |
| **하이브리드** | 커널과 사용자 스레드 혼합 |

---

## 멀티스레드 장점

- 응답성 향상
- 자원 공유
- 경제성 (생성/문맥교환 비용 절감)
- 다중 CPU 활용
