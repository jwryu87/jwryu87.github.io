## 정의/개념

**가상메모리 관리기법**은 물리적 주기억장치의 한계를 극복하기 위해 프로그램 일부를 보조기억장치에 저장, 필요시에 **할당**, **배치**, **인출**, **교체**하는 **메모리 관리 기술**입니다.

## 관리기법 상세

### 할당기법 (Allocation)

| 구분 | 핵심 기법 | 설명 |
|:-----|:---------|:-----|
| **연속할당기법** | - | 고정할당(정적)기법, 가변할당(동적)기법 |
| **불연속할당기법** | - | 페이징, 세스멘테이션, 페이지드 세그멘테이션 |

### 배치기법 (Placement)

| 구분 | 핵심 기법 | 설명 |
|:-----|:---------|:-----|
| **First-Fit** | - | 가장 처음에 남는 공간에 할당하는 기법 |
| **Best Fit** | - | 전체 스캔을 하고, 최적을 공간을 찾아 할당하는 기법 |
| **Next Fit** | - | 최근 할당한 블록부터 다음 위치에 할당하는 기법 |
| **Worst Fit** | - | 가장 맞지 않은 공간에 할당하는 기법 |

### 인출기법 (Fetch)

| 구분 | 핵심 기법 | 설명 |
|:-----|:---------|:-----|
| **Demand Fetch (요구)** | - | 실행 프로그램이 요구할 때 참조된 페이지나 세그먼트 만을 주기억장치로 옮겨지는 적재기법 |
| **Pre Fetch (예측)** | - | 실행 프로그램에 의해 참조될 것을 예상하여 미리 주기억장치로 옮겨지는 적재기법 |

### 교체기법 (Replacement)

| 구분 | 핵심 기법 | 설명 |
|:-----|:---------|:-----|
| **FIFO** | First In First Out | 가장 먼저 들어온 페이지를 교체 |
| **LFU** | Least Frequently Used | 현재 기준 사용횟수가 가장 적은 페이지 교체 |
| **LRU** | Least Recently Used | 현재 기준 가장 오랫동안 사용되지 않은 페이지 교체 |
| **OPT** | Optimal Page | 가장 오랫동안 사용되지 않을 페이지 선택 교체 |
| **NUR** | Not Used Recently | 최근 사용되지 않는 페이지를 2bit 이용해 교체 |
