# 지역성 (Locality)
{: .fs-8 }

4.3 문제/최적화
{: .label .label-blue }

> 암기: **공시순**

---

## 핵심 키워드

`공간 지역성` `시간 지역성` `순차 지역성`

---

## 정의/개념

**지역성(Locality)**은 프로그램이 실행되는 동안 현재 실행되는 부분 **집중 참조 현상**입니다.

---

## 지역성 유형

| 지역성 유형 | 설명 |
|:-----------|:-----|
| **공간 지역성** | 특정 클러스터 기억 장소들에 대해 참조가 집중적 액세스 현상 |
| **시간 지역성** | 최근 사용 기억 장소들이 집중적 액세스 현상 |
| **순차 지역성** | 데이터 순차적 액세스 현상 |

---

## 지역성 사례

| 구분 | 기법 | 지역성 |
|:-----|:-----|:-------|
| **가상메모리** | Working Set | 공간 지역성 |
| | TLB | 공간 지역성 |
| **캐시메모리** | LRU | 시간 지역성 |
| **네트워크** | CDN | 공간 지역성 |

---

## 지역성 원리 활용

| 활용 분야 | 설명 |
|:---------|:-----|
| **캐시 메모리** | 지역성 기반 데이터 미리 적재 |
| **페이지 교체** | LRU 알고리즘에서 시간 지역성 활용 |
| **Working Set** | 공간 지역성 기반 페이지 관리 |
| **Prefetching** | 순차 지역성 기반 미리 로드 |
