---

## 정의/개념

머신러닝 파이프라인에서 피처 엔지니어링, 하이퍼파라미터 최적화, 신경망 구조 탐색 진행 머신러닝 프로세스

> 반복적이나 고도의 전문성이 요구되는 최적화 알고리즘 선정 작업을 자동화하여 모델 개발자 개입을 최소화

## 프로세스

| # | 프로세스 | 구성요소 |
|:--|:--------|:---------|
| 1 | **피처 엔지니어링** | PCA<br>K-means clustering |

---

## 단계별 상세 설명

### 1. 피처 엔지니어링 (Feature Engineering)

| 기술 | 설명 |
|:-----|:-----|
| **PCA** | 차원 축소를 통한 주요 특성 추출 |
| **K-means clustering** | 유사 데이터 그룹화 |
| **Feature Selection** | 중요 특성 자동 선택 |

### 2. 하이퍼파라미터 최적화 (Hyper-Parameter Optimization)

| 기술 | 설명 |
|:-----|:-----|
| **그리드 탐색** | 모든 조합 시도 (Grid Search) |
| **랜덤 탐색** | 무작위 조합 시도 (Random Search) |
| **베이지안 최적화** | 확률 기반 최적값 탐색 |

### 3. 신경망 구조탐색 (Neural Architecture Search)

| 기술 | 설명 |
|:-----|:-----|
| **진화 알고리즘** | 유전 알고리즘 기반 구조 탐색 |
| **강화학습** | 보상 기반 구조 최적화 |
| **성능 추정 전략** | 빠른 성능 예측으로 탐색 효율화 |
