## 핵심 키워드

`ETL` `정규화` `편향 분석` `하이퍼 파라미터` `모델 병렬처리`

---

## 정의/개념

데이터 수집, 모델 준비, 알고리즘 학습, 예측 제공 전 단계 자동화 머신러닝 아키텍처

> 수집, 전처리, 모델 교육, 예측의 4 단계로 구성되는 머신러닝 파이프라인

## 프로세스

| # | 프로세스 | 구성요소 |
|:--|:--------|:---------|
| 1 | **데이터 흡수** | Data Lake<br>ETL |
| 2 | **데이터 준비** | 정규화<br>편향 분석 |
| 3 | **모델 교육** | 하이퍼 파라미터<br>모델 병렬처리 |
| 4 | **예측 제공** | 학습모델 배치 |

---

## 단계별 상세 설명

### 1. 데이터 흡수 (INGESTION)

| 구성요소 | 설명 |
|:---------|:-----|
| **Data Lake** | 다양한 소스의 원시 데이터 저장소 |
| **ETL** | Extract, Transform, Load 프로세스 |

### 2. 데이터 준비 (PREPARATION)

| 구성요소 | 설명 |
|:---------|:-----|
| **정규화** | 데이터 스케일 조정, 결측값 처리 |
| **편향 분석** | 데이터 편향 탐지 및 보정 |

### 3. 모델 교육 (MODEL TRAINING)

| 구성요소 | 설명 |
|:---------|:-----|
| **하이퍼 파라미터** | 학습률, 배치 크기 등 최적화 |
| **모델 병렬처리** | 분산 학습으로 성능 향상 |

### 4. 예측 제공 (MODEL DEPLOYMENT)

| 구성요소 | 설명 |
|:---------|:-----|
| **At client** | 엣지 디바이스 배포 |
| **On backend** | 서버 배포 |

---

## 재학습 (RETRAINING)

| 유형 | 설명 |
|:-----|:-----|
| **Incremental Retraining** | 새로운 데이터로 점진적 모델 업데이트 |
| **Full Retraining** | 전체 데이터로 모델 재학습 |
