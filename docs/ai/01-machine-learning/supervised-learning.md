# 분류(Classification), 군집(Clustering), 회귀(Regression) 비교
{: .fs-8 }

1.2 지도학습
{: .label .label-purple }

---

## 핵심 키워드

- **분류**: 지도학습, KNN, SVM
- **군집**: 비지도학습, K-means, Clustering
- **회귀**: 지도학습, 선형회귀, 로지스틱 회귀

---

## 정의/개념

| 유형 | 정의 |
|:-----|:-----|
| **분류** | 기존의 데이터의 레이블 속성을 파악하고, 새로 관측된 데이터의 레이블을 판별하는 지도학습 |
| **군집** | 데이터 간의 유사도를 정의하고 그 유사도에 가까운 데이터부터 순서대로 합쳐가는 지도학습 |
| **회귀** | 레이블링된 학습 데이터를 통해 데이터 특성과 레이블의 관계를 함수식으로 표현하는 지도 학습 |

---

## 분류, 군집, 회귀 상세 비교

| 비교 | 분류 | 군집 | 회귀 |
|:-----|:-----|:-----|:-----|
| **개념도** | + / - 그룹 분리선 | ○ 그룹핑 | 데이터 추세선 |
| **목적** | Label이 존재하며, 새로운 데이터의 그룹을 예측하기 위함 | Label 또는 카테고리가 없는 상황에서 데이터 자체의 특성에 대해서 알고자 함 | 독립변수와 종속변수 사이의 구체적인 함수식을 찾고 예측하고자 함 |
| **학습유형** | 지도 학습 | 비지도 학습 | 지도 학습 |
| **레이블** | 있음 | 없음 | 있음 |
| **활용** | A, B, C 중의 하나 분류 | 유형의 분류 | 연속값 예측 |
| **사례** | 필기체 인식<br>스팸메일 필터링 | 구매자 분류<br>임상실험 환자군 | 투자예측<br>수요/공급 예측 |
| **알고리즘** | kNN(k Nearest Neighbor)<br>SVM(Support Vector) | k-means<br>h-clustering | 선형 회귀 분석<br>로지스틱 회귀 분석 |
