# 회귀분석(Regression Analysis), 상관분석(Correlation Analysis) 비교
{: .fs-8 }

1.2 지도학습
{: .label .label-purple }

---

## 핵심 키워드

- **회귀분석**: 독립변수, 종속변수, 회귀계수, 회귀방정식
- **상관분석**: 분산, 공분산, 상관계수, 두개의 변수

---

## 정의/개념

| 분석 | 정의 |
|:-----|:-----|
| **회귀분석** | 독립변수와 종속변수간 상관관계 분석 및 함수식 표현 분석기법 |
| **상관분석** | 두 변수간 상관계수 측정, 선형적 관계 분석 기법 |

---

## 관계

```
회귀분석(Regression Analysis)          상관분석(Correlation Analysis)

    독립      인과관계     종속           선형관계
    변수  ─────────→   변수      A 변수  ←────────→  B 변수
```

---

## 회귀분석과 상관분석 상세 비교

| 구분 | 회귀분석(Regression Analysis) | 상관분석(Correlation Analysis) |
|:-----|:------------------------------|:------------------------------|
| **개념도** | Y = β₀ + β₁X (직선 그래프) | 양의 상관관계 / 음의 상관관계 (산점도) |
| **구성요소** | 독립변수, 종속변수, 회귀계수, 회귀방정식 | 분산, 공분산, 상관계수, 두개의 변수 |
| **유형** | 선형 회귀분석: Y = β₀ + β₁X + ε<br>로지스틱 회귀분석: Y = 1/(1+e⁻ˣ) = sigmoid(z) | 단순상관분석: 단순히 두개의 변수가 어느 정도 강한관계에 있는지 측정<br>다중상관분석: 2개 이상의 변수간 관계 강도를 측정 |
| **계수** | 회귀계수(SSE): 오차의 제곱의 합으로 계산<br>SSE = Σeᵢ² = e₁² + e₂² + ... + eₙ² | 피어슨 상관계수: 기본적 사용 기법<br>cov(X,Y) / σₓσᵧ (σ: 표준 편차, cov: 공분산) |
