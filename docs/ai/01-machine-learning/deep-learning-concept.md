---

## 알고리즘 경량화

기존 알고리즘의 불필요한 파라미터를 제거하거나, 파라미터의 공통된 값을 공유하거나, 파라미터의 표현력을 잃지 않으면서 기존 모델의 크기를 줄이는 경량화 기법

| 기술 분류 | 기술 요소 | 설명 |
|:---------|:---------|:-----|
| **모델 압축** | 가중치 가지치기<br>양자화/이진화<br>가중치 공유 | 기존 알고리즘의 불필요한 파라미터 제거, 공유 등을 통해 파라미터의 표현력을 잃지 않으면서 모델의 크기를 줄이는 방법 |
| **지식 증류** | 지식증류/전이학습<br>Teacher 모델<br>Student 모델 | 학습된 기본 모델을 통해 새로운 모델의 생성 시 파라미터 값을 활용하여 학습시간을 줄이는 기술 |
| **하드웨어 가속화** | 선형대수학 라이브러리(BLAS)<br>벡터/행렬 연산 병렬처리 | BLAS: Basic Linear Algebra Subprograms<br>행렬·행렬, 행렬·벡터, 벡터·벡터 연산 가속화<br>모바일 기기를 중심으로 뉴럴 프로세싱 유닛(NPU)을 통해 추론 속도를 향상시키는 기술 |
| **모델 압축 자동 탐색** | PocketFlow<br>AMC | AMC: AutoML for Model Compression<br>알고리즘 경량화 연구 중 일반적인 모델 압축 기법을 적용한 강화 학습 기반의 최적 모델 자동 탐색 기술 |

---

## 모델 구조 경량화

효율적인 네트워크 구조를 설계하거나 합성곱 연산의 다양한 변이, 모델 자동 탐색과 같이 모델의 구조를 경량화 시키는 알고리즘 기법

| 기술 분류 | 기술 요소 | 적용 모델 | 설명 |
|:---------|:---------|:---------|:-----|
| **병목 구조 변경 기술** | 잔여 블록(Residual Block)<br>병목 구조(Bottleneck Architecture)<br>밀집 블록(Dense Block)<br>3×3 필터를 1×1로 대체 | ResNet<br>DenseNet<br>SqueezeNet | 기존 계층구조는 다른 단 구조 구조를 이용하여 파라미터 더 수 및 모델 성능을 개선하는 기술 |
| **효율적인 합성곱 필터 기술** | 채널별(Depthwise) 합성곱<br>픽셀별(Pointwise) 합성곱 | MobileNet<br>SuffleNet | 일반 신경망의 가장 큰 계열량을 요구하는 합성곱 필터의 연산량을 효율적으로 줄이는 기술 |
| **경량 모델 자동 탐색 기술** | 신경망 자동 생성<br>연산량 대비 모델 압축비 조정 | NetAdapt<br>MNasNet | 특정 요소(지연시간, 에너지 소모 등)가 주어진 경우, 강화 학습을 통해 최적 모델을 자동 탐색하는 기술 |
