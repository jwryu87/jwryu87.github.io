# 강화학습, 역강화학습, 모방학습
{: .fs-8 }

1.4 강화학습
{: .label .label-yellow }

---

## 핵심 키워드

- **강화학습**: 에이전트, 환경, 상태, 행동, 보상, 정책
- **역강화학습**: 보상 함수 추정, 전문가 시연
- **모방학습**: 전문가 시연 모방, 행동 복제

---

## 정의/개념

| 학습 유형 | 정의 |
|:---------|:-----|
| **강화학습** | 보상을 통한 학습 |
| **역강화학습** | 에이전트 정책이나 행동 이력을 통해 그 행동을 설명하는 보상 함수 추정 학습 기법 |
| **모방학습** | 전문가 행동 모방, 직접적 보상 미요구, 정책 직접 설계 학습 기법 |

---

## 강화학습 vs 역강화학습

| 구분 | 강화학습 | 역강화학습 |
|:-----|:---------|:---------|
| **개념** | 에이전트(agent)가 어떤 상태(state)에서 행동(action)을 수행할 때마다 그 성능에 대한 피드백을 제공하는 보상 함수를 통해 학습하는 알고리즘 | 에이전트의 정책이나 행동 이력을 통해 그 행동을 설명하는 보상 함수를 구하는 알고리즘<br>에이전트가 최선의 행동을 선택했다는 가정하에 이 행동에 대한 보상 함수를 추정하는 학습 방식<br>주어진 설정이 강화 학습의 역이 됨 |

### 개념도

```
강화학습 (RL)                    역강화학습 (IRL)
    
    환경                             환경
     │                               │
보상함수 ─→ RL ─→ 최적정책      최적정책 ─→ IRL ─→ 보상함수

주어진 보상 함수를 통해          최신의 행동 이력(최적 정책)을
최적 정책을 계산                 입력으로 보상 함수를 찾음
```

## 모방학습

| 유형 | 핵심 |
|:-----|:-----|
| **행동복제** | 동작 시퀀스 궤적 수집 정책 학습 알고리즘 |
| **견습학습** | 전문가 시연 궤적, 보상 함수 추정 학습 알고리즘 |
