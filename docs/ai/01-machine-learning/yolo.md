# YOLO(You Only Look Once)
{: .fs-8 }

5.1 딥러닝
{: .label .label-blue }

---

## 핵심 키워드

`S x S 그리드` `B개의 바운딩 박스` `C개의 클래스 확률 이용 객체를 인식`

---

## 정의/개념

하나의 뉴럴넷을 사용하여 이미지 하나에서 S x S 그리드, B개의 바운딩 박스, C개의 클래스 확률 이용 객체를 인식하는 실시간 객체 탐지 알고리즘

---

## 아키텍처

```
                신뢰도
              ┌───→  신뢰도
              │       높음
    입력    ┌─┴─┐   ┌─────┐   
           │   │ → │바운딩│ → 최종 객체 인식
    이미지  │   │   │ 박스 │
           └───┘   └─────┘
          Resize   클래스 확률 맵
          S x S Grid
```

> 기본값으로 448 × 448 사이즈로 Resize, 7 x 7 그리드로 구성하여 객체인식 수행

---

## 절차

| 절차 | 핵심 기술 | 설명 |
|:-----|:---------|:-----|
| **S x S 그리드 분할** | 이미지 Resize, Grid 셀 분할 | 해당 셀에 물체의 중심이 있는지 판단 |
| **바운딩 박스 생성** | 각 셀당 신뢰도 측정<br>신뢰도 점수 = 확률(객체) × IoU(truth, predict) | 모델이 박스안에 물체가 어느 정도 있는지 판단하는 지와 정확도를 표시 |
| **클래스 확률 맵 구성** | P(Class \| Object) | 해당 셀에 물체가 있을 때 어떤 클래스가 인을지 조건부 확률을 계산 |
| **최종 객체 인식** | CNN Network<br>Non-Maximal Suppression | GoogleLeNet, VGGNet 등 CNN Network 이용 객체 분류 |

---

## 기술 활용 분야

| 서비스 분류 | 세부 기능 | 설명 |
|:-----------|:---------|:-----|
| **전자상거래** | 결제, 출금, 환전 등 | 국내외 은행 및 핀 테크 기업 등에서 결제 서비스 제공을 위해 얼굴 정보 활용 |
| **신분확인 및 출입통제** | 신분확인 서비스 | 각종 스마트 디바이스 및 산업기계, 시험 응시와 같이 본인 확인이 필요한 분야에 사용 |
| **엔터테인먼트** | 영상변형, 합성, 개선 등 | 스마트 디바이스에 장착된 카메라와 디스플레이를 이용한 서비스 개발<br>얼굴이나 배경과 같은 특정 영역을 AI를 융하여 개선하는 기능 추가 |
