## 핵심 키워드

## 정의/개념

기계학습 모델링과 운영 결합, 기계학습 거버넌스, 생명주기 관리 및 자동화 방법론

---

## 아키텍처

ModelOps는 AI모델이 생성되는 방식에 관계없이 비즈니스를 효과적으로 관리

```
[Model Development] ←→ [Model Operations] ←→ [Business Outcomes]

        Test → Build → Approve → Deploy → Configure
                          ↓
        Validate ← Register ← Package ← Monitor → Interact
```

---

## 구성요소

| 구분 | 구성요소 | 핵심 |
|:-----|:---------|:-----|
| **프로세스** | Discovery | 가설 검증, 모델 실험 |
| | Governance | 검증 모델 Repository 저장 |
| | Deployment | CI/CD 통한 자동화 파이프라인 생성 |
| **기술요소** | 운영 자동화 | 모델 운영 자동화 |
| | 생명주기 관리 | 규제, 위험관리 |
| | AI 가속화 | MLOps 결합 |
