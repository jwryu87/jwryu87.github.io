---

## 핵심 키워드

`입력 데이터` `학습 데이터` `출력 데이터` `Training Data` `예측 모델`

---

## 정의/개념

컴퓨터가 학습할 수 있도록 Training Data 이용하여 알고리즘 개발 및 정보추론 학습 기법

---

## 알고리즘 구조

```
학습 데이터(D) = {(xᵢ, yᵢ) | i=1,...,N}
       ↓
  기계학습 알고리즘
       ↓
    예측 모델 M
       
입력 데이터(xᵢ) → 예측 모델 M: ŷᵢ = fₘ(xᵢ; W) → 출력 데이터(결과 예측치)(yᵢ)
```

W: 모델 M의 파라미터 집합

---

## 주요 구성요소

| 구분 | 요소 | 설명 |
|:-----|:-----|:-----|
| **입력** | 학습 데이터 | 입력-출력 쌍의 집합 D = {(xᵢ, yᵢ)} |
| **처리** | 기계학습 알고리즘 | 데이터로부터 패턴을 학습하는 알고리즘 |
| **출력** | 예측 모델 | 학습된 파라미터 W를 가진 함수 fₘ |
