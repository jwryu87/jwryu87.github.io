# 다중공선성(Multicollinearity) 문제
{: .fs-8 }

1.2 지도학습
{: .label .label-purple }

---

## 핵심 키워드

`분산팽창요인(VIF) 계산` `전진선택법` `후진소거법`

---

## 정의/개념

다중 회귀분석 사용 독립변수 간 상관관계 높아 오버피팅 발생 부정적 영향 현상

---

## 개념도

```
독립변수 A          독립변수 B
    ↕   높은 결정계수(R²) 값   ↕
  독립변수A와 독립변수B 간의 높은 상관관계
        - 높은 상관관계
        - 선형 형태의 분포
```

### 문제점

- 독립변수 간의 상관관계가 강하여 결정계수(R²) 값이 높아 과최적합(Over-fitting) 발생
- 독립변수 간의 높은 상관관계로 인하여 종속변수와 유의한 상관관계가 나타나더라도 결과값의 왜곡발생 가능

---

## 해결방안

| 구분 | 해결방안 | 핵심 |
|:-----|:---------|:-----|
| **문제 확인** | 분산팽창요인(VIF) 계산 | 10 이상 문제, 30 초과 심각한 문제 |
| | 독립변수간 상관관계 분석 | 상관계수 0.9 이상 다중공선성 문제 |
| **해결방안** | 선행연구 수행 | 사전 정보 이용 |
| | 문제 독립변수 제거 | VIF 값이 10이상 독립변수 제거 |
| | 자료 변경 | 관측값 확대 |
| | 독립변수 변경 및 추가 | 상관도 값 하락 유도 |

---

## 변수 선택 방법

> 다중 회귀 모델에서 다중공선성 문제 예방 위해 모델 설계 과정 독립변수 선택 중요

| # | 방법 | 핵심 |
|:--|:-----|:-----|
| 1 | **전진선택법** | Y절편만 있는 상수모형부터 시작해 독립변수 추가 |
| 2 | **후진소거법** | 독립변수 모두 포함 상태에서 가장 적은 영향 변수 하나씩 제거 |
