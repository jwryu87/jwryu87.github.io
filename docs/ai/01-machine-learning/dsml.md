## 핵심 키워드

`데이터 준비` `모델 학습/튜닝` `모델 스캐폴딩` `모델 배포`

---

## 정의/개념

Data Science를 통해 얻은 고품질 데이터를 ML 통해 활용 프로젝트

---

## 수명주기

| # | 수명주기 |
|:--|:---------|
| 1 | 프로젝트 계획 수립 및 착수 |
| 2 | 데이터 준비 및 학습환경 구축 |
| 3 | 모델 학습/튜닝 - 모델 스캐폴딩 |
| 4 | 모델 배포 및 생산 모니터링 |

---

## 실패원인

| 구분 | 실패 원인 |
|:-----|:---------|
| **데이터** | 데이터 보안, 낮은 품질 |
| **개발** | 플랫폼 충돌, 편향 모델 |
| **문화** | 거버넌스 부재, 잘못된 머신러닝 문화 |
| **비즈니스** | 비즈니스 이해 부족, 인력 수급 문제 |
