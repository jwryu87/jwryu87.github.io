## 정의/개념

다중회귀분석은 하나의 종속변수(Y)가 두 개 이상의 독립변수(X1, X2, ...)에 의해 어떻게 설명되는지 분석하는 통계 기법

---

## 다중회귀분석과 로지스틱 회귀 분석 비교

| 구분 | 다중회귀분석 | 로지스틱 회귀 분석 |
|:-----|:------------|:------------------|
| **개념** | 다수의 연속형 독립변수가 연속형 종속 변수에 미치는 영향을 검증하는 분석기법 | 연속형/범주형 독립변수가 범주형 종속 변수에 미치는 영향을 검증하는 분석기법 |
| **개념도** | 추세선 그래프 | Sigmoid 곡선 (0~1) |
| **수행 목적** | 예측 문제 해결<br>연속형 결과 변수 값 예측 | 분류 문제 해결<br>비연속형 결과 범주 예측 |

### 변수 척도

| 구분 | 다중회귀분석 | 로지스틱 회귀분석 |
|:-----|:------------|:-----------------|
| **독립변수** | 연속형 변수 | 연속형 변수 / 범주형 변수 |
| **종속변수** | 연속형 변수 | 범주형 변수 |
| **독립 변수 척도** | 등간 척도, 비율 척도 | 명목 척도, 서열 척도, 등간 척도, 비율 척도 |
| **종속변수 척도** | 등간 척도, 비율 척도 | 명목 척도, 서열 척도 |
