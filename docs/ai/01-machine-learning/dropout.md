# 드랍아웃(Drop Out)
{: .fs-8 }

6. 퍼셉트론
{: .label .label-purple }

---

## 핵심 키워드

`노드 비활성화` `Overfitting` `co-adaption` `droprate`

---

## 정의/개념

심층 신경망의 과적합을 방지하기 위해 은닉층의 일부 노드를 무작위로 비활성화 시켜 정규화(성능 일반화)하는 신경망 학습 기법

---

## 드랍아웃 기술

### 학습 단계

| 구분 | 동작원리 | 설명 |
|:-----|:--------|:-----|
| **Dropout Rate 입력** | | 드랍아웃의 하이퍼 파라미터로 입력된 수치만큼 노드가 제거될지 말지를 랜덤하게 결정<br>0.5가 입력되면 50%확률로 노드 비활성화를 결정 |
| **노드 비활성화** | | 은닉층의 임의의 노드를 선택해 확률 P 기준으로 비활성화 |
| **신경망 학습** | | 임의 노드가 비활성화 된 상태에서 학습을 수행 |
| **오류 역전파** | | 노드 비활성화를 다시하여 반복해 학습을 수행 |

### 테스트 단계

| 구분 | 동작원리 | 설명 |
|:-----|:--------|:-----|
| **테스트 수행** | | 비활성화된 노드를 복원하여 확률 P와 가중치 W를 곱하여 연산 |

---

## 드랍아웃 유형

| 유형 | 설명 |
|:-----|:-----|
| **Fast Dropout** | 드랍아웃은 매번 무작위로 노드를 선택하기 때문에 속도가 매우 느려 노드를 가우시안 마스크로 추출해 적용 |
| **Ad-hoc Dropout** | Fast Dropout과 비슷하지만 가우시안 분포 대신에 0에서 1 사이에서 균일 분포로 추출한 마스크를 사용 |
| **DropConnect** | 드랍아웃이 노드를 비활성화 하는 것이라면 DropConnect는 가중치를 비활성화 시키고 노드는 그대로 두는 방법 |
