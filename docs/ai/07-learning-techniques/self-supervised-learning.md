# 자기지도학습(Self-supervised Learning)
{: .fs-8 }

학습 기법
{: .label .label-green }

---

## 핵심 키워드

`프리텍스트 태스크(pre-text task)` `다운스트림 태스크(downstream task)` `전이학습` `오토인코더` `GAN기반` `문맥기반` `대조학습기반`

---

## 정의/개념

다량의 레이블이 없는 원데이터로부터 데이터 부분들의 관계를 통해 레이블을 자동으로 생성하여 지도학습에 이용하는 비지도학습 기법

---

## 메커니즘

```
┌─────────────┐    ┌─────────────┐    ┌────────────┐    ┌─────────────┐    ┌─────────────┐
│  Unlabeled  │ →  │  Pre-text   │ →  │ Knowledge  │ →  │   Labeled   │ →  │ Downstream  │
│   Dataset   │    │    Task     │    │  Transfer  │    │   Dataset   │    │    Task     │
└─────────────┘    └─────────────┘    └────────────┘    └─────────────┘    └─────────────┘
        │                                                       │
        └───────── Self-supervised ─────────────────────────────┴───── Supervised ─────
```

---

## 자기지도학습의 메커니즘과 종류

| 구분 | 단계 | 설명 |
|:-----|:-----|:-----|
| **학습 유형** | 프리텍스트 테스크 (pre-text task) | 레이블이 있는 데이터를 대상으로 하는 작업으로, 데이터의 핵심 표현을 추출하는 레이어(feature extraction layer)를 학습 |
| | 다운스트림 테스크 (downstream task) | 연구자가 최종적으로 원하는 이미지 분류, 물체 인식과 같은 작업을 소량의 레이블 데이터만으로 수행 |

---

## 지도학습 vs 자기지도학습

| 구분 | 지도학습 | 자기지도학습 |
|:-----|:--------|:-----------|
| **개념** | 레이블링 된 데이터셋으로 인공지능 모델을 학습하는 기법 | 레이블이 없는 원데이터로부터 데이터 부분들의 관계를 통해 레이블을 자동으로 생성하여 지도학습에 이용하는 비지도학습 기법 |
| **목적** | 학습데이터 기반의 예측모델 생성 | 데이터 편향성, 시간/노동력 소모 감소 |
| **레이블** | 사전에 정의된 레이블 보유<br>- 수동 레이블링 | 데이터 관계를 통해서 레이블 자동 생성<br>- 자동레이블링 |
| **효율성** | 많은 시간 및 노동력 필요 | 신규 데이터셋 구축 비용 및 시간 절감 |
| **편향성** | 사람의 개입으로 편향성 문제 보유 | 사람의 개입이 없어 편향성 최소화 |
| **유형** | 회귀(Regression), 분류(Classification) | 영상 재건/생성 기반, 문맥기반, 대조학습 기반 |

---

## 자기지도학습 유형

| 유형 | 설명 |
|:-----|:-----|
| **오토인코더 기반** | 입력 데이터 압축 후 복원하며 특징 학습 |
| **GAN 기반** | 생성자-판별자 구조로 데이터 생성 |
| **문맥 기반** | 이미지 위치, 순서 예측 등 문맥 학습 |
| **대조학습 기반** | 유사/비유사 샘플 구분하며 표현 학습 |
