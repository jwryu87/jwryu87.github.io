# 능동학습(Active Learning)
{: .fs-8 }

학습 기법
{: .label .label-green }

---

## 핵심 키워드

`오토 라벨링` `QBC` `Oracle` `큐레이터` `어노테이터`

---

## 정의/개념

데이터 일부 학습 시킨 후, 나머지 데이터를 라벨링하는 오토 라벨링 기법

> 라벨링에서 발생하는 오차를 인간이 계속 교정하면서 학습을 강화

---

```
    ┌─────────────┐        ┌─────────────┐

          ▲                       ▼

    ┌─────┴─────┐
    │   Oracle   │◀───────────────────────┐

    └───────────┘      Algorithm          │
                            │              │
                            ▼              │
                   ┌──────────────┐        │
                   │  unLabeled   │────────┘
```

## 프로세스

| # | 프로세스 | 핵심 | 특징 |
|:--|:--------|:-----|:-----|
| 1 | **모델 학습** | Labeled Data | 라벨링 된 데이터 일부 학습 |
| 2 | **데이터 선별** | QBC | 2개 이상 학습 모델 사용 시 불일치 높은 데이터 선별 |
| 3 | **데이터 교정** | Oracle | 큐레이터, 어노테이션 사용 |
| 4 | **모델 평가** | Precision, Accuracy | 정확도, 정밀도 측정 |

> **QBC (Query by Committee)**: 기계 학습에서 모델의 불확실성을 줄이기 위한 샘플 선택 기법 중 하나

---

## 능동학습 쿼리 전략

| 전략 | 설명 |
|:-----|:-----|
| **불확실성 샘플링** | 모델이 가장 불확실한 샘플 선택 |
| **QBC (Query by Committee)** | 여러 모델의 예측 불일치가 큰 샘플 선택 |
| **Expected Model Change** | 가장 큰 모델 변화를 유발하는 샘플 선택 |
| **Density-weighted** | 대표성 있는 샘플 우선 선택 |

---

## Active Learning vs Passive Learning

| 구분 | Active Learning | Passive Learning |
|:-----|:----------------|:-----------------|
| **데이터 선택** | 능동적 선택 | 무작위 샘플링 |
| **라벨링 비용** | 낮음 | 높음 |
| **효율성** | 높음 | 낮음 |
| **적용 상황** | 라벨링 비용이 높을 때 | 데이터가 충분할 때 |
