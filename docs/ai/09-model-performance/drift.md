# 컨셉 드리프트(Concept Drift), 데이터 드리프트(Data Drift)
{: .fs-8 }

모델 성능
{: .label .label-yellow }

---

## 핵심 키워드

- **컨셉 드리프트**: 정답 라벨 관계성 변화, 개념 변화
- **데이터 드리프트**: 배포 환경 변화, 통계적 분포 차이

---

## 정의/개념

- **컨셉 드리프트**: 머신러닝에서 입력 데이터로부터 예측하려고 하는 정답 라벨의 관계성이 모델 훈련때와 비교하여 변경되어 모델 예측 저하 현상
- **데이터 드리프트**: 모델 훈련 시 입력 데이터의 통계적 분포와 배포 환경에서 입력 데이터의 통계적 분포 차이로 인한 편향으로 모델 예측 저하 현상

---

## 컨셉 드리프트

```
                     Margin           Original
                                       Margin
                        ╲              ╱
                         ╲            ╱
        ┌─────────────────╲──────────╱─────────────────┐
        │                  ╲        ╱                  │
        │    Concept        ╲      ╱       Class 0     │
        │     Drift    ●●●●  ╲    ╱   ○○○○            │
        │             ●●●●    ╲  ╱   ○○○○              │
        │            ●●●●      ╲╱   ○○○○               │
        │                       ╲  ○○○○                │
        │    Class 1             ╲                     │
        └─────────────────────────────────────────────┘
```

> 사회나 기술발전에 따라 정답라벨의 개념이 변화하여 학습모델의 예측 정확도가 저하됨

## 데이터 드리프트

```
                         입력데이터
                          분포변화
                            │
                            ▼
    ┌─────────────┐    ┌─────────────┐
    │   훈련환경   │    │  배포환경   │
    └──────┬──────┘    └──────┬──────┘
           │     Data Drift   │
           │ ─────────────────│
           │    (반년후)       │
           ▼                  ▼
    ┌─────────────┐    ┌─────────────┐
    │   훈련환경   │    │  배포환경   │
    │    고기능    │    │   성능감소  │
    └─────────────┘    └─────────────┘
```

> 입력 데이터의 분포가 훈련때와 실제 배포환경에서 변화하여 데이터 편향 증가로 인해 발생

---

## 컨셉 드리프트 vs 데이터 드리프트

| 비교 항목 | 컨셉 드리프트 | 데이터 드리프트 |
|:---------|:------------|:--------------|
| **원인** | 정답라벨의 개념 변화 (데이터와 라벨의 관계성 변화) | 훈련과 배포환경의 입력데이터 분포 변화 (입력데이터의 분포 변화) |
| **사례** | 금융사기 예측모델에서 금융사기의 정의가 바뀐 경우 | 계절성에 따라 여름에 효과 있는 모델이 겨울에는 성능이 저하되는 경우 |
| **해결방안** | - Online Learning<br>- Feature dropping | - 드리프트 모니터링<br>- 모델의 재학습, 재배포 |

> 이외에도 Label Drift, Prediction drift 등이 존재하며 MLOps를 통해 지속적 모델관리가 필요

## 드리프트 탐지 방법

| 방법 | 설명 |
|:-----|:-----|
| **Statistical Tests** | KS 검정, Chi-squared 검정 |
| **모니터링** | 성능 지표 추적, 분포 시각화 |
| **Window 기반** | 슬라이딩 윈도우로 변화 탐지 |
