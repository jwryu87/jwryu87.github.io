## 정의/개념

컴퓨터과학과 인공지능과 언어학이 합쳐져서 기계가 인간의 자연어를 이해하고 자연어로 상호 작용하기 위한 인공지능 기술

## 구성요소

### NLU (Natural Language Understanding) - 자연어 이해

| 세부 구성요소 | 설명 |
|:-------------|:-----|
| **One-hot 인코딩** | 자연어로 표현된 단어를 벡터화하는 과정<br>- K 개의 단어 중 해당 단어의 인덱스만 1로 표현 |
| **Word Embedding** | 여러 문장을 모델에 제공하고 문장의 문맥을 통해 단어의 의미를 학습시키는 방식<br>- CBOW(Continuous Bag of Words), Skip-gram 방식 |
| **Word2vec** | Word Embedding 통해 단어의 유사성을 제공하는 기법 |

### NLG (Natural Language Generation) - 자연어 생성

| 세부 구성요소 | 설명 |
|:-------------|:-----|
| **답변생성** | 사용자에게 원하는 질문의 답변 위해 상황에 적합한 자연어로 변환하는 과정 |
| **문장계획** | 질문에 적합한 자연어 문법을 계획 및 생성 |
| **어휘선택** | 생성된 문장에서 구문(명사, 동사, 형용사 및 부사)을 선택 |
| **문장생성** | 구문의 규칙에 따라 올바른 문장 텍스트를 만드는 과정 |
| **형태생성** | 상황에 적합하지 않은 표현 등을 검출, 최종 문장 확인 과정 |

---

## NLU vs NLG 비교

| 구분 | NLU (자연어 이해) | NLG (자연어 생성) |
|:-----|:-----------------|:-----------------|
| **방향** | 자연어 → 기계 이해 | 기계 → 자연어 출력 |
| **목적** | 의미, 의도 파악 | 텍스트 생성 |
| **기술** | 형태소 분석, 구문 분석 | 문장 계획, 어휘 선택 |
| **활용** | 챗봇 이해, 검색 | 답변 생성, 요약 |

---

## NLP 주요 태스크

| 태스크 | 설명 |
|:-------|:-----|
| **토큰화(Tokenization)** | 텍스트를 단어/문장 단위로 분리 |
| **형태소 분석** | 단어의 형태소 단위 분석 |
| **개체명 인식(NER)** | 고유명사, 날짜 등 개체 인식 |
| **품사 태깅(POS)** | 단어의 품사 부착 |
| **감성 분석** | 텍스트의 긍정/부정 판별 |
| **기계 번역** | 언어 간 번역 |
