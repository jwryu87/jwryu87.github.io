# 알고리즘 성능평가
{: .fs-8 }

머신러닝 검증
{: .label .label-blue }

---

## 핵심 키워드

`시간복잡도` `공간복잡도` `빅오 표기법` `알고리즘 효율성`

---

## 정의/개념

알고리즘 수행 시 필요로 하는 시간 및 공간에 대한 지표를 기준으로 알고리즘 성능을 판단하는 프로세스

---

## 알고리즘 성능평가의 유형

| 구분 | 유형 | 설명 |
|:-----|:-----|:-----|
| **성능분석** | 알고리즘 복잡도 분석 | - 직접 구현하지 않아서 수행 시간을 분석하는 것<br>- 알고리즘이 수행하는 연산의 횟수 측정 비교<br>- 일반적으로 연산의 횟수는 n 의 함수<br>- 시간복잡도 분석: 수행시간 분석<br>- 공간복잡도 분석: 수행 시 필요로 하는 메모리 공간 분석 |
| **성능측정** | 수행시간 측정 | - 알고리즘의 실제 수행시간을 측정하는 것<br>- 알고리즘의 실제 구현물이 필요<br>- 동일한 하드웨어 사용 필요 |

> 알고리즘 분석에는 시간 복잡도(Time Complexity)와 공간 복잡도(Space Complexity)를 이용

---

## 시간복잡도와 공간복잡도

```
                    ┌─────────────────┐
                    │   알고리즘 효율성   │
                    └────────┬────────┘
                             │
            ┌────────────────┴────────────────┐
            │                                 │
            ▼                                 ▼
    ┌───────────────┐                 ┌───────────────┐
    │    시간요인     │                 │    공간요인     │
    └───────┬───────┘                 └───────┬───────┘
            │                                 │
            ▼                                 ▼
    - 시간복잡도                        - 공간복잡도
      (수행시간)                         (사용 메모리 공간)
```

---

## 시간복잡도 vs 공간복잡도

| 구분 | 시간복잡도(Time Complexity) | 공간복잡도(Space Complexity) |
|:-----|:--------------------------|:---------------------------|
| **정의** | 알고리즘에 사용되는 **연산횟수의 총량** | 알고리즘에 사용되는 **메모리 공간의 총량** |
| **평가방법** | 입력크기의 값에 대하여, 단위연산을 몇 번 수행하는지를 계산함으로써, 알고리즘의 수행시간을 평가 | 알고리즘 수행에 필요한 메모리의 사용량을 평가 |
| **평가척도** | 알고리즘 수행시간 | 메모리 공간 사용량 |
| **복잡도 계산** | 연산의 횟수 | - 고정영역(프로그램 사이즈)+<br>- 가변영역(데이터 사이즈) |

---

## 시간복잡도 표기법

| 표기법 | 의미 | 설명 |
|:-------|:-----|:-----|
| **O (Big-O)** | 상한 | 최악의 경우 실행 시간 |
| **Ω (Omega)** | 하한 | 최선의 경우 실행 시간 |
| **Θ (Theta)** | 평균 | 평균적인 실행 시간 |

---

## 주요 시간복잡도

| 복잡도 | 명칭 | 예시 알고리즘 |
|:-------|:-----|:-------------|
| **O(1)** | 상수 시간 | 해시 테이블 접근 |
| **O(log n)** | 로그 시간 | 이진 탐색 |
| **O(n)** | 선형 시간 | 선형 탐색 |
| **O(n log n)** | 로그 선형 | 병합 정렬, 퀵 정렬 |
| **O(n²)** | 이차 시간 | 버블 정렬, 삽입 정렬 |
| **O(2ⁿ)** | 지수 시간 | 재귀 피보나치 |
| **O(n!)** | 팩토리얼 | 순열 생성 |

---

## 복잡도 증가 그래프

```
시간 ↑
     │                                    O(n!)
     │                              O(2ⁿ)  /
     │                         O(n²)    /
     │                    O(n log n)  /
     │              O(n)           /
     │       O(log n)           /
     │  O(1)                  /
     └────────────────────────────────→ 입력 크기(n)
```
