AI 기타

## 핵심 키워드

`일반 언어 처리 기술` `이미지 처리 기술` `음성 처리 기술` `데이터 마이닝 기술`

---

## 정의/개념

텍스트, 이미지, 음성, 표정 등의 다양한 모달리티를 동시에 받아들이고 사고하는 AI 모델

---

## 특징

```
        Modality1(text)                         Modality4(audio)
              ●●●●                                    ●●●
             ●●●●●                                   ●●●●
            ●●●●●●     Link                         ●●●●●     Modality6(speech)
             ●●●●●    type (1,3)                     ●●●●          ●●●
              ●●●●        Link                        ●●          ●●●●
                       type (2,3)   Link type (3,4)              ●●●●●
                            Link    Link type (3,5)               ●●●●
                         type (3,4)                                ●●
                              ●●●●●
                             ●●●●●●     Link type (5,6)
        Modality2           ●●●●●●●
         (Image)           ●●●●●●●●
          ●●●●            Modality3(sensor)
         ●●●●●                                  Modality5(meta-data)
        ●●●●●●      Link                              ●●●●
         ●●●●      type (3,6)                        ●●●●●
          ●●                                        ●●●●●●
                                                     ●●●●●
                                                      ●●●
```

| 구분 | 설명 |
|:-----|:-----|
| **정의** | 텍스트, 이미지, 음성, 표정 등의 다양한 모달리티를 동시에 받아들이고 사고하는 AI 모델 |
| **특징** | 다양한 감각 기관 상호 작용<br>- 시각, 청각 등 다양한 감각기관 상호 작용 통해 사람처럼 사고 |
| | 유의미한 정보 산출<br>- 서비스 분야별로 필요한 도메인 지식 포함하는 상황 인식 모델 구성 |
| | 인간의 삶에 융화<br>- 인간과 AI가 보다 자연스러운 의사소통을 수행 할 수 있도록 함 |

> 멀티 모달 인공지능 구축을 위해 다양한 기술들이 응용되고 있음

---

## 기술요소

| 기술요소 | 설명 | 세부기술 |
|:--------|:-----|:--------|
| **일반언어 처리 기술** | 단어 분리, 단어의 유형 라벨링, 키워드 검색, 유의어, 반의어, 정보의 분석 및 추출, 관계 추출, 일반언어 이해 | NLP, NLU, NLG<br>Word Embedding |
| **이미지 처리 기술** | 세분화, 이미지 이해, 얼굴 인식, 연령 및 성별 인식,, 글자 인식, 이미지 기반 검색, 이미지 기반 예측 분석 | Image Scaling<br>Filtering, Morphology |
| **음성 처리 기술** | 언어 모델, 음성 머신러닝 모델을 구축, hot word 자동 인식, 노이즈 필터링 | STT(Speech-To-Text)<br>Signal Processing |
| **데이터 마이닝 기술** | 빅데이터 수집 및 처리 KPI 예측, 빅데이터 기반 예측 분석, 머신러닝을 위한 자동 데이터 생성 | 회귀분석 시계열분석<br>클러스터링, 연관분석 |

---

## 멀티모달 AI 활용 예시

| 분야 | 활용 |
|:-----|:-----|
| **자율주행** | 카메라, 라이다, 레이더 데이터 융합 |
| **의료** | 영상, 텍스트, 수치 데이터 통합 분석 |
| **챗봇** | 텍스트 + 이미지 + 음성 이해 |
| **교육** | 음성, 표정, 텍스트 기반 학습 분석 |
