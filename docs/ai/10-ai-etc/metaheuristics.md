# 메타휴리스틱스(Metaheuristics)
{: .fs-8 }

AI 기타
{: .label .label-purple }

> 암기: **진군물** (진화알고리즘, 군집지능, 물리기반)

---

## 핵심 키워드

`진화알고리즘` `군집지능 알고리즘` `물리기반 알고리즘`

---

## 정의/개념

특정 문제에 한정되지 않고 어떠한 문제에 대해서도 범용적으로 대응할 수 있도록 설계된 상위수준의 휴리스틱 기법

> 메타휴리스틱스 알고리즘으로는 유전알고리즘, 개미군락최적화, 시뮬레이티드 어닐링, 타부서치 등을 많이 활용

---

## 알고리즘 상세

| 구분 | 알고리즘 | 설명 |
|:-----|:--------|:-----|
| **진화알고리즘 (Evolutionary Algorithms)** | 유전알고리즘 (GA, genetic algorithms) | 생물체가 환경에 적응하면서 진화해가는 모습을 모방하여 최적해를 찾아내는 알고리즘<br>- 선택→교차→변이→대체의 과정을 반복하여 적합도를 만족하는 최적해 탐색 |
| **군집지능 알고리즘 (Swarm Intelligence Algorithms)** | 개미군락최적화 (ACO, ant colony optimization) | 자연계에서 일어나는 개미의 행동양식을 모방한 최적탐색기법<br>- 페로몬을 통한 경로 최적화 |
| **물리기반 알고리즘 (Physics-based Algorithms)** | 시뮬레이티드 어닐링 (SA, simulated annealing) | 온도를 낮추거나 최적의 상태를 얻는 어닐링 과정을 모방하여 최적화 문제에 적용하는 알고리즘 |
| | 타부서치 (TS, tabu search) | 기억 과정을 모방하여 지역 최적점(local optimum)에 대한 정보를 저장 및 회피하여 전체 최적점(Global optimum)을 찾는 알고리즘 |

---

## 활용 분야

| 분야 | 활용 |
|:-----|:-----|
| **산업영역** | 일정관리, SCM, 공정 최적화 |
| **컴퓨터과학분야** | Internet Routing, Web page Clustering |
| **헬스케어** | RNA 구조 예측 |

> 각 메타휴리스틱 알고리즘의 단점을 상호보완하면서 장점을 결합하여 함께 적용 요함 (유전알고리즘+타부탐색)
