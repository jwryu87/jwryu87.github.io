# 협업필터링(Collaborative Filtering)
{: .fs-8 }

AI 기타
{: .label .label-purple }

---

## 핵심 키워드

`메모리 기반 접근법` `모델 기반 접근법` `사용자 기반` `아이템 기반` `유클리디안 유사도` `코사인 유사도` `피어슨 유사도` `자카드 유사도`

---

## 정의/개념

사용자의 과거 경험과 행동 방식에 기반하여 사용자와 아이템간 상호 관계를 분석하여 새로운 사용자와 아이템의 관계를 찾아주는 시스템

## 협업 필터링(Collaborative Filtering) 알고리즘

### 유사도 측정 방법

```
    유클리디안 유사도         코사인 유사도           피어슨 유사도         자카드 유사도
    
        상품 X                  상품 X                 •    •            ┌─────────┐
           │                      │                  •      •          │    ○    │
           │    •                 │   ╲              •        •        │  ╱   ╲  │
           │  •                   │    ╲                      •        │ ○     ○ │
           └──────── 상품 Y       └──────── 상품 Y                      └─────────┘
    
    두 지점간 벡터 거리 기준   두 벡터간 각도의         피어슨 상관계수 기준      교집합과 합집합 기준
                           코사인 값 기준
```

| 유사도 | 설명 |
|:-------|:-----|
| **유클리디안 유사도** | 두 지점간 직선 거리 기반 |
| **코사인 유사도** | 두 벡터간 각도의 코사인 값 기준 |
| **피어슨 유사도** | 피어슨 상관계수 기준 |
| **자카드 유사도** | 교집합과 합집합 기준 |

---

## 사용자 기반 vs 아이템 기반

| 구분 | 사용자 기반 | 아이템 기반 |
|:-----|:-----------|:-----------|
| **방식** | 유사한 사용자 찾아 추천 | 유사한 아이템 찾아 추천 |
| **예시** | "당신과 비슷한 취향의 사용자가 좋아한..." | "이 상품을 본 사람들이 함께 본..." |
| **장점** | 새로운 아이템 추천 가능 | 계산 효율성 높음 |
| **단점** | 확장성 문제 | Cold Start 문제 |
