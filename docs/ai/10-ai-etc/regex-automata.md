## 핵심 키워드

- **정규표현식**: 연결, 합집합, 반복
- **유한오토마타**: DFA, NFA

---

## 정의/개념

- **정규표현식**: 특정한 규칙을 가진 문자열의 집합을 표현하는데 사용하는 형식 언어
- **유한오토마타**: 컴퓨터의 프로그램과 전자논리회로를 설계하기 위해 이산적인 입력과 출력을 가지는 시스템의 수학적 모형

---

## 정규표현식 (Regular Expression)

| 구분 | 설명 |
|:-----|:-----|
| **개념** | 특정한 규칙을 가진 문자열의 집합을 표현하는 데 사용하는 형식 언어 |

### 표현

| 표현 | 기호 | 설명 |
|:-----|:-----|:-----|
| **기본 표현** | φ | 공집합, 어떤 문자열과도 매칭되지 않음<br>- {ε} = {} |
| | ε | 공집합 {ε}를 나타내는 정규표현<br>- 빈 문자열(길이가 0 인), 문자를 포함하지 않는 문자열 |
| | a ∈ V_T | L(a) = {a}, where a ∈ terminal symbol<br>- 정규 (a)는 단일문자로 된 정규표현, 알파벳에 속한 단일문자 a |
| **기본 연산** | 연결 (concatenation) | '*' 문자를 사용하거나 생략가능<br>- N1*N2, {01} = {01} |
| | 합집합 (Union) | '+' 문자를 사용하여 표현<br>- N1+N2, {0,1} = {0,1} |
| | 반복 (closure or Kleene star) | '*' 문자를 사용하여 표현<br>- N*, L* = { } ∪ L1∪L2 ∪ L3∪ L1n |

> 정규표현식과 유한오토마타는 상호 동치 관계로 변환 가능

---

## 유한오토마타 (Finite Automata)

| 구분 | 설명 |
|:-----|:-----|
| **개념** | 컴퓨터 프로그램과 전자논리회로를 설계하기 위해 이산적인 입력과 출력을 가지는 시스템의 수학적 모형 |
| **특징** | ① 단일상태 : 한번에 오로지 하나의 상태만 가짐<br>② 전이(Transition) : 한 상태에서 다른 상태로 변화하는 성질<br>③ 상태(State) : 진입동작(진입 시 수행 동작), 퇴장동작(퇴장 시 수행 동작) |
| **표현** | M = ( Q, Σ, δ, s0, F )<br>- Q : 상태의 공집합이 아닌 유한 집합<br>- Σ : 입력문자(유한하며, 비어있지 않은 기호의 집합) - δ : 상태전이 함수<br>- s0 : 초기상태이며, Q 의 원소<br>- F : 최종상태의 집합이며, Q 의 원소 |

---

## 유한오토마타 유형

### 결정적 유한 오토마타 (DFA, Deterministic Finite Automata)

모든 상태는 각각의 가능한 입력에 대해 정확히 하나의 변환된 상태를 가질 수 있는 유한 오토마타

```
                      a           b
            start ──▶ (A) ──▶ (B) ──▶ (C)
```

- 상태 A : a 입력시(→) B 상태로 전이
- 상태 B : a → B, b → C 상태로 전이
- ⇒ 입력 값에 따라 유일한 상태 결정

### 비결정적 유한 오토마타 (NFA, Non Deterministic Finite Automata)

입력은 주어진 상태에 대해 한가지 또는 여러 가지의 변환된 상태를 가질 수 있는 유한 오토마타

```
                               b
                      a    ┌──▶(C)
            start ──▶ (A) ──▶ (B)
                           └──▶ b
```

- A상태 : a 입력시 상태 B로 전이
- B상태 : b 입력시 B, C상태로 전이
- ⇒ 어떤 입력에 대해서 하나 이상의 상태가 가능한 유한 오토마타

> 그 외 ε-NFA(NFA With ε-transitions)유형이 있으며, NFA 는 DFA로 변환이 가능함.

---

## DFA vs NFA 비교

| 구분 | DFA | NFA |
|:-----|:----|:----|
| **결정성** | 확정적 | 비확정적 |
| **상태 전이** | 입력당 하나의 상태 | 입력당 여러 상태 가능 |
| **구현** | 단순 | 복잡 |
| **변환** | - | DFA로 변환 가능 |
