# 데이터 라벨링, 데이터 어노테이션
{: .fs-8 }

1.1 빅데이터 개념
{: .label .label-purple }

---

## 핵심 키워드

**데이터 라벨링:** `텍스트` `이미지` `음성`

**데이터 어노테이션:** `바운딩 박스` `세그멘테이션` `랜드마크` `폴리곤`

---

## 정의/개념

| 구분 | 정의 |
|:-----|:-----|
| **데이터 라벨링** | 다수의 어노테이션을 통칭, 데이터 확보 위해 이미지에 라벨 부착 기법 |
| **데이터 어노테이션** | 원천데이터의 인공지능 이해 위해 메타데이터를 태그 형식 및 데이터 셋 추가 기법 |

---

## 방식

### 데이터 라벨링(Data Labeling) 및 어노테이션(Annotation) 방식

| 데이터 유형 | 라벨링 기능 | 어노테이션 방식 |
|:-----------|:-----------|:---------------|
| **텍스트** | 텍스트 분류(Text Classification) | 클래스 라벨(단일, 다중) |
| | 개체명 인식(Named Entity Recognition) | 단어(구문) 라벨 |
| | 관계·의존성 정의(Relation-Dependencies) | 단어(구문) 라벨링 및 두 단어 사이의 관계 |
| **이미지** | 이미지 분류(Image Classification) | 클래스 라벨(단일, 다중) |
| | 객체 인식(Object Recognition) | 바운딩 박스(사각형)<br>폴리곤(다각형) |
| **동영상** | 동영상 분류(Video Classification) | 클래스 라벨(단일, 다중) |
| | 객체 인식(Object Recognition) | 바운딩 박스(사각형)<br>키 포인트(점) |
| | 객체 추적(Object Tracking) | 폴리곤(다각형)<br>폴리라인(선) |
| **오디오** | 오디오 분류(Audio Classification) | 클래스 라벨 |
| | 오디오 세그멘테이션(Audio Segmentation) | 클래스 라벨 |
| | 음성인식(음성 > 텍스트 변환)<br>(Speech to Text) | 텍스트 전사 |

---

## 어노테이션 유형

| 유형 | 설명 | 활용 분야 |
|:-----|:-----|:---------|
| **바운딩 박스** | 객체를 사각형으로 표시 | 객체 탐지, 자율주행 |
| **폴리곤** | 다각형으로 정밀 영역 표시 | 이미지 세그멘테이션 |
| **키 포인트** | 특정 점 위치 표시 | 자세 추정, 얼굴 인식 |
| **세그멘테이션** | 픽셀 단위 분류 | 의료 영상, 자율주행 |
| **폴리라인** | 선 형태 표시 | 차선 인식 |
