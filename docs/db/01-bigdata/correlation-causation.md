# 빅데이터 분석 상관관계(Correlation), 인과관계(Causation)
{: .fs-8 }

1.3 분석 기법
{: .label .label-green }

---

## 핵심 키워드

**상관관계:** `변화` `선형` `비선형` `피어슨`

**인과관계:** `원인/결과` `직접` `간접` `회귀 분석` `공분산`

---

## 정의/개념

| 구분 | 정의 |
|:-----|:-----|
| **상관관계** | 서열 척도, 등간 척도, 비율 척도, 관련성 분석 기법 |
| **인과관계** | 독립변인, 종속변인 통계적 방법 |

---

## 개념 비교

| 비교 | 상관관계(Correlation) | 인과관계(Causation) |
|:-----|:---------------------|:--------------------|
| **정의** | 서열 척도, 등간 척도, 비율 척도로 측정된 변수들 간의 관련성 정도를 확인하기 위한 분석 기법 | 독립 변인이 종속 변인에 영향을 미치는지에 대해서 둘 간의 관계를 분석하는 통계적 방법 |
| **개념도** | 현상A ←관련성→ 현상B | 원인 → 결과 |

---

## 상세 비교

| 비교 | 상관관계(Correlation) | 인과관계(Causation) |
|:-----|:---------------------|:--------------------|
| **관계 분석 목적** | 두 가지 현상 가운데 한쪽이 변화되면 다른 현쪽도 따라서 변화하는 관계 분석 | 한 현상은 다른 현상의 원인이 되고 그 다른 현상은 먼저의 현상의 결과가 되는 관계 분석 |
| **관계 분석** | 두 변수 간 선형적 또는 비선형적 관계 | 하나의 사건이 다른 사건을 일으킬 때 그 두 사건의 관계 |
| **변수간 관계** | 상호 의존성 또는 상호 관련성 | 원인과 결과 |
| **변수 관계도** | 변수 y ↔ 변수 x (상관계수) | 독립변수 → 종속변수 (원인 → 결과) |
| **관계 유형** | 양의 관계, 음의 관계 | 직접 인과 관계, 간접 인과 관계 |
| **도출 모형** | 머신 러닝 기반 예측 모형 | 기준선 모형 |
| **관리 지표** | 선행지표, 후행지표 | 선행 지표 |
| **분석기법** | 피어슨(Pearson) 상관계수<br>스피어만 순위(Spearman Rank) 상관계수<br>Kendall Tau(켄달 Tau) | 회귀 분석, 경로 분석, 요인 분석<br>공분산구조 분석 |

---

## 주요 특성

{: .highlight }
> 상관 관계는 인과 관계의 필요 조건이 될 수 있으나, 충분조건이 될 수 없음

- 두 변수 간 상관관계가 있다고 해서 반드시 인과관계가 있는 것은 아님
- 인과관계를 증명하려면 실험 설계나 추가적인 통계적 검증 필요
