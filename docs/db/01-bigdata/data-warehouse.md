# 데이터 웨어하우스 (Data Warehouse)
{: .fs-8 }

1.2 처리·플랫폼
{: .label .label-yellow }

---

## 핵심 키워드

`정형 데이터` `Schema-On-Write` `DM` `OLAP` `ETL`

---

## 정의/개념

정형 데이터 활용, DM 구성 및 OLAP, ETL 활용 데이터 저장소

---

## Data Warehouse vs Data Lake 비교

| 비교 | 데이터 웨어하우스 | 데이터 레이크 |
|:-----|:-----------------|:-------------|
| **데이터 유형** | 정형 데이터 | 정형/반정형/비정형 데이터 |
| **스키마** | Schema-on-Write | Schema-on-Read |
| **데이터 처리** | Pre-Processing | Post-Processing |

---

## 관계

```
┌──────────────────┐     비정형 데이터      ┌─────────────┐    데이터      ┌─────────────┐
│ 데이터 웨어하우스 │ ──── 분석 요구 ────▶ │ 데이터 레이크 │ ── 사일로 해결 ─▶│ 데이터 패브릭 │
└──────────────────┘                      └─────────────┘                └─────────────┘
     • Biz. 지원              • 빅데이터 분석 지원            • Biz. 민첩성 확보
     • 정형 데이터            • 정형 + 비정형 데이터          • 메타 데이터 / Catalog 관리
     • 데이터 선 정제         • 실시간 데이터 처리            • 데이터 거버넌스/가상화
```

          │                       │
          ▼                       ▼

    ┌─────┴─────┐                     │
