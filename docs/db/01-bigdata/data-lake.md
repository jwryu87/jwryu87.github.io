---

## Data Warehouse vs Data Lake 비교

| 비교 | 데이터 웨어하우스 | 데이터 레이크 |
|:-----|:-----------------|:-------------|
| **데이터 유형** | 정형 데이터 | 정형/반정형/비정형 데이터 |
| **스키마** | Schema-on-Write | Schema-on-Read |
| **데이터 처리** | Pre-Processing | Post-Processing |

---

## 특징

| 특징 | 설명 |
|:-----|:-----|
| **빅데이터 분석 지원** | 대용량 비정형 데이터 분석 가능 |
| **정형 + 비정형 데이터** | 다양한 형태의 데이터 저장 |
| **실시간 데이터 처리** | 스트리밍 데이터 처리 지원 |
| **Schema-on-Read** | 데이터 읽을 때 스키마 적용 |
