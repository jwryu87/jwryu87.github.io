## 정의/개념

데이터 저장된 위치 관계없이 적시에 데이터 액세스 통합 프레임워크

---

## 아키텍처

```
┌───────────────────────────────────────────────────────────┐
│                    Data Consumers                          │
├─────────────────┬────────────────────────────────────────┤
│  Data Pipeline  │            Data Ops                     │
│  ┌───────────┐  │  ┌─────────────────┐  ┌───────────────┐│
│  │ 자동 수집  │  │  │      ④         │  │      ⑥       ││
│  ├───────────┤  │  │ Recommendation  │  │   Metadata    ││
│  │ 추천엔진  │  │  │    Engine       │  │  Activation   ││
│  ├───────────┤  │  ├─────────────────┤  └───────────────┘│
│  │ 메타데이터 │  │  │      ⑤         │                   │
│  ├───────────┤  │  │Data Preparation │                   │
│  │ 데이터 검색│  │  │and Data Integration                │
│  ├───────────┤  │  ├─────────────────┤                   │
│  │데이터 카탈로그│  │ │      ②         │                   │
│  └───────────┘  │  │Semantics Knowledge Graph            │
│                 │  ├─────────────────┤                   │
│                 │  │      ①         │                   │
│                 │  │Augmented Data Catalog               │
├─────────────────┴──┴─────────────────┴───────────────────┤
│                    Data Sources                           │
└───────────────────────────────────────────────────────────┘
```

- 데이터 소스의 자동화된 플로우 및 파이프라인을 통해 데이터 액세스를 지원하는 구조

---

## 구성요소

| 구분 | 구성요소 | 핵심 |
|:-----|:---------|:-----|
| **Data<br>Source** | 증강 데이터 카탈로그 | 대용량 실시간 로그 처리 |
| | 시멘틱 지식 그래프 | DB Scale-out 제공 위한 읽기<br>처리량 향상 |
| | 메타데이터 활성화 | HTTPS 형태 API |
| **Data<br>Consumer** | 추천 엔진 | Data Lake 최적화 |
| | 데이터 준비 및 전달 | 키 값과 데이터만 저장 |
| | DataOps | 대량 RDB를 HDFS 전송 |
