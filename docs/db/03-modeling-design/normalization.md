## 핵심 키워드

---

## 정의/개념

이상현상이 발생하는 릴레이션을 무손실 분해하여 이상현상을 없애는 과정

---

## 정규화 (Normalization)

| 구분 | 설명 |
|:-----|:-----|
| **개념** | 이상현상이 발생하는 릴레이션을 무손실 분해하여 이상현상을 없애는 과정 |
| **원칙** | 정보의 무손실, 데이터 중복성 감소, 분리의 원칙 |

### 정규화 단계

| 정규화 | 설명 |
|:-------|:-----|
| **1정규화** | 복수의 속성값을 갖는 속성 분리 |
| **2정규화** | 주식별자에 종속적이지 않은 속성 분리(부분함수종속 제거) |
| **3정규화** | 속성에 종속적인 속성 분리(이행함수종속 분리) |
| **BCNF** | 다수의 주식별자 분리 |
| **4정규화** | 다가 종속(Multi-Valued Dependency) 속성 분리 |
| **5정규화** | 결합 종속(Join Dependency)일 경우 두 개 이상의 N개로 분리 |

---

## 역정규화 (De-normalization)

| 구분 | 설명 |
|:-----|:-----|
| **개념** | 정규화 된 릴레이션에서 자주 참조가 발생하여 성능이 저하가 우려될 때 중복을 일정부분 허용하여 구조를 재조정하는 과정 |

### 역정규화 유형

| 유형 | 설명 |
|:-----|:-----|
| **테이블 역정규화** | 테이블병합: 1:1/1:M 관계 테이블병합, 슈퍼/서브타입 테이블병합<br>테이블분할: 수직분할, 수평분할<br>테이블추가: 중복/통계/이력/부분테이블 추가 |
| **컬럼 역정규화** | 중복/파생/이력테이블 칼럼추가<br>PK에 의한 칼럼 추가, 응용시스템을 위한 칼럼 추가 |
| **관계 역정규화** | 중복관계 추가 |
