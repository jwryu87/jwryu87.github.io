# 함수적 종속성 & 암스트롱 공리
{: .fs-8 }

3.1 정규화
{: .label .label-purple }

> 암기: **반확이/합분유**

---

## 핵심 키워드

`반사의 공리` `확대의 공리` `이행의 공리` `합집합의 성질` `분해의 성질` `유사 이행적 성질`

---

## 정의/개념

주어진 릴레이션 R에 대해 X, Y, Z라는 속성의 집합이 주어졌을 경우 여러가지 함수종속(FD, Functional Dependency)의 성질을 유도해 폐포(F+)를 도출할 수 있는 추론 규칙

---

## 가. 기본 규칙

| 규칙 | 개념도 | 설명 |
|:-----|:-------|:-----|
| **반사의 공리** | X → Y | Y가 X의 부분 집합이면, X → Y이다 |
| **확대의 공리** | X, Z → Y, Z | 만약 X → Y이면, XZ → YZ이다 |
| **이행의 공리** | X → Y → Z | 만약 X → Y이고 Y → Z이면 X → Z이다 |

---

## 나. 부가 규칙

| 규칙 | 개념도 | 설명 |
|:-----|:-------|:-----|
| **합집합의 성질** | X → Y, Z | 만약 X → Y이고 만약 X → Z이면<br>X → YZ이다 |
| **분해의 성질** | X → Y, Z 분리 | X → YZ이면 X → Y이고 X → Z이다 |
| **유사 이행적 성질** | X → Y, Z → W | 만약 X → Y이고 YZ → W이면,<br>XZ → W이다 |

- 암스트롱 공리를 이용해서 함수 종속성 집합의 폐포(F+)를 도출 가능

---

## 암스트롱 공리의 폐포(F+) 알고리즘

```
X⁺ := X;
Repeat
    Old X⁺ := X⁺;
    For each functional dependency Y → Z in F do
        If Y⊆X⁺ then X⁺ := X⁺∪Z;
Until (old X⁺ := X⁺);
```

- 함수적 종속 관계의 집합 F에서 특정 속성 X가 함수적으로 결정하는 속성들의 집합
