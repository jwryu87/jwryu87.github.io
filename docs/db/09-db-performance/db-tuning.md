---

## 설계 단계 튜닝 기법

| 구분 | 튜닝 기법 | 핵심 |
|:-----|:---------|:-----|
| **테이블 측면** | 반정규화 | 테이블, 칼럼, 관계 |
| | 파티셔닝 | 논리적 테이블 분할 |
| | 이력 테이블 관리 | 이력 테이블 시작, 종료, 현재 진행 상태 |
| | 복제 테이블 생성 | 스냅샷 이용 |
| **인덱스 측면** | 적정 인덱스 지정 | 인덱스 개수 지정 |
| | 인덱스 키 규칙 | 키 구성 칼럼 순서 조정 |
| | FK 잠금 회피 | FK 인덱스 생성 |

---

## 튜닝 유형별 특징

| 유형 | 핵심 내용 |
|:-----|:----------|
| **DB 설계 튜닝** | 테이블, 인덱스 구조 최적화 |
| **DB 환경 튜닝** | 메모리, 디스크 I/O, CPU 최적화 |
| **SQL 튜닝** | 쿼리 최적화, 실행계획 개선 |
