# 데이터 결측값 (Missing Value)
{: .fs-8 }

7.4 결측값 처리
{: .label .label-purple }

> 암기: **완무비**

---

## 핵심 키워드

**유형:** `완전 무작위 결측` `무작위 결측` `비 무작위 결측(완무비)`

**제거:** `목록` `단일값` `완전`

**단일대체:** `평균` `Hot/Cold` `영역적`

**다중 대체:** `혼합`

**예측모델:** `기계적 패턴` `SVM` `회귀 분석`

---

## 정의/개념

실험이나 조사 시 무관심, 부주의 등의 이유로 관측되어야 할 데이터를 얻지 못한 값

---

## 유형

| 종류 | 설명 |
|:-----|:-----|
| **완전 무작위 결측** | 변수상 결측 데이터 관측, 미관측 된 다른 변수와 연관 없음 |
| **무작위 결측** | 결측 데이터가 다른 변수와 연관되나 자체 비관측값과 연관 없음 |
| **비 무작위 결측** | 결측값이 완전 무작위, 무작위 결측이 아니면 비 무작위 결측 |

{: .highlight }
> 대부분 데이터는 완전 무작위 결측과 무작위 결측이라 가정하며, 데이터 분석을 위해 처리가 요구됨

---

## 처리 기법

### 기본적 결측값 처리, 제거 및 단일 대체

| 분류 | 주요 기법 | 설명 |
|:-----|:---------|:-----|
| **제거** | 목록 삭제 | 결측치 있는 행은 전부 삭제 |
| | 단일값 삭제 | 행 전체가 결측치인 행만 삭제 |
| | 완전 제거법 | 하나의 변수라도 결측치 존재시 분석 대상 제외 |
| **단일 대체** | 평균 대체 | 관측 자료의 평균값, 중앙값으로 대체<br>쉽게 적용가능하나 편향 발생 가능성 존재 |
| | 단순 확률 대체 | 평균 대치 관측자료 토대로 적정 확률값 부여 대체<br>(핫덱) 비슷한 성향 응답자의 자료로 대체 |
| | (콜드덱) 외부 출처 또는 이전의 연구자료에서 대체 |
| | 영역적 대체 | 논리적 제약과 다른 기록 의거 결측값 논리적 유추 |

{: .note }
> 데이터 제거가 과정상 쉬우나, 데이터 부족문제로 단일 대체 적용 다수

### 복합적 결측값 처리, 다중 대체 및 예측 모델

| 분류 | 주요 기법 | 설명 |
|:-----|:---------|:-----|
| **다중 대체** | 혼합 방법 | 몇 가지 방법을 혼합하여 다중 대체 처리<br>결측치 대체 자료를 분석 후 결과 혼합하여 한번 이상 대체 |
| **예측 모델** | 기계적 패턴 추출 | 결측치들이 패턴을 가진다고 가정 후 진행<br>결측 없는 컬럼 활용 패턴 추출 후 컬럼 예측 |
| | 분석 기술 활용 | 회귀 분석 기술 활용 또는 SVM과 기계학습으로 대체 |

{: .highlight }
> 예측 모델은 결측치 특성이 완전히 관찰되지 않는다고 가정하고 진행되며, 타 모델 대비 오버헤드 큼

---

## 적용기술

| 기법 | 적용기술 |
|:-----|:---------|
| **삭제** | 극단간 절단(가변균등 전 의한 절단, 하상단 % 를 이용한 제거) |
| **대체법(Imputation)** | 단일대체법(평균대체법, 연역적 대체법, 회귀 대체 법, 위치대로 대체법, Hot-Deck대체법)<br>다중대체법 |
| **변환(Transformation)** | 극단간 조정(Winsorizing, 상하한 감 변하이나는 값을 하한-상한으로 변경하여 활용) |
| **박스플롯(Box-Plot)활용** | 박스플롯 해석을 통해 사분위수를 이용해 제거<br>박스플롯의 수염(Whiskers, Q1과 Q3으로부터 IQR의 1.5배 내에 있는 가장 멀리 떨어진 데이터까지 이어진 선)보다 바깥쪽 데이터를 이상값으로 분류 |
| **분류(Classification)** | 그룹별 이상치 판별<br>개괄 분석을 실행하여 처리<br>각각의 그룹에 대한 통계적 모형을 생성하고, 결과를 결합하는 방법 |
