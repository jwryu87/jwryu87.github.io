# 데이터 이상값 (Data Outlier)
{: .fs-8 }

7.5 이상치 처리
{: .label .label-purple }

---

## 핵심 키워드

**원인:** `데이터 입력` `측정` `실험` `고의적` `표본추출 오류`

**검출방법:** `관찰` `통계값` `시각화` `K-means` `마할라노비스 거리` `LoF` `iForest`

**처리방법:** `삭제(극단값 절단)` `대체(Cold, Hot-Deck)` `변환(극단값 조정)` `Box-Plot` `분류`

---

## 정의/개념

입력 데이터 오류, 데이터 처리 오류, 학습과정 알고리즘 오류 특정 범위에서 벗어난 데이터 값

---

## 발생원인

| 발생원인 | 상세원인 | 발생 예시 |
|:---------|:---------|:----------|
| **데이터 입력오류** | 데이터를 수집하는 과정에서 발생하는 예외<br>전체 데이터 분포에서 쉽게 발견가능 | 10을 입력해야 하는데 100을 입력하는 경우 |
| **측정오류** | 데이터의 측정이 괴정에서 발생하는 예외<br>계가 비정상 동작하는 경우 비정상적동록 제측을 측정하는 경우 | 개가 비정상 동작하거나 1개의 제측계가 정상 동작하고 1분자 측정하는 경우 |
| **실험오류** | 동일하지 않은 실험조건으로 발생하는 예외<br>물이 끓는 시간을 측정 시에 뜨거운 불과 차가운 물로 측정하는 경우 | 물이 끓는 시간을 측정 시에 뜨거운 물과 차가운 물로 측정하는 경우 |
| **고의적 이상값** | 자기 보고식 측정(Self-Reported Measures)로 발생되는 예러<br>정상값이어도 이상값으로 판단될 확률분포 | 청소년 음연율을 조사하는 경우 일부 청소년은 정확한 수치를 기입하는 경우 |
| **표본추출 예러** | 데이터 샘플링(Sampling)과정에서 발생하는 예러<br>관측된 데이터 범위에서 많이 벗어나서 데이터 품질저하에 영향을 미치기 때문에 이상값 감을 줄 처리 필요 | 신장대상 평균신장을 조사하는 경우 에러간 중의 기가 포함되는 경우 |

---

## 검출방법

| 검출 방법 | 설명 | 적용기술 |
|:---------|:-----|:---------|
| **개별 데이터 관찰** | 개별 데이터 추이나 특이사항 관찰을 통한 이상값 검출 | 확률추정(단순무작위추정, 체계적추정, 군집추정) |
| **통계값** | 통계 지표 데이터(평균, 중앙, 최빈값)과 데이터 분산도(범위, 분산)를 활용한 이상값 검출 | 집중화경향(평균, 중앙, 최빈값), 산포도(표준편차, 최소값, 범위, 사분위편차, 변량, 산포도차트, 분산도, 첨도) |
| **시각화** | 데이터 시각화(Data Visualization)을 이용한 지표로, 이상값 검출 | 시각화마이닝(산점도그래프, 히스토그램), 분포 관계시각화(스캐터플롯, 버블차트, 히스토그램) |
| **머신러닝기법** | 데이터 군집화를 통한 이상값 검출 | K-평균 알고리즘 |
| **마할라노비스** | 데이터 밀도(분산)을 고려한 다변량 이상 | 데이터 중심으로부터 마할라노비스 거리 |

---

## 처리 방법

| 처리 방법 | 설명 | 적용기술 |
|:---------|:-----|:---------|
| **삭제**<br>(Deleting Observations) | 이상값으로 판단되는 관측값을 제외하고 분석하는 방법<br>추정치분산은 작아지지만 실제보다 과소(또는 과대) 추정되어 편의가 발생가능<br>양극단 값을 절단(Trimming)하는 동이 조사수치의 결과를 제대로 반영하지 못하는 경우가 발생할 수 있음 | 극단간 절단(기변균등 전 의한 절단, 하상단 %를 이용한 제거) |
| **대체법**<br>(Imputation) | 하한값과 상한값을 결정한 후 하한값보다 작으면 하한값으로 대체하고 상한값보다 크면 상한값으로 대체<br>데이터 결측값(Missing Value)처리 방법과 동일하게 처리가능 | 단일대체법(평균대체법, 연역적 대체법, 회귀 대체법, 위치대로 대체법, Hot-Deck대체법)<br>다중대체법 |
| **변환**<br>(Transformation) | 극단적인 값으로 인해 이상값이 발생하는 경우 자연로그 그룹 취해 값을 감소시키는 방법으로 실측값을 변환 상하한 감 변화이나는 값을 하한-상한으로 변경하여 활용 | 극단간 조정(Winsorizing, 상하한 값 범위이나는 값을 하한-상한으로 변경하여 활용) |
| **박스플롯**<br>(Box-Plot)활용 | 박스플롯 해석을 통해 사분위수를 이용해 제거<br>박스플롯의 수염(Whiskers, Q1과 Q3으로부터 IQR의 1.5배 내에 있는 가장 멀리 떨어진 데이터까지 이어진 선)보다 바깥쪽 데이터를 이상값으로 분류 | 박스플롯 해석을 통해 이상값 제거 |
| **분류**<br>(Classification) | 이상값이 많은 경우 사용하며, 서로 다른 그룹으로 통계 개괄 분석을 실행하여 처리<br>각각의 그룹에 대한 통계적 모형을 생성하고, 결과를 결합하는 방법 | 그룹별 이상치 판별 |
