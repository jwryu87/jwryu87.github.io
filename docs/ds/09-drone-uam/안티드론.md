# 안티드론 (Anti-Drone)
{: .fs-8 }

9. 드론/UAM
{: .label .label-purple }

> 암기: 동작단계 - **탐식무** (탐지, 식별, 무력화)

---

## 정의

통제 구역 및 중요 시설에 대한 드론의 **불법 침입과 공격 시도를 탐지**하고 대응함으로써 범죄나 위협에 이용되는 **드론의 공격을 방어 또는 무력화**하는 시스템

---

## 동작단계 (탐식무)

| 단계 | 설명 |
|:-----|:-----|
| **탐지** | 드론의 존재 감지 |
| **식별** | 드론의 종류/위협 수준 판별 |
| **무력화** | 드론 제거/무력화 |

---

## 탐지 기술

| 기술 | 설명 |
|:-----|:-----|
| **음향 탐지센서** | 드론 프로펠러 소음 감지 |
| **방향 탐지센서** | 드론 방향 추적 |
| **영상 센서** | 카메라 기반 시각 탐지 |
| **레이더 센서** | 전파 기반 탐지 |
| **RF Sensing** | 무선 주파수 신호 감지 |
| **Electro-optical** | 전자광학 탐지 |
| **Magnetic** | 자기장 기반 탐지 |
| **Acoustic** | 음향 기반 탐지 |

---

## 무력화 기술

| 기술 | 설명 |
|:-----|:-----|
| **전파 교란 (Jamming)** | 드론 통신 신호 교란 |
| **GPS 스푸핑 (Spoofing)** | 위조된 GPS 신호로 혼란 유발 |
| **파괴 기술** | 물리적 파괴 |
| **포획 기술** | 그물 등으로 포획 |
| **EMP** | 전자기 펄스로 전자기기 무력화 |
| **Laser** | 레이저 빔으로 파괴 |
| **Firearm** | 화기로 격추 |

---

## 무력화 기술 분류

| 방식 | 기술 | 특징 |
|:-----|:-----|:-----|
| **소프트킬** | Jamming, Spoofing | 비파괴, 통신/GPS 교란 |
| **하드킬** | EMP, Laser, Firearm | 물리적 파괴 |
