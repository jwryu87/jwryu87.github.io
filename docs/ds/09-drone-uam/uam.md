# UAM (Urban Air Mobility)
{: .fs-8 }

9. 드론/UAM
{: .label .label-purple }

---

## 핵심 키워드

`eVTOL` `PAV` `Vertiport` `MRO` `K-UAM`

---

## 정의/개념

도심 상공에서 사람이나 화물을 운송하는 항공교통 수단으로, 기체 개발부터 인프라 구축, 플랫폼, 서비스, 유지보수 등 관련 사업을 모두 포괄하는 **도시 항공 운송 체계**

---

## 특징

| 특징 | 설명 |
|:-----|:-----|
| **수직 이착륙** | VTOL (Vertical Take-Off and Landing) |
| **배터리/첨단기술** | 전기 동력 기반 친환경 |
| **안전기술** | 자율비행, 충돌회피 |

---

## eVTOL 기술 유형 (Electronic VTOL)

| 유형 | 특징 | 속도 | 거리 | 기술난이도 |
|:-----|:-----|:-----|:-----|:---------|
| **Multi Rotor** | 다수 로터 | 70~120km/h | 단거리 | 낮음 |
| **Lift & Cruise** | 수직/수평 로터, 날개 | 150~200km/h | 인접도시 | 중간 |
| **Tilt** | 이착륙/주행 시 틸트 회전 | 150~300km/h | 인접도시 | 높음 (12인 이상) |

---

## K-UAM 로드맵 (안시확보)

> 암기: **안시확보** (안정성, 시범운영, 확대운영, 보편화)

| 단계 | 내용 | 목표 |
|:-----|:-----|:-----|
| **1단계** | 법제도/안정성 확보 | 제도 정비 |
| **2단계** | 시범 운영 | 제한적 노선 |
| **3단계** | 확대 운영 | 노선 확대 |
| **4단계** | 보편/콜 운영 | 대중화 |

**기대효과**: 경쟁력 강화, 출퇴근 교통체증 감소, 일자리 창출

---

## 연계기술 (관규통버)

| 구분 | 기술 | 설명 |
|:-----|:-----|:-----|
| **관제** | PSU | Provider of Services for UAM |
| **규제** | ATC | 항공교통관제 (Air Traffic Control) |
| **통신** | GPS, 5G, 6G, V2V | 위치 및 통신 |
| **버티포트** | 에너지 최적화, 배터리, 통신 | 인프라 |

## UAM 기술요소

| 구분 | 핵심 기술 | 설명 |
|:-----|:-----|:-----|
| **기체/부품** | - UAM 경량화 제작 기술<br>- 동력 추진 시스템<br>- eVTOL | - 경량/복합소재 인증체계 및 기준/절차, 신소재 시험 평가<br>- 전기구동장치, 프로펠러, 전기동력 저장/공급 시스템<br>- 전력 사용 수직으로 호버링(정지비행), 이착륙 모빌리티 |
| **항행/교통관리** | - UAM CNSi<br>- UAM 통합관리 | - 통신시스템(C) 항법시스템(N), 감시시스템(S)<br>- UAM 실시간 운항, 스케줄링, 위험도 관리, 가상통합 |
| **인프라** | - 버티포트 시스템<br>- 5G NSA/SA | - Vertical Flight + Port, UAM용 이착륙, 충전인프라 제공<br>- Mobility와 Vertiport 시스템, 관제센터 무선 통신 제공 |
| **서비스** | - MRO<br>- 운항정보 수집분석 기술 | - 항공기 정비(Maintenance, Repair and Operations)<br>- 도심운용 3차원 공역공간 정보 생성, 도심기상 정보 수집분석 |
| **비행기술** | - UAM 자율 항법 기술<br>- UAM 기체소음 진동저감 | - UAM 완전 자율비행 조정, 전진후 상황인지 충돌회피 기술<br>- 소음진동저감 프롭로터 기술, 기체용 소음진동 제어기술 |

---

## 주요 용어

| 용어 | 설명 |
|:-----|:-----|
| **eVTOL** | Electric Vertical Take-Off and Landing, 전기 수직 이착륙 항공기 |
| **PAV** | Personal Air Vehicle, 개인항공기 |
| **Vertiport** | Vertical + Airport, UAM용 수직 이착륙장 |
| **MRO** | Maintenance, Repair and Operations, 항공기 정비 |
| **K-UAM** | 한국형 도심항공교통 로드맵 |
| **FIMS** | Flight Information Management System, 비행정보관리시스템 |
| **CNSi** | Communication, Navigation, Surveillance integration |

---

## UAM 생태계 구성

| 구분 | 역할 |
|:-----|:-----|
| **UAM 이용자** | 승객, 화물 운송 수요자 |
| **운송사업자** | UAM 운영, 기체 보유/운영 |
| **버티포트 운영자** | 이착륙장 운영, 충전, 정비(MRO) |
| **UAM 교통관리** | 항공정보, 무인기 교통정보 관리 |
| **정부** | 공역관리, FIMS 운영, 공공치안 |
| **기체 제조사** | UAM 기체 개발 및 제조 |
