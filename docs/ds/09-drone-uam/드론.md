## 핵심 키워드

`무인항공기(UAV)` `원격조종` `자율비행` `비행제어(FC)` `GNSS` `IMU` `ESC`

---

## 정의/개념

조종사 없이 무선전파의 유도에 의하여 비행 및 원격조종이 가능한 비행기나 헬리콥터의 **무인항공기(UAV)의 총칭**

- GNSS: Global Navigation Satellite System (위성항법)
- FC: Flight Controller (비행제어기)

---

## 특징

| 특징 | 설명 |
|:-----|:-----|
| **이동성** | 3차원 공간 자유 이동 |
| **확산성** | 다양한 분야로 확산 |
| **대체성** | 사람이 접근 어려운 환경 대체 |
| **드론실명제** | 2kg 이상 드론 신고 의무 |

---

## 구성 (이통지비데임)

> 암기: **이통지비데임** (이착륙장치, 통신, 지상통제, 비행체, 데이터링크, 임무탑재체)

| 구성요소 | 설명 |
|:---------|:-----|
| **이착륙장치** | 드론 이착륙 시스템 |
| **지상통제장치** | GCS (Ground Control Station) |
| **지상지원** | 지상 지원 시스템 |
| **비행체** | 드론 본체 |
| **데이터 링크** | 통신 연결 |
| **임무탑재체** | 카메라, 센서 등 |

---

## 핵심기술 (관기멀보)

> 암기: **관기멀보** (관제, 기체, 멀티미디어, 보안)

| 분류 | 기술 | 설명 |
|:-----|:-----|:-----|
| **관제시스템** | 클라우드, AI기반 자동관제 | 원격자율비행, 3D공간정보 맵 |
| **기체 기술** | 스마트 안테나, 멀티콥터 | 기체제작, 자세 안정화 |
| **멀티미디어** | 영상촬영 실시간 제어 | 멀티미디어 스트리밍 |
| **보안** | 항재밍/항기만 GNSS | 통신네트워크 보안 |

## 세부기술 2 (통제구)

> 암기: **통제구** (통신부, 제어부, 구동부)

| 부 | 암기 | 구성 |
|:---|:-----|:-----|
| **통신부** | RW | RC(원격조정기), WiFi |
| **제어부** | 비항 | 비행제어, 항법제어 |
| **구동부** | 회병전 | 회전운동, 병진운동, 전원 |

---

## 구성요소

| 구분 | 구성요소 | 설명 |
|:-----|:-----|:-----|
| **기체** | 드론 프레임 | 드론의 물리적 구조 |
| | 모터, 프로펠러 | 추진력 생성 |
| **비행 제어 시스템** | 비행제어컴퓨터(FCC) | 비행 제어 및 안정화 |
| | 변속기(ESC) | 모터 속도 제어 |
| **탑재체** | 센서, 카메라 | 데이터 수집 및 촬영 |
| | 배터리 | 전원 공급 |
| **통신** | 통신모듈 | 지상제어장치와 통신 |
| **지상 통제 시스템** | 지상제어장치(GCS) | 드론에게 명령 전달 또는 직접 원격제어 |

---

## 드론 센서 시스템

| 센서 | 비행데이터 측정값 | 설명 |
|:-----|:-----|:-----|
| **GPS** | 위치 | 위치 정보 획득 |
| **GPS/기압계** | 고도, 속도 | 고도 및 속도 측정 |
| **자이로** | 자세 | 회전 각속도 측정 |
| **지자계** | 방향 | 방위각 측정 |
| **가속도계** | 각속도 | 가속도 측정 |

※ IMU(Inertial Measurement Unit): 자이로, 지자계, 가속도계를 통합한 관성측정장치

---

## 비행제어 시스템 흐름

```
센서(GPS, 자이로, 지자계 등)
         ↓
  비행제어 컴퓨터(FCC)  →  명령값(200Hz이상)
         ↓
     변속기(ESC)
         ↓
        모터
```

| 구성요소 | 역할 |
|:-----|:-----|
| **센서** | 위치, 고도, 자세, 방향, 각속도 등 비행데이터 측정 |
| **비행제어 컴퓨터(FCC)** | 센서 데이터 기반 비행 제어 명령 생성 (200Hz 이상) |
| **변속기(ESC)** | 모터 속도 제어 |
| **모터** | 프로펠러 구동 |
| **GPS/PMU** | 위치 정보 및 전원 관리 |
| **송수신기** | 지상제어장치와 통신 |
| **배터리** | 전원 공급 |

---

## 드론 서비스 구성

```
정보제공장치 ←→ 드론 ←→ 지상제어장치 ←→ 서비스 제공자
(서비스 관련     (임무를      (드론에게 명령      (드론 서비스
정보 제공 장치)   수행하는      전달 또는 직접      제공자)
                 드론)        원격제어하는 장치)
                                   ↑
                            서비스 요청자
                          (드론 서비스 요청자)
```
