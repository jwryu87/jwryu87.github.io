# 오픈 API (Open API)
{: .fs-8 }

13. API
{: .label .label-purple }

> 암기: 핵심 기술 - **레오데보** (REST, OAuth 2.0, XML/JSON, SSL/TLS)

---

## 정의

특정 프로그램의 기능이나 데이터를 다른 프로그램이 접근할 수 있도록 **미리 정한 통신 규칙**

---

## 핵심 기술 (레오데보)

| 기술 | 설명 |
|:-----|:-----|
| **REST** | RESTful API 아키텍처 |
| **OAuth 2.0** | 인증/인가 표준 |
| **XML/JSON** | 데이터 교환 형식 |
| **SSL/TLS** | 보안 통신 |

---

## API 유형

| 유형 | 설명 |
|:-----|:-----|
| **폐쇄형 (Close API)** | 내부 사용 전용 |
| **공개형 (Open API)** | 외부 개발자에게 공개 |

---

## 스크래핑 vs API 비교

| 구분 | 스크래핑 | API |
|:-----|:---------|:----|
| **인증** | 직접 로그인 필요 | 정보 제공 |
| **표준화** | 어려움 | 쉬움 |
| **정보처리** | 화면 파싱 | 송수신 데이터 |
| **접근** | 전체 페이지 | 토큰 기반 |
| **보안** | 취약 | 강함 |
