# API Gateway
{: .fs-8 }

13. API
{: .label .label-purple }

> 암기: 역할 - **보서** (보안, 서비스 연결) / 기능 - **보라중기** (보안, 라우팅, 중재, 기타)

---

## 정의

클라이언트가 요청한 API 서비스를 내부에서 처리가 가능한 API 형태로 **변환, 전달하는 Gateway**

---

## 역할 (보서)

| 역할 | 설명 |
|:-----|:-----|
| **보안** | 내부 데이터 보호, 접근통제, 로깅 및 모니터링 |
| **서비스 연결** | 클라이언트 요청 변환, 백엔드 처리결과 반환 |

---

## 기능 (보라중기)

| 기능 | 세부 | 설명 |
|:-----|:-----|:-----|
| **보안** | 인암로 | 인증 및 인가, 암호화 통신, 로그 기능 |
| **라우팅** | 매로 | 서비스 매칭, 로드밸런싱 |
| **중재 (Mediation)** | - | HTTP/JSON 기반 프로토콜 변환 |
| **기타** | 오디통 | 서비스 오케스트레이션, 서비스 디스커버리, 서비스 통계 |

---

## 아키텍처

```
└──────────────┘     │  - 인증/인가     │     │  - Service A     │
                     │  - 라우팅        │     │  - Service B     │
                     │  - 로드밸런싱    │     │  - Service C     │

                     └──────────────────┘     └──────────────────┘
```
