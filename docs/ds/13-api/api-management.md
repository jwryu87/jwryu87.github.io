# APIM (API Management)
{: .fs-8 }

13. API
{: .label .label-purple }

> 암기: 구성요소 - **게포서저** (게이트웨이, 포털, 서버, 저장소)

---

## 정의

기업 및 클라우드 전반에서 애플리케이션과 데이터를 연결하는 API를 **배포, 제어, 분석하고 중앙집중식으로 관리**하는 시스템

---

## 구성요소 (게포서저)

| 구성요소 | 설명 |
|:---------|:-----|
| **API 게이트웨이** | 변환엔진, 보안, 부하분산, API 오케스트레이션 |
| **API 포털** | 문서화, 사용자 등록, 분석, 커뮤니티 |
| **API 서버** | API 요청 처리 |
| **API 저장소** | API Repository |

## 주요 기능

| 기능 | 설명 |
|:-----|:-----|
| **API 라이프사이클 관리** | 생성, 배포, 폐기 관리 |
| **개발자 포털** | API 문서 및 테스트 환경 제공 |
| **접근 제어** | 인증/인가 관리 |
| **사용량 분석** | API 호출 통계 및 분석 |
| **정책 관리** | 사용량 제한, 보안 정책 |
