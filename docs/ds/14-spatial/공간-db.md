# 공간 DB (Spatial Database)
{: .fs-8 }

14. 공간정보
{: .label .label-purple }

---

## 핵심 키워드

`공간 데이터` `위치 정보` `GIS` `공간 질의` `공간 분석` `인덱싱` `R-Tree` `Quad Tree` `3차원`

---

## 정의/개념

위치 정보와 관련된 **공간 데이터를 효율적으로 저장, 관리, 분석, 검색**하기 위한 **데이터베이스**

- 전통적인 데이터베이스가 문자나 숫자와 같은 속성 정보를 주로 다루는 반면, 공간 데이터베이스는 **점, 선, 면, 다각형 등의 기하학적 객체**와 이와 관련된 공간적인 관계를 표현하고 처리하는 기능을 제공

---

## 공간 DB 필요성

```
위치/좌표 정보  →  일반 데이터와 상이 특징  →  공간 DB  →  공간 연산 함수
(점, 선, 면 등)    - RDBMS 연산 한계           화        공간 관계 함수
                   - 데이터의 표현 한계                   공간 데이터 타입
    ↓                     ↓                    ↓              ↓
 공간 데이터    →    공간 DB 필요성    →   공간 DB화  →  공간 DB 연산
```

---

## 특징

| 특징 | 설명 |
|:-----|:-----|
| **공간 인덱스 사용** | R-Tree, Quad Tree 등의 공간 인덱스 사용 |
| **위상, 기하적 처리 특성** | 공간적 특성 반영 비정형 데이터, 대량 데이터 처리 |
| **공간 데이터 표현, 연산** | 공간데이터 특화 연산 함수 및 데이터 모델 존재 |

---

## 구성요소

| 구분 | 설명 |
|:-----|:-----|
| **공간연산자** | 공간 데이터에 대한 연산 수행 |
| **공간질의/통합질의 처리** | 공간 질의 및 속성 질의 통합 처리 |
| **파일시스템 및 볼륨관리자** | 공간 데이터 저장 관리 |
| **공간 인덱스** | R-Tree, Quad Tree 등 공간 인덱스 |

---

## 공간 인덱스

| 인덱스 | 설명 |
|:-----|:-----|
| **R-Tree** | 사각형(Rectangle) 기반 계층적 인덱스, 다차원 공간 데이터 |
| **Quad Tree** | 2차원 공간을 4개 영역으로 재귀적 분할 |
| **K-D Tree** | K차원 이진 탐색 트리 |
| **Grid Index** | 격자 기반 인덱스 |

---

## 공간 DB 활용 분야

| 분야 | 활용 예시 |
|:-----|:-----|
| **DB구축 및 도면작성** | 지리정보 데이터베이스 구축 |
| **래스터(셀) 자료 분석** | 위성영상, 항공사진 분석 |
| **GIS 데이터 분석** | 지리공간 데이터 분석 |
| **공간정보 컨설팅** | 공간정보 시스템 설계 |
| **영상정보 구축** | 영상 데이터 관리 |
| **네트워크 분석** | 경로 탐색, 최단거리 분석 |
| **지오코딩 지오태깅** | 주소-좌표 변환 |
| **통계 및 그래프작성** | 공간 통계 분석 |
