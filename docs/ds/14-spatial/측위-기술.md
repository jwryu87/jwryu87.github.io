## GPS (범지구 위치 결정 시스템)

### GPS 구성 (우제사)

| 구분 | 구성요소 | 설명 |
|:-----|:---------|:-----|
| **우주부** | GPS 위성, 원자 시계 | 위성 신호 송출 |
| **지상제어부** | 주관제소, 지상안테나, 신호감시국 | 위성 관제 및 보정 |
| **사용자부** | 비행기, 선박, 자동차 등 | GPS 수신기 |

### GPS 종류

| 종류 | 설명 |
|:-----|:-----|
| **Standalone GPS** | 단일 위성항법시스템 |
| **LADGPS** | 지역보정위성항법시스템 |
| **SBAS** | 위성기반보정 |
| **DGPS** | 보정위성항법시스템 |
| **AGPS** | 보조 GPS |
| **WADGPS** | 광역보정위성항법시스템 |
| **CDGPS** | 반송파 보정 위성항법시스템 |

## 실내 측위 기술

### 핑거프린트 (Fingerprint)

| 항목 | 설명 |
|:-----|:-----|
| **정의** | 실내 공간 내 임의의 수집 위치 좌표와 수신신호 세기(RSSI)를 결합하여 전파맵(Radiomap/Heatmap)을 구축하는 기술 |
| **정확도** | 매우 정확 (약 5m) |
| **특징** | 사전 수집된 데이터 기반 위치 추정 |

### 삼각측량 (Triangulation)

| 항목 | 설명 |
|:-----|:-----|
| **정의** | 신호 교차점을 이용한 위치 추정 |
| **기술** | RSSI, ToA, RTT (Round Trip Time) |
| **정확도** | 비교적 정확 (약 10m) |

## 지오펜싱 (Geofencing)

### 정의

실제 지형에 구획된 **가상의 경계선**으로 구획된 영역에 대해 디바이스의 **진입/진출을 감지**하는 측위 기반 기술

### 아키텍처

```
클라이언트(앱) ←→ ZONE 정보 송수신 ←→ 지오클라이언트 ←→ 지오서버
                                    (존 등록/모니터링)
```

### 활용 분야

| 분야 | 활용 예시 |
|:-----|:---------|
| **마케팅** | 위치 기반 푸시 알림 |
| **물류** | 배송 지역 관리 |
| **보안** | 출입 통제 |
| **IoT** | 스마트홈 자동화 |

## 측위 기술 비교

| 기술 | 정확도 | 환경 | 특징 |
|:-----|:-------|:-----|:-----|
| **GPS** | 3~15m | 실외 | 위성 기반 |
| **핑거프린트** | ~5m | 실내 | 전파맵 기반 |
| **삼각측량** | ~10m | 실내 | 신호 교차점 |
| **비콘** | ~3m | 실내 | BLE 기반 |
| **UWB** | ~30cm | 실내 | 초광대역 |
