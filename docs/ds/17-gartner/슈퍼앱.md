# 슈퍼앱 (Super App)
{: .fs-8 }

17. 가트너 전략
{: .label .label-purple }

> 암기: 기술요소 - **프백디에클** (Frontend, Backend, DB, AI, Cloud)

---

## 정의

사용자가 서비스별 앱을 설치하지 않고, **하나의 앱으로 다양한 서비스를 이용**할 수 있는 모바일 애플리케이션, 플랫폼, 생태계의 기능을 하나로 통합한 환경

---

## 서비스 유형

| 서비스 | 설명 | 예시 |
|:-------|:-----|:-----|
| **검색 서비스** | 통합 검색 기능 | 웹, 뉴스, 지도 검색 |
| **금융 서비스** | 결제, 송금, 투자 | 간편결제, 계좌이체 |
| **주문 서비스** | 상품/서비스 주문 | 배달, 쇼핑 |
| **OTT 서비스** | 콘텐츠 스트리밍 | 영상, 음악 |

---

## 기술요소 (프백디에클)

| 기술 | 설명 |
|:-----|:-----|
| **Frontend** | 사용자 인터페이스 |
| **Backend** | 서버 로직 처리 |
| **DB** | 데이터 저장 및 관리 |
| **AI** | 개인화, 추천 |
| **Cloud** | 인프라 확장성 |

### 연관 기술

- API
- HTML5
- Web2.0
- 블록체인
- 인증 (OAuth 등)

---

## 슈퍼앱 사례

| 지역 | 앱 이름 | 특징 |
|:-----|:--------|:-----|
| **중국** | WeChat | 메시징, 결제, 쇼핑, SNS 통합 |
| **동남아** | Grab, Gojek | 차량, 배달, 결제 통합 |
| **한국** | 카카오, 네이버 | 메시징, 결제, 콘텐츠, 금융 |
| **미국** | - | 슈퍼앱 개념 도입 중 |

---

## 슈퍼앱 특징

| 특징 | 설명 |
|:-----|:-----|
| **원스톱 서비스** | 하나의 앱에서 모든 서비스 |
| **생태계 구축** | 미니앱/미니프로그램 지원 |
| **사용자 락인** | 높은 전환 비용 |
| **데이터 통합** | 사용자 데이터 일원화 |
