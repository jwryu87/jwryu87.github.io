# 스테이블 코인 (Stable Coin)
{: .fs-8 }

3.6 블록체인 경제
{: .label .label-purple }

---

## 정의

가격 변동성을 최소화하기 위해 법정화폐, 암호화폐, 알고리즘 등에 **가치를 연동(Pegging)**시킨 암호화폐

---

## 특징

| 특징 | 설명 |
|:-----|:-----|
| **수수료 절감** | 국제 송금 등 수수료 절감 |
| **소요 시간 단축** | 거래/정산 시간 단축 |
| **정산 비용 절감** | 중개 기관 없이 직접 정산 |

---

## 유형

| 유형 | 설명 | 예시 |
|:-----|:-----|:-----|
| **법정화폐 담보형** | 달러 등 법정화폐 1:1 담보 | USDT, USDC |
| **암호화폐 담보형** | 암호화폐를 담보로 발행 | DAI |
| **무담보 알고리즘형** | 알고리즘으로 공급량 조절 | UST (실패), FRAX |

---

## 유형별 비교

| 구분 | 법정화폐 담보형 | 암호화폐 담보형 | 알고리즘형 |
|:-----|:--------------|:--------------|:----------|
| **담보** | 법정화폐 | 암호화폐 | 없음 |
| **탈중앙화** | 낮음 | 중간 | 높음 |
| **안정성** | 높음 | 중간 | 낮음 |
| **투명성** | 감사 필요 | 온체인 확인 | 코드 공개 |
