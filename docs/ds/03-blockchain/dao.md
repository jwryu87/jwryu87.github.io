# DAO (Decentralized Autonomous Organization)
{: .fs-8 }

3.7 블록체인 생태계
{: .label .label-purple }

> **Web 3.0의 핵심 구성요소**

---

## 정의

블록체인과 스마트 컨트랙트를 기반으로 운영되는 **탈중앙화 자율 조직**

---

## 특징

| 특징 | 설명 |
|:-----|:-----|
| **탈중앙화** | 중앙 관리자 없이 운영 |
| **수평성** | 참여자 간 평등한 의사결정 |
| **자율성** | 스마트 컨트랙트 기반 자동 실행 |
| **투명성** | 모든 거래/의사결정 공개 |

---

## 구현요소

### DAO 작동방식

| 요소 | 설명 |
|:-----|:-----|
| 규칙 설정 | 조직 운영 규칙 정의 |
| 자금조달 및 이익배분 구조 설계 | 토큰 이코노미 설계 |
| 스마트 컨트랙트 | 규칙의 자동 실행 |

### DAO 유형

| 유형 | 설명 |
|:-----|:-----|
| **프로토콜 DAO** | 정책 관련 의사결정 (DeFi 거버넌스) |
| **투자 DAO** | 공동 투자 및 수익 배분 |
| **컬렉터 DAO** | NFT 등 자산 공동 수집/관리 |

### 관련 자산

| 자산 | 설명 |
|:-----|:-----|
| **STO** | 증권형 토큰 |
| **스테이블 코인** | 가치 안정 코인 |

---

## DAO vs 전통 조직

| 구분 | DAO | 전통 조직 |
|:-----|:----|:----------|
| **구조** | 수평적 | 계층적 |
| **의사결정** | 투표 기반 | 경영진 결정 |
| **투명성** | 완전 공개 | 제한적 |
| **실행** | 자동 (스마트 컨트랙트) | 수동 |
