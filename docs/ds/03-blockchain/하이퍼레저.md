# 하이퍼레저 (Hyperledger)
{: .fs-8 }

3.7 블록체인 생태계
{: .label .label-purple }

---

## 서비스

| 서비스 | 기능 |
|:-------|:-----|
| **신원** | 참여자, 자산에 대한 신원 확인 |
| **정책** | 거래, 계약 정보에 대한 접근제어, 합의 정책 |
| **블록체인** | 원장 관리, 통신 프로토콜 |
| **스마트 계약** | 스마트 계약 실행 및 검증 |

---

## 세부 프로젝트

### 하이퍼레저 Framework

| 프로젝트 | 설명 |
|:---------|:-----|
| **Fabric** | 대표 개발도구, 모듈형 아키텍처 |
| **INDY** | 중개자 없이 신원 확인 (DID) |
| **Burrow** | PBFT 기반 |
| **Iroha** | 모바일 환경 최적화 |

### 하이퍼레저 Tool

| 도구 | 설명 |
|:-----|:-----|
| **Composer** | 시스템 통합 |
| **CALIPER** | 성능 측정 |
| **Cello** | 운영 관리 |
| **Explorer** | 모니터링 |
| **QUILT** | 연동 프로토콜 |

---

## 핵심기술

| 분류 | 기술 | 설명 |
|:-----|:-----|:-----|
| **멤버쉽 서비스** | 외부 등록 | 사용자 등록/관리 |
| | 인증 | 신원 인증 |
| | 감사 | 활동 감사 |
| **블록체인 서비스** | HTTP/2 | 통신 프로토콜 |
| | P2P | 분산 네트워크 |
| | 해시 알고리즘 | 데이터 무결성 |
| **체인코드 서비스** | 보안 OS | 안전한 실행 환경 |
| | Go lang, Java, Node.js | 지원 언어 |
| **기타** | REST API | 외부 연동 |
| | EVENT | 이벤트 처리 |
