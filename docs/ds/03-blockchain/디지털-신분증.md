# 디지털 신분증
{: .fs-8 }

3.5 ID/Wallet
{: .label .label-purple }

> 암기: **필요성** (경사기), **기술** (모인보)

---

## 필요성 (경사기)

| 측면 | 필요성 |
|:-----|:-------|
| **경제적** | 경제적 가치 창출, 현행 플라스틱 신분증의 불편 사항 및 비용 개선 |
| **사회적** | 비대면, 디지털 전환 시대 돌입 대응, 디지털 신원증명의 중요성 확대 |
| **기술적** | 해킹 대응 위한 최신 기술 적용 필요, 개인정보보호 문제 해결 |

---

## 기술 (모인보)

### 모바일/기기

| 기술 | 설명 |
|:-----|:-----|
| 블루투스 | 근거리 통신 |
| NFC 및 RFID | 비접촉 통신 |
| 하이브리드 앱 | 네이티브/웹 결합 |
| 웹 2.0 | 웹 기반 서비스 |

### 인증

| 기술 | 설명 |
|:-----|:-----|
| 블록체인 | 분산 신뢰 기반 |
| DID | Decentralized Identity |
| 바이오 인증 | 생체 인증 |
| FIDO (2.0) | 패스워드리스 인증 |

### 보안

| 기술 | 설명 |
|:-----|:-----|
| DKMS | 분산 키 관리 시스템 |
| TEE | Trust Execution Environment, 신뢰 실행 환경 |
