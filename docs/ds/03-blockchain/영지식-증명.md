# 영지식 증명 (Zero-Knowledge Proof)
{: .fs-8 }

3.1 블록체인 기본
{: .label .label-purple }

---

## 핵심 키워드

`증명자(Prover)` `검증자(Verifier)` `완전성` `건전성` `영지식성`

---

## 정의/개념

정보를 주고받을 때, **상대방에게 정보를 노출하지 않으면서** 정보의 진위 여부를 증명하는 **암호학적 기술**

---

## 영지식 증명 메커니즘

### 참여자

| 참여자 | 설명 |
|:-------|:-----|
| **증명자 (Prover)** | 어떤 문장이 참이라는 것을 증명하려는 참여자 |
| **검증자 (Verifier)** | 증명 과정에 참여하여 증명자(Prover)와 정보를 주고받는 참여자 |

### 증명 과정 (예시: 미로 문제)

| 과정 | 설명 |
|:-----|:-----|
| ① | Bob은 A 위치에서 대기 |
| ② | Alice는 B에서 C 혹은 D까지 이동 |
| ③ | Bob은 B 위치로 이동 |
| ④ | Bob은 Alice에게 C나 D 중 하나로 나오라고 요청 |
| ⑤ | Alice가 키가 있을 경우 요청하는 방향이 어디든지 나가는 것이 가능 |
| ⑥ | Alice가 키가 없을 경우 요청한 방향으로 나갈 수 있는 확률은 50% |

→ 이 과정을 여러 번 반복하면 Alice가 키를 가지고 있는지 높은 확률로 검증 가능

### zk-SNARK 검증방식

```
증명자 → 증명 → 검증자 → 검증 결과
   ↓              ↓
증인 매개변수  증명 매개변수
   ↓              ↓
증명함수     →   검증함수
```

---

## 영지식 증명 특징

| 특징 | 설명 |
|:-----|:-----|
| **완전성 (Completeness)** | 어떤 문장이 참일 경우, 정직한 증명자는 정직한 검증자에게 이 문장이 사실이라고 납득시킬 수 있어야 한다는 성질 |
| **건전성 (Soundness)** | 어떤 문장이 거짓일 경우, 어떠한 부정직한 증명자라도 정직한 검증자에게 이 문장이 사실이라고 납득시킬 수 없어야 한다는 성질 |
| **영지식성 (Zero-Knowledgeness)** | 어떤 문장이 참일 경우, 검증자는 문장의 참 거짓 외에는 아무것도 알 수 없어야 한다는 성질 |
