## 주요 암호기술

| 분류 | 기술 | 설명 |
|:-----|:-----|:-----|
| **서명 기술** | 다중 서명 | 여러 사용자의 서명 필요 |
| | 임계 서명 | 임계값 이상의 서명 필요 |
| | 은닉 서명 | 서명 내용 비공개 |
| | 환 서명 | 서명자 익명성 보장 |
| **정보 분산** | 영지식 증명 | 정보 노출 없이 증명 |
| | 안전한 다자간 계산 | MPC (Multi-Party Computation) |
| | 비밀 분산 기법 | 정보를 분산 저장 |
| **기타** | 위탁 방식 | 신뢰 기관에 위탁 |
| | 불확정 전송 | Oblivious Transfer |
| | 검증 가능한 랜덤 함수 | VRF |
| | Accumulator | 집합 멤버십 증명 |

---

## 암호기술 가이드라인 (암계전합 원컨데)

| 구분 | 가이드라인 | 세부 내용 |
|:-----|:-----------|:----------|
| **암호 사용 기준** | 검증된 알고리즘 | 국내외 표준 암호 알고리즘 사용 |
| **계정 관리** | 관리, 공개키, 폐기 | 계정 생성/관리/폐기 절차 |
| **데이터 전송 보안** | 전송 암호화, 상호 인증 | TLS/SSL 등 보안 프로토콜 |
| **합의 프로토콜** | 안전한 합의 프로토콜 | 노드 수, 참여자 고려 결정 |
| **원장** | 공격 방어, 서명 검증 | 원장 무결성 보장 |
| **스마트 컨트랙트** | 결과 검증, 취약점 미존재 | 코드 보안 감사 |
| **데이터베이스** | 장애 시 신뢰성 보장 | 백업 및 복구 체계 |
