# 라이트닝 네트워크 (Lightning Network)
{: .fs-8 }

3.7 블록체인 생태계
{: .label .label-purple }

---

## 특징

| 특징 | 설명 |
|:-----|:-----|
| **거래수수료 절감** | 온체인 대비 수수료 대폭 감소 |
| **빠른 처리 속도** | 즉시 결제 가능 |
| **높은 확장성** | TPS 한계 극복 |

---

## 프로세스

```
OnChain → OffChain → OnChain
```

1. 온체인에서 지불채널 개설
2. 오프체인에서 다수 거래 수행
3. 최종 결과만 온체인에 기록

---

## 구성

| 구분 | 요소 | 설명 |
|:-----|:-----|:-----|
| **체인** | 온체인 | 블록체인 메인넷 |
| | 오프체인 | 별도 지불채널 |
| | 지불채널 | 양자 간 거래 채널 |
| **트랜잭션** | Funding Transaction | 지불채널 개설 (블록체인 기록) |
| | Commitment Transaction | 블록체인에 기록되지 않는 약속 거래 |

---

## 문제점 및 해결

| 문제 | 해결방안 |
|:-----|:---------|
| **OffChain 거래기록 조작** | 타임스탬프 사용 |
| | 모니터링 제도 도입 |
