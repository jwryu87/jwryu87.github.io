# 블록체인 경량화 기술
{: .fs-8 }

3.1 블록체인 기본
{: .label .label-purple }

---

## 라이트 노드 (Light Node)

블록 헤더 정보만 저장해 경량화하는 기술

| 항목 | 설명 |
|:-----|:-----|
| **개념** | 블록 헤더만 저장해 풀 노드 대비 저장 용량 **10,000배 이상 절약** |
| **검증 방식** | SPV (Simplified Payment Verification) |
| **SPV 절차** | 풀노드 경로정보 요청 → 정보획득 및 체크 |
| **한계점** | 풀노드 정보를 봐야 하므로 완전한 탈중앙화 아님 |

---

## 트랜잭션 프루닝 기술 (Transaction Pruning)

최신 트랜잭션 정보만 저장하는 기술

| 유형 | 설명 |
|:-----|:-----|
| **Selective Transaction Pruning** | 트랜잭션 선별 제거 |
| **Verifiable Pruning** | 위변조 증명데이터 별도 저장 |

| 항목 | 설명 |
|:-----|:-----|
| **특징** | 새로운 제네시스 블록을 만드는 형태 |
| **한계점** | 완전한 탈중앙화 아님 |

---

## 경량화 기술 비교

| 구분 | 라이트 노드 | 트랜잭션 프루닝 |
|:-----|:-----------|:---------------|
| **저장 대상** | 블록 헤더만 | 최신 트랜잭션만 |
| **검증 방식** | SPV | 별도 증명 데이터 |
| **용량 절감** | 매우 큼 (10,000배) | 중간 수준 |
| **한계** | 풀노드 의존 | 탈중앙화 제한 |
