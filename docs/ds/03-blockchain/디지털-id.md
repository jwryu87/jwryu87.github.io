# 디지털 ID (Digital Identity)
{: .fs-8 }

3.5 ID/Wallet
{: .label .label-purple }

> 암기: **발전단계** (센연사자), **관리기술** (개중연분), **신원증명 프로세스** (등발이검)

---

## 정의

각종 ID, 주민번호, 전자우편주소, 신용카드번호, 주소 등 **사이버 스페이스 상에서 개인을 식별할 수 있는 속성들의 집합**

---

## 발전단계 (센연사자)

| 단계 | 유형 | 설명 |
|:-----|:-----|:-----|
| 1 | **센터형** | 중앙 집중식 ID 관리 |
| 2 | **연계형** | 여러 기관 간 ID 연계 |
| 3 | **사용자 중심형** | 사용자가 ID 선택/관리 |
| 4 | **자기주권형** | 사용자가 완전히 소유/통제 |

---

## 관리기술 유형 (개중연분)

| 유형 | 설명 |
|:-----|:-----|
| **개방형 (Silo)** | 서비스별 독립 ID |
| **중앙집중형 (Centralized)** | 단일 기관 관리 |
| **연계형 (Federated)** | 기관 간 ID 연계 |
| **분산형 (Decentralized)** | 블록체인 기반 분산 관리 |

---

## 자기주권형 신원증명 프로세스 (등발이검)

| 단계 | 설명 |
|:-----|:-----|
| **분산 ID 등록 단계** | 블록체인에 DID 등록 |
| **증명서 발급 단계** | 발급기관이 VC 발급 |
| **증명서 이용 단계** | 사용자가 서비스에 제출 |
| **증명서 검증 단계** | 서비스가 블록체인으로 검증 |

---

## 자기주권형 신원증명 기술 (분체영)

| 기술 | 설명 |
|:-----|:-----|
| **분산 ID (DID)** | 탈중앙화 식별자 |
| **블록체인** | 신뢰 기반 인프라 |
| **영지식 증명** | 정보 노출 없이 검증 |

---

## 디지털 ID 통합 플랫폼 아키텍처

| 참여자 | 역할 |
|:-------|:-----|
| **신원관리 사업자** | ID 발급/관리 |
| **보관자** | 증명서 보관 |
| **이용자** | ID 소유/제출 |
| **서비스 제공자** | ID 검증/서비스 제공 |
