## 개념도

```
┌─────────────────────────────────────────────┐
│           암호화폐 지갑 (Wallet)              │
│  ┌──────────────────────────────────────┐  │
│  │    코인 → 공개키로 잠금               │  │
│  │    코인 ← 개인키로 잠금해제           │  │
│  └──────────────────────────────────────┘  │
└─────────────────────────────────────────────┘
```

---

## 유형

### 핫월렛 (Hot Wallet) - Online

| 유형 | 설명 |
|:-----|:-----|
| **웹 지갑** | 웹 브라우저 기반 지갑 |
| **데스크톱 지갑** | PC에 설치하는 지갑 |
| **모바일 지갑** | 스마트폰 앱 기반 지갑 |

### 콜드월렛 (Cold Wallet) - Offline

| 유형 | 설명 |
|:-----|:-----|
| **하드웨어 지갑** | 전용 하드웨어 장치 (Ledger, Trezor) |
| **종이 지갑** | 개인키를 종이에 인쇄 (진짜 인쇄물) |

---

## 핫월렛 vs 콜드월렛 비교

| 구분 | 핫월렛 | 콜드월렛 |
|:-----|:-------|:---------|
| **연결 상태** | 항상 온라인 | 오프라인 |
| **편의성** | 높음 | 낮음 |
| **보안성** | 낮음 | 높음 |
| **용도** | 일상 거래 | 장기 보관 |

---

## 보안강화: 멀티시그 월렛 (Multi-Signature Wallet)

| 항목 | 설명 |
|:-----|:-----|
| **개념** | 여러 명이 동시 서명해야 출금 가능 |
| **방식** | 2of3, 3of5 등 |
| **키 구성** | 사용자키, 거래소키, 백업키 + 추가 사용자키 |
| **장점** | 단일 실패점 제거, 보안 강화 |
