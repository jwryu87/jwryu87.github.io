# 스마트 공장 (Smart Factory)
{: .fs-8 }

5.1 스마트공장
{: .label .label-purple }

---

## 핵심 키워드

`제품 기획부터 생산 및 판매까지 전 단계`

---

## 정의/개념

제품 기획부터 생산 및 판매까지 ICT 기술 사용 전 단계 **자동화 최첨단 지능형 공장**

---

## 스마트공장 구성요소

| 구분 | 기술 |
|:-----|:-----|
| **응용시스템 S/W** | MES, SCM, POP, ERP |
| **제어자동화** | PLC, HMI, CNC |
| **현장자동화** | 스마트 센서, 지능형 로봇, 액츄에이터 |
| **기반 기술** | 클라우드, CPPS(Cyber Physical Production System), AI 지능형 기술, 디지털 트윈 |
| **네트워크** | 5G 특화망, IIoT(Industrial IoT), 엣지컴퓨팅 |
| **참조모델** | 퍼듀모델, RAMI 4.0 |
| **정보보안** | DLP, IAM, SIEM, SOAR |

---

## 스마트공장 아키텍처

```
                      주문 →
            ┌──────────────────────────┐
            │  협력사   공급사/관리/   협력사  │
            │  시스템    무역관리       │
┌───────┐  │                        │
│ 응용  │  │   전자정보    Cloud    │
│ 시스템 │  │    ─         클라우드  │
│ S/W   │  │   수동관      생산설비  │  기업별
└───────┘  │   공정계획              │
            │                        │
            └──────────────────────────┘
┌──────────────────────────────────────┐
│  제어    제어장비    공정    작업명령·가동정리   │
│ 자동화   인터페이스  관리               │
│          인터페이스  (CNC)   산업 센서    │
└──────────────────────────────────────┘
            ↓       설비 제어       ↓
            생산·가동 데이터
┌──────────────────────────────────────┐
│  현장    로봇,    협업    생산    CPPS   │
│ 자동화   생산설비  로봇,   지시    가상-실체사  │
│          제어    산업용  시스템            │
└──────────────────────────────────────┘
              AI 지능형 기술
```
