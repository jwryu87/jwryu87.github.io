# 디지털 트윈 (Digital Twin)
{: .fs-8 }

5.1 스마트공장
{: .label .label-purple }

> 암기: **모관모연자**

---

## 핵심 키워드

- **현실세계**: 센서, 액츄에이터, 컨트롤러
- **가상세계**: 디지털 스레드, 가상공간 자산, 3D 모델링
- **통합세계**: VR/AR 연계, 양방향 전달, 공정제어
- **UR**: Unified Repository

---

## 정의/개념

현실 세계 센서 데이터, **복제**, **재현**, **가시성** **동적 시뮬레이션 기술**

---

## 디지털 트윈의 특징

| 구분 | 특징 | 설명 |
|:-----|:-----|:-----|
| **현실측면** | 복제성 | 물리 모델을 사이버에 그대로 구현 |
|  | 재현성 | 관찰 후 생산현장 즉각적인 피드백 |
|  | 추천성 | 의사결정 모수 등을 활용, 기대 결과 도출 또는 예측 |
|  | 보존성 | 수집된 데이터와 분석한 데이터의 결과 등을 반영구적 보존 |
| **운영측면** | 가시성 | 분석된 데이터와 모델에 대한 Visualization |
|  | 분석 가능성 | 물리적인 자산의 현재 상태, 운영방식, 생산 스케줄 등 예측 |

- 실제 물리적인 물체 및 시스템을 가상의 디지털 세계에 고스란히 재현하는 방식의 복제

---

## 디지털 트윈 기술요소

| 구분 | 기술 | 설명 |
|:-----|:-----|:-----|
| **현실세계 정보수집** | 센서(sensor) | 센서 데이터 및 메타 데이터 수집 |
|  | 액츄에이터(actuator) | 절삭/가공에 의한 마모도 및 공정 데이터 수집 |
|  | 컨트롤러(controller) | WPAN, PLC, Modbus, RS-232, RS-485 등 사용 가능 |
| **가상세계 구성** | 디지털 스레드 | 물리 자산의 디지털화 |
|  | 가상공간 자산 | 디지털 자산 설계, 테스트, 피드백 가능 |
|  | 3D 모델링 | 전체 공정의 디지털화 가능 |
| **현실/가상통합** | VR/AR 연계 | 현실세계와 가상세계의 연결 역할 |
|  | 양방향 전달 | 제품에 대한 가상세계에서 다양한 테스트 가능 |
|  | 공정제어 | 공정의 시뮬레이션 가능 |

- 디지털 트윈으로 현실 세계를 최적화하기 위하여 디지털 트윈의 기술 발전 단계별 모델 적용

---

## 디지털 트윈 기술 발전 5단계별 모델

| 단계 | 모델 | 설명 |
|:-----|:-----|:-----|
| **5단계** | 자율 디지털 트윈 (Autonomous) | 개발 및 복합 디지털 트윈에서 자율적으로 문제점을 인지하고 해결하여 물리대상 최적화 |
| **4단계** | 연합 디지털 트윈 (Federated) | 최적화의 기반 클러대상들이 상호 연계된 복합 디지털 트윈 체계 구성 및 물리대상 상호운영 최적화 |
| **3단계** | 모의 디지털 트윈 (Modeling & Simulation) | 디지털 트윈 모의결과를 적용한 물리대상 최적화 |
| **2단계** | 관제 디지털 트윈 (Monitoring) | 디지털 트윈 기반 물리대상 모니터링 및 관제분석을 통한 제어 |
| **1단계** | 모사 디지털 트윈 (Mirroring) | 물리대상을 디지털 트윈으로 복제 |

- 디지털 트윈 모델에 따른 디지털 핵심 선도 위해선 제도권 주도의 발전 전략이 필요
