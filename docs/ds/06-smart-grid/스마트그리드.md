# 스마트그리드 (Smart Grid)
{: .fs-8 }

6.1 스마트 그리드
{: .label .label-purple }

---

## 정의

기존 전력망에 **ICT 기술**을 융합하여 **양방향 전력 정보 교환**으로 에너지 효율을 최적화하는 **차세대 지능형 전력망**

---

## 기존 전력망 vs 스마트그리드

| 구분 | 기존 전력망 | 스마트그리드 |
|:-----|:-----------|:------------|
| **통신** | 단방향 | 양방향 |
| **발전** | 중앙 집중형 | 분산형 |
| **소비자** | 수동적 | 능동적 (프로슈머) |
| **정보** | 제한적 | 실시간 |
| **복구** | 수동 복구 | 자가 치유 |

---

## 스마트그리드 구성요소

| 구분 | 구성요소 | 설명 |
|:-----|:---------|:-----|
| **발전** | 신재생에너지 | 태양광, 풍력 등 분산 발전 |
| | ESS | 에너지 저장 시스템 |
| **송배전** | SCADA | 원격 감시 제어 |
| | EMS | 에너지 관리 시스템 |
| | DMS | 배전 관리 시스템 |
| **수용가** | AMI | 지능형 검침 인프라 |
| | HAN | 홈 에어리어 네트워크 |
| | HEMS | 홈 에너지 관리 시스템 |
| **통신** | PLC | 전력선 통신 |
| | RF Mesh | 무선 메시 네트워크 |

---

## 스마트그리드 핵심 기술

| 분류 | 기술 | 설명 |
|:-----|:-----|:-----|
| **지능형 검침** | AMI 2.0 | 양방향 실시간 검침 |
| **수요 반응** | DR | Demand Response |
| **전력 품질** | FACTS | 유연 AC 송전 시스템 |
| **보안** | 전력 사이버보안 | NERC CIP 등 |

---

## 기대 효과

| 측면 | 효과 |
|:-----|:-----|
| **경제성** | 피크 부하 감소, 전력 비용 절감 |
| **효율성** | 전력 손실 감소, 설비 이용률 향상 |
| **환경** | 신재생에너지 확대, 탄소 배출 감소 |
| **안정성** | 자가 치유, 정전 시간 단축 |
