# 분산 에너지 시스템 (DER, Distributed Energy Resources)
{: .fs-8 }

6.2 에너지 기술
{: .label .label-purple }

---

## 정의

소규모 분산 전원을 전력 계통에 연결하여 **지역 단위로 전력을 생산/저장/소비**하는 **분산형 에너지 시스템**

---

## 중앙 집중형 vs 분산형

| 구분 | 중앙 집중형 | 분산형 |
|:-----|:-----------|:-------|
| **발전** | 대규모 발전소 | 소규모 분산 전원 |
| **송전** | 장거리 고압 송전 | 지역 내 단거리 |
| **효율** | 송전 손실 발생 | 손실 최소화 |
| **안정성** | 단일 장애점 | 분산 구조로 안정 |

---

## 분산 에너지 자원 종류

| 분류 | 자원 | 설명 |
|:-----|:-----|:-----|
| **발전** | 태양광 | PV (Photovoltaic) |
| | 풍력 | 소형 풍력 터빈 |
| | 연료전지 | 수소 연료전지 |
| | CHP | 열병합 발전 |
| **저장** | ESS | 에너지 저장 시스템 |
| | V2G | 전기차 배터리 활용 |
| **관리** | DERMS | 분산자원 관리 시스템 |
| | VPP | 가상 발전소 |

---

## 가상 발전소 (VPP)

| 구분 | 설명 |
|:-----|:-----|
| **정의** | 분산 에너지 자원을 통합 관리하여 하나의 발전소처럼 운영 |
| **구성** | 태양광 + ESS + DR + EV |
| **기능** | 수요 예측, 발전 최적화, 전력 거래 |

---

## 기대 효과

| 효과 | 설명 |
|:-----|:-----|
| **에너지 자립** | 지역 단위 에너지 자급자족 |
| **계통 안정** | 피크 부하 분산 |
| **환경** | 신재생에너지 확대 |
| **경제** | 전력 거래 수익 |
