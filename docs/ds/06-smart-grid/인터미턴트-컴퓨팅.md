# 인터미턴트 컴퓨팅 (Intermittent Computing)
{: .fs-8 }

6.2 에너지 기술
{: .label .label-purple }

---

## 정의

에너지 수확 기반의 **불규칙한 전력 공급** 환경에서 **전력 중단에도 연산을 지속**할 수 있는 컴퓨팅 패러다임

---

## 기존 컴퓨팅 vs 인터미턴트 컴퓨팅

| 구분 | 기존 컴퓨팅 | 인터미턴트 컴퓨팅 |
|:-----|:-----------|:-----------------|
| **전원** | 안정적 공급 | 불규칙 공급 |
| **메모리** | 휘발성 (RAM) | 비휘발성 (NVRAM) |
| **실행** | 연속 실행 | 단속 실행 |
| **상태** | 전원 차단 시 손실 | 상태 보존 |

---

## 핵심 기술

| 기술 | 설명 |
|:-----|:-----|
| **체크포인팅** | 주기적 상태 저장 |
| **NVRAM** | 비휘발성 메모리 활용 |
| **태스크 기반 실행** | 원자적 태스크 단위 실행 |
| **에너지 인지 스케줄링** | 가용 에너지 기반 태스크 스케줄링 |

---

## 체크포인팅 방식

| 방식 | 설명 | 장단점 |
|:-----|:-----|:-------|
| **주기적** | 일정 주기로 저장 | 오버헤드 발생 |
| **에너지 기반** | 에너지 임계값 도달 시 저장 | 효율적 |
| **태스크 기반** | 태스크 완료 시 저장 | 일관성 보장 |

---

## 적용 분야

| 분야 | 적용 예시 |
|:-----|:---------|
| **IoT** | 배터리 없는 센서 노드 |
| **웨어러블** | 에너지 수확 기반 디바이스 |
| **환경 모니터링** | 원격 무전원 센서 |
| **의료** | 인체 삽입형 기기 |
