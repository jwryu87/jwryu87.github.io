# 에너지 수확 (Energy Harvesting)
{: .fs-8 }

6.2 에너지 기술
{: .label .label-purple }

---

## 정의

주변 환경에서 발생하는 **버려지는 에너지**를 수집하여 **전기 에너지로 변환**하는 기술

---

## 에너지 수확 원천

| 에너지원 | 변환 방식 | 출력 범위 | 적용 분야 |
|:---------|:---------|:---------|:---------|
| **태양광** | 광전 효과 | mW ~ W | 웨어러블, IoT |
| **진동** | 압전 효과 | μW ~ mW | 산업 센서 |
| **열** | 열전 효과 | μW ~ mW | 체온 센서 |
| **RF** | 전자기파 | μW | RFID, 무선 충전 |
| **바람** | 소형 터빈 | mW ~ W | 옥외 센서 |

---

## 에너지 수확 시스템 구성

| 구성요소 | 기능 |
|:---------|:-----|
| **에너지 변환기** | 환경 에너지 → 전기 에너지 |
| **전력 관리 IC** | 전압 조절, 효율 최적화 |
| **에너지 저장** | 슈퍼커패시터, 소형 배터리 |
| **부하** | IoT 센서, 웨어러블 기기 |

---

## 압전 에너지 수확

| 구분 | 내용 |
|:-----|:-----|
| **원리** | 압전 소자에 압력/진동 가해지면 전기 발생 |
| **재료** | PZT, PVDF |
| **적용** | 바닥 발전, 신발 발전, 산업 진동 |

---

## 적용 분야

| 분야 | 적용 예시 |
|:-----|:---------|
| **IoT** | 배터리 없는 센서 노드 |
| **웨어러블** | 체온/움직임 기반 충전 |
| **스마트시티** | 도로 압전 발전 |
| **헬스케어** | 인체 삽입형 의료기기 |
