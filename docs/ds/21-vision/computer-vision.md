# Computer Vision (컴퓨터 비전)
{: .fs-8 }

21. 영상처리/영상보안
{: .label .label-purple }

> 암기: 절차 - **영전특해고** / 기술 - **전특탐분** / 요소기술 - **영특에매**

---

## 정의

감지된 영상의 특징 정보 추출, 매칭, 객체 인식 등의 영상 처리 기술을 통해 기계 또는 컴퓨터가 **사람 눈처럼 보고 인지하고 이해**할 수 있는 분석 시스템

---

## 절차 (영전특해고)

```
영상 → 전처리 → 특징추출 → 해석 → 고급묘사
```

---

## 기술 (전특탐분)

| 분류 | 기술 |
|:-----|:-----|
| **전처리** | 그레이 스케일, 이진화, 확대/축소 |
| **특징추출** | STIP (행동추출), SIFT (특징맵핑), SURF (관심점 계산) |
| **탐지/분할** | PCA, Neural Network |
| **분류** | SOM, SVM, K-Means |

---

## 요소기술 (영특에매)

| 기술 | 세부 기술 |
|:-----|:---------|
| **영상분할** | Mean Shift, 최소신장트리, 워터셰드 |
| **특징기술** | 전역 기술자, 지역관계 기술자, PCA |
| **에지검출** | Zero Crossing, Canny Edge |
| **매칭** | 빠른 최근접 이웃탐색, 기하 정렬 |

---

## 평가방법

- 인식 성능 측정
- 혼동 행렬
- 재현율 (Recall)
- 정확률 (Precision)
