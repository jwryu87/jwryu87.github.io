# 지능형 영상분석 기술
{: .fs-8 }

21. 영상처리/영상보안
{: .label .label-purple }

> 암기: 이벤트 - **OBS** / 단계 - **배식추이** / 기술 - **전탐추보**

---

## 정의

영상의 정보를 분석하여 **자동으로 이상 행위(이벤트)를 탐지**하고 관리자에게 경보를 전송하는 기술

---

## 이벤트 유형 (OBS)

| 이벤트 | 설명 |
|:-------|:-----|
| **Object (객체인식)** | 객체 신원 식별, 자동차 번호판 |
| **BI (비즈니스 인텔리전트)** | 비즈니스 유용 통계정보 |
| **Security (보안)** | 이상 탐지 |

---

## 분석 단계 (배식추이)

| 단계 | 설명 |
|:-----|:-----|
| **배경영역 분리** | Gaussian |
| **객체 식별** | 탐지 객체 식별 |
| **객체 추적** | 칼만 필터, 파티클 필터, CAMSHIFT |
| **이벤트 탐지** | VMS (Video Management System) |

---

## 기술 (전탐추보)

| 기술 | 설명 |
|:-----|:-----|
| **영상 전처리** | 동적 배경 모델링, 세그멘테이션, 영상 보정 |
| **객체 탐지/분류** | 객체 탐지, 객체 보정 |
| **객체 추적** | 단일/다중 카메라 객체 추적 |
| **영상 보안** | 보안 침해방지, 프라이버시 보호 |

---

## 활용 분야

- 교통 정보
- 통과 차량 진행 방향
- 비정상적인 정차 감지
- 특정 지역 침입 감지
- 근거리 화재 또는 연기 검지
