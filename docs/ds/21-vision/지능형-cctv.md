# 지능형 CCTV
{: .fs-8 }

21. 영상처리/영상보안
{: .label .label-purple }

---

## 정의

CCTV 영상에 출현하는 특정 객체, 행위, 소리 등을 **자동으로 감지/분석/식별**하여 이상징후를 탐지하는 차세대 지능형 영상분석 기반 보안기술

---

## 특징

- 영상정보 감지
- 영상정보 검색
- 영상정보 분석
- 24시간 자동 감시
- 네트워크 연결
- 야간 식별

---

## 프로세스

```
감시(배식추이) → 영상전송 → 상황분석 → 판단(안전→무시 / 위험→상황전파)
```

---

## 성능시험 항목

### 민간 분야 (KISA 평가)

| 구분 | 항목 |
|:-----|:-----|
| **필수** | 배회, 침입 |
| **선택** | 유기, 싸움, 방화, 쓰러짐, 마케팅 |

### 연구개발 분야 (미평가)

- 얼굴 인식, 걸음걸이 인식
- 불법 주정차, 중앙선 침범
- 차량 불법 진입

---

## 평가 기준

| 기준 | 설명 |
|:-----|:-----|
| **적응시간** | Adaptation Time |
| **정상검출** | True Positive |
| **미검출** | False Negative |
| **오검출** | False Positive |
