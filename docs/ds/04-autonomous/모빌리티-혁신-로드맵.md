# 모빌리티 혁신 로드맵
{: .fs-8 }

4.4 모빌리티
{: .label .label-purple }

---

## 정의

미래 모빌리티 산업의 발전 방향과 **정부 정책 목표를 담은 중장기 계획**

---

## 한국 모빌리티 혁신 로드맵 비전

**비전**: 세계 최고 수준의 자율주행·모빌리티 서비스 선도국

---

## 주요 추진 과제

| 분야 | 과제 | 내용 |
|:-----|:-----|:-----|
| **자율주행** | Level 4 상용화 | 2027년 완전자율주행 상용화 |
| | 인프라 구축 | C-ITS 전국 확대 |
| **모빌리티 서비스** | MaaS 활성화 | 통합 모빌리티 플랫폼 |
| | UAM 도입 | 도심 항공 모빌리티 |
| **친환경** | 전기차 보급 | 2030년 전기차 450만대 |
| | 충전 인프라 | 전국 충전 네트워크 구축 |

---

## 자율주행 로드맵

| 시기 | 목표 | 주요 내용 |
|:-----|:-----|:---------|
| **~2025** | Level 3 확대 | 고속도로 자율주행 확대 |
| **~2027** | Level 4 도입 | 특정 지역 완전자율주행 |
| **~2030** | Level 4 확산 | 전국 자율주행 서비스 |

---

## UAM 로드맵

| 시기 | 단계 | 내용 |
|:-----|:-----|:-----|
| **~2024** | 실증 | UAM 실증 사업 |
| **~2025** | 상용화 초기 | 제한된 노선 서비스 |
| **~2030** | 성숙기 | 광역 UAM 네트워크 |
