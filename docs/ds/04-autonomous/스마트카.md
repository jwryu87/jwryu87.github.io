# 스마트카 (Smart Car)
{: .fs-8 }

4.4 모빌리티
{: .label .label-purple }

---

## 정의

ICT 기술과 자동차 기술이 융합된 **안전하고 편리한 지능형 자동차**

---

## 스마트카 vs 자율주행차 비교

| 구분 | 스마트카 | 자율주행차 |
|:-----|:---------|:----------|
| **범위** | ICT 융합 자동차 전체 | 운전 자동화에 초점 |
| **핵심 기술** | 커넥티비티, ADAS, 인포테인먼트 | 인지, 판단, 제어 |
| **운전자** | 운전자 중심 | 시스템 중심 (Level 3+) |

---

## 스마트카 핵심 기술

| 분류 | 기술 | 설명 |
|:-----|:-----|:-----|
| **커넥티비티** | V2X | 차량-외부 통신 |
| | 텔레매틱스 | 원격 차량 관리 |
| | OTA | 무선 소프트웨어 업데이트 |
| **ADAS** | ACC | 적응형 크루즈 컨트롤 |
| | LKA | 차선 유지 보조 |
| | AEB | 자동 긴급 제동 |
| **인포테인먼트** | IVI | 차량 내 인포테인먼트 |
| | HUD | 헤드업 디스플레이 |
| | 음성인식 | 음성 명령 제어 |

---

## CASE 트렌드

| 트렌드 | 의미 | 설명 |
|:-------|:-----|:-----|
| **C** | Connected | 연결된 자동차 |
| **A** | Autonomous | 자율주행 |
| **S** | Shared | 공유 모빌리티 |
| **E** | Electric | 전기차 |
