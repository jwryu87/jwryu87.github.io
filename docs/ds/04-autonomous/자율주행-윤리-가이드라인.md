# 자율주행 윤리 가이드라인
{: .fs-8 }

4.1 자율주행 기술
{: .label .label-purple }

---

## 정의

자율주행차가 사고 상황에서 **윤리적 의사결정**을 내리기 위한 **원칙과 지침**

---

## 트롤리 딜레마 (Trolley Problem)

자율주행차가 불가피한 사고 상황에서 **누구를 보호할 것인가**에 대한 윤리적 딜레마

| 선택지 | 내용 |
|:-------|:-----|
| **승객 보호** | 차량 탑승자 우선 보호 |
| **보행자 보호** | 외부 보행자 우선 보호 |
| **최소 피해** | 피해를 최소화하는 방향 선택 |

---

## 윤리 원칙 (독일 윤리위원회)

| 원칙 | 내용 |
|:-----|:-----|
| **인명 보호** | 재산보다 인명 우선 |
| **차별 금지** | 나이, 성별, 인종 등으로 차별 금지 |
| **투명성** | 알고리즘 결정 과정 공개 |
| **책임 소재** | 명확한 책임 주체 규정 |
| **데이터 보호** | 운행 데이터 프라이버시 보호 |

---

## 국가별 윤리 가이드라인

| 국가 | 기관 | 주요 내용 |
|:-----|:-----|:---------|
| **독일** | 윤리위원회 | 20개 윤리 원칙 제시 |
| **미국** | NHTSA | 안전 우선 원칙 |
| **일본** | 내각부 | 사회 수용성 중심 |
| **한국** | 국토부 | 자율주행차 윤리 가이드라인 |

---

## 한국 자율주행차 윤리 가이드라인

| 원칙 | 내용 |
|:-----|:-----|
| **인간 존엄성** | 인간 생명과 안전 최우선 |
| **공공 안전** | 사회 전체의 안전 도모 |
| **합리적 판단** | 예측 가능하고 일관된 판단 |
| **운전자 의무** | 운전자의 주의 의무 유지 |
