## 자율주행 기술요소

### 1) 인지 기술

**① 센서 기술**
- 위치인식 : IMU, GPS
- 물체인식 : Camera, LiDAR, RADAR, 초음파

**② 통신 기술 (V2V, V2I, IVN)**
- 외부통신 : WAVE, C-V2X, CICAS, CALM
- 내부통신 : CAN, LIN, TTCAN

**③ 측위 기술**
- LDM, SLAM

### 2) 판단 기술

- ECU
- 5G MEC
- 딥러닝, 머신러닝

### 3) 제어 기술

**① ADAS** : FCW, ACC, LDWS, LKAS, BSD
**② C-ITS**

### 4) 표준/법규

- **표준** : ISO26262, ISO/PAS 21448, ISO/SAE 21434, SAE J3016
- **법규** : 도로교통법, 자동차관리법
