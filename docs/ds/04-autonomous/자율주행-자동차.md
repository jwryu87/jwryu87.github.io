## 자율주행 기술요소

### 1) 인지 기술

**① 센서 기술**
- 위치인식 : IMU, GPS
- 물체인식 : Camera, LiDAR, RADAR, 초음파

**② 통신 기술 (V2V, V2I, IVN)**
- 외부통신 : WAVE, C-V2X, CICAS, CALM
- 내부통신 : CAN, LIN, TTCAN

**③ 측위 기술**
- LDM, SLAM

### 2) 판단 기술

- ECU
- 5G MEC
- 딥러닝, 머신러닝

### 3) 제어 기술

**① ADAS** : FCW, ACC, LDWS, LKAS, BSD
**② C-ITS** : 협력 지능형 교통 시스템

---

## C-ITS (Cooperative ITS)

> 암기: **서비스** (협전안교 긴약대)

### 구성요소

| 구분 | 기술요소 |
|:-----|:---------|
| **차량** | V2V, GPS, WAVE, OBU, V2X |
| **도로** | V2I, RSU, 보행감지기, 지능형CCTV |
| **센터** | 도로공사, 빅데이터, AI, 현장지원 |

### C-ITS 서비스 (협전안교 긴약대)

| 서비스 | 내용 |
|:-------|:-----|
| **협력형 교통관리** | 위치기반 차량데이터 수집, 위치기반 교통정보 제공 |
| **전자지불** | 요금징수 시스템 |
| **안전운전 지원** | 도로위험구간 정보제공, 차량 추돌방지 지원, 노면기상정보제공 |
| **교차로 통행지원** | 교차로 신호위반 위험 경고, 우회전 안전운행 지원 |
| **긴급상황 지원** | 긴급차량 접근 경고, 차량 긴급상황 경고 |
| **교통약자 보호** | 옐로우 버스 운행안내, 스쿨 존 속도 제어, 보행자 충돌방지 경고 |
| **대중교통** | 버스 운행 관리 |

### ITS vs C-ITS 비교

| 구분 | ITS | C-ITS |
|:-----|:----|:------|
| **정보수집** | 단방향 | 양방향 |
| **목적** | 교통관리 | 교통안전 |
| **기술** | 영상인식 | V2V, V2I |
