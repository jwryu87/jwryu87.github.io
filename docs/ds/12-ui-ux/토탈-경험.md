# 토탈 경험 (Total Experience, TX)
{: .fs-8 }

12. UI/UX
{: .label .label-purple }

> 암기: 전략구성 4가지 - **사고복임** (사용자경험, 고객경험, 복합경험, 임직원경험)

---

## 정의

고객경험(CX), 직원경험(EX), 사용자경험(UX), 멀티경험(MX)의 **기술적 형태를 하나로 통일**시켜서 비즈니스를 혁신하는 패러다임

---

## 등장 배경

| 배경 | 설명 |
|:-----|:-----|
| **X(Experience) 흐름 변화** | 경험 중심 패러다임으로 전환 |
| **디지털 혁신 고도화** | 기술 융합 가속화 |
| **언택트 환경 가속화** | 비대면 서비스 확대 |

---

## 전략 구성 (사고복임)

| 구분 | 영문 | 설명 |
|:-----|:-----|:-----|
| **사용자경험** | UX (User Experience) | 제품/서비스 사용 경험 |
| **고객경험** | CX (Customer Experience) | 전체 고객 여정 경험 |
| **복합경험** | MX (Multi Experience) | 다양한 채널 경험 |
| **임직원경험** | EX (Employee Experience) | 직원 업무 경험 |

---

## 실현 방안

### 전략 관점

| 방안 | 설명 |
|:-----|:-----|
| **CX vs UX 구분** | 고객 경험과 사용자 경험의 차이 구분 |
| **개인화 도입** | 경험 최적화된 개인화 도입 |
| **개인화 엔진 최적화** | 맞춤형 서비스 제공 |

### 조직 관점

| 방안 | 설명 |
|:-----|:-----|
| **사일로(Silo) 허물기** | 조직 간 벽 제거 |
| **직원 경험 브랜딩** | 내부 고객 경험 강화 |
| **다중 채널 커뮤니케이션** | 옴니채널 소통 |
