# 협업 필터링 (Collaborative Filtering)
{: .fs-8 }

20. 추천 시스템
{: .label .label-purple }

> 암기: 프로세스 - **측선알추저** / 유사도 알고리즘 - **유마코피자**

---

## 정의

많은 사용자들로부터 누적된 대규모 데이터를 활용하여 이를 분류하고, 새로운 데이터에 대입하여 **향후 관심사를 분석/예측**하는 기술

---

## 프로세스 (측선알추저)

```
유사도 측정 → 항목 선택 → 알고리즘 적용 → 추천 수행 → History 저장
```

---

## 유형 (기모)

| 유형 | 방식 | 설명 |
|:-----|:-----|:-----|
| **기억 기반** | 사용자 기반 | 유사한 사용자가 선호하는 아이템 추천 |
| | 아이템 기반 | 유사한 아이템 추천 |
| | Hybrid | 사용자+아이템 결합 |
| **모델 기반** | 군집화/베이즈 | 나이브 베이즈, K-means |
| | 데이터 마이닝 | 패턴 분석 |
| | AI/ML 혼합 | 딥러닝 모델 |

---

## 유사도 알고리즘 (유마코피자)

| 알고리즘 | 설명 |
|:---------|:-----|
| **유클리디안 거리** | 점과 점 간 최단거리 |
| **마할라노비스 거리** | 두 집단 사이의 거리 |
| **코사인 유사도** | 코사인 사용, 벡터 간 유사도 |
| **피어슨 유사도** | 상관관계 해석 |
| **자카드 유사도** | 교집합/합집합으로 판단 |
