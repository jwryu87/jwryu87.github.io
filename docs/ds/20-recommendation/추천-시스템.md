# 추천 시스템 (Recommendation System)
{: .fs-8 }

20. 추천 시스템
{: .label .label-purple }

> 암기: 알고리즘 - **컨협딥** (콘텐츠 기반, 협업 필터링, 딥러닝)

---

## 정의

개인 맞춤형 서비스를 제공하기 위해 사용자의 구매패턴 등의 과거 데이터를 분석하여 **개인에 맞는 상품을 추천**하는 시스템

---

## 알고리즘 (컨협딥)

| 알고리즘 | 설명 |
|:---------|:-----|
| **콘텐츠 기반 필터링 (CB)** | 아이템 속성 기반 유사 아이템 추천 |
| **협업 필터링 (CF)** | 사용자 기반, 아이템 기반 |
| **딥러닝 융합 모델** | ML/DL 기반 추천 |

---

## 평가 지표 (비품)

### 비즈니스/서비스 관점

- PV (Page View)
- CTR (Click Through Rate)

### 품질 관점

- 연관성 (Relevance)
- 다양성 (Diversity)
- 새로움 (Novelty)
- 불예측성 (Serendipity)
- 커버리지 (Coverage)

---

## Cold Start Problem

| 문제 | 설명 |
|:-----|:-----|
| **정의** | 신규 유저/아이템, 비인기 상품, 비로그인 유저 등 데이터 부재로 추론 불가 |
| **해결방법** | 하이브리드 필터링, CBF→CF 단계적 적용, 인기 아이템 추천 |
