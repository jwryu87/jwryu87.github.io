# u-Health (Ubiquitous Health)
{: .fs-8 }

10. 디지털 헬스케어
{: .label .label-purple }

> 암기: 서비스구성 - **측전분피** (측정, 전송, 분석, 피드백)

---

## 정의

네트워크 접속을 통해 원격의료기술을 이용하여 **언제 어디서나 예방, 진단, 치료의 건강관리를 받을 수 있는** 건강/의료 서비스

---

## 서비스 구성 (측전분피)

| 단계 | 구성요소 | 설명 |
|:-----|:---------|:-----|
| **센싱 및 측정** | 의료기기 센서 | 심전계, 뇌파계, 근전계, 혈압계, 맥박계 |
| | 모바일 기기 | HW 센서 |
| **취합 및 전송** | 통신 기술 | BCN, Zigbee, WiFi, 4G, Bluetooth LE |
| **분석** | 분석 기법 | 주성분 분석기법, 영상처리기법 |
| **피드백** | 알림 | 주기적 이메일 발송, 휴대폰 푸시 |

---

## 의료기관 시스템 (EOP PHD)

| 시스템 | 설명 |
|:-------|:-----|
| **PACS** | 의료영상저장전송시스템 |
| **OCS** | 처방전달시스템 |
| **PHR** | 개인건강기록 |
| **EHR/EMR** | 전자건강기록/전자의무기록 |
| **DICOM** | 의료영상 표준 |
| **HL7** | 의료정보 교환 표준 |

---

## 응용환경에 따른 기술구분

| 환경 | 서비스 |
|:-----|:-------|
| **U-Hospital** | 원격진료, 의료스마트카드, 병원정보/예약관리 |
| **홈/모바일 헬스케어** | 모바일 원격진료, 위치추적 서비스 |
| **웰니스** | 모바일 스트레스 관리, u-Fitness, 운동량 체크 |
