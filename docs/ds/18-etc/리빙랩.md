# 리빙랩 (Living Lab) / SOS랩 (Solution in Our Society Lab)
{: .fs-8 }

18. DS 기타
{: .label .label-purple }

> 암기: **리빙랩(운탐실평)** / **SOS랩(조개구실공사)**

---

## 핵심 키워드

- **리빙랩**: 운영 기획, 대안 탐색, 대안 실험, 대안 평가, 피드백 (**기탐실평피**)
- **SOS랩**: 조직화, 개념화, 구체화, 실체화, 공유화, 사업화

---

## 정의/개념

| 구분 | 정의 |
|:-----|:-----|
| **리빙랩** | 시민참여, **사용자 주도 연구소** |
| **SOS랩** | 지역사회 문제 해결, SW 사용, **사회문제 연구소** |

---

## 리빙랩 (Living Lab)

### 프로세스

```
    ┌─────────────────────────────────────────────────────────────┐
    │                    피드백 및 개선                            │
    │  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐        │
    │  │운영 기획 │→│대안 탐색 │→│대안 실험 │→│대안 평가 │        │
    │  │(planning)│ │(exploration)│(experimentation)│(evaluation)│
    │  └─────────┘  └─────────┘  └─────────┘  └─────────┘        │
    └─────────────────────────────────────────────────────────────┘
                  ↑                                   
           설계, 개발 공동 창조(co-creation)           
```

### 단계별 활동

| 단계 | 활동 | 설명 |
|:-----|:-----|:-----|
| **운영 기획 (Planning)** | - 사회문제 구체화 및 비즈니스 모델 탐색, 추진체계 설계<br>- 사용자 그룹 선정 및 참여 활성화, 리빙랩 운영 환경 형성 | 문제 정의 및 계획 수립 |
| **대안 탐색 (Exploration)** | - 아이디어 발굴, 개념화(공동 설계) | 해결책 아이디어 발굴 |
| **대안 실험 (Experimentation)** | - 프로토타입 개발, 프로토타입 테스트/피드백 | 시제품 개발 및 테스트 |
| **대안 평가 (Evaluation)** | - 제품,서비스 개발<br>- 프로토타입 실증, 확산<br>- 제품 피드백 | 결과물 평가 및 확산 |

---

## SOS랩 (Solution in Our Society Lab)

### 프로세스

```
조직화 → 개념화 → 구체화 → 실체화 → 공유화 → 사업화
                      ↓
                   문제 해결
                      ↓
                   가치 창출
```

### 단계별 활동

| 단계 | 영문 | 설명 |
|:-----|:-----|:-----|
| **조직화** | Creating Organization | - 혁신 조직(중간지원조직 구성 및 구축), 자원 확보(커뮤니티/이해관계자 자원 확보) |
| **개념화** | Concept | - 가치와 목표(핵심가치 도출, 미래 탐색)<br>- 문제발굴(문제발굴 접근방식 결정), 공감(관찰, 페르소나 설정)<br>- 문제정의(이해관계자 거버넌스 구성, 5Why 인터뷰, 명목집단) |
| **구체화** | Co-design | - 아이디에이션(자료조사, 솔루션 탐색), 프로토타입(컨셉 설계, 자원과 기술 탐색) |
| **실체화** | Co-creation | - 공동창조(RFP, 기업선정, 거버넌스), 컨셉 디자인(프로토타이핑, 단위테스트/검증) |
| **공유화** | Commonization | - 실증 및 확산(필드테스트, 교차실증), 데이터 관리(DB관리) |
| **사업화** | Commercialization | - 비즈니스 모델(BM)(사업화 모델링, 펀딩), 지원 프로그램(Scale-up, 기업지원 프로그램) |

---

## 리빙랩 vs SOS랩 비교

| 구분 | 리빙랩 | SOS랩 |
|:-----|:-----|:-----|
| **초점** | 사용자 주도 연구 | 사회문제 해결 |
| **참여자** | 시민, 사용자 | 지역사회, SW 개발자 |
| **핵심 특징** | 피드백 및 개선 반복 | 6단계 체계적 접근 |
| **두음매직** | 운탐실평 | 조개구실공사 |

---

## 핵심 원칙

| 원칙 | 설명 |
|:-----|:-----|
| **공동 창조 (Co-creation)** | 사용자와 전문가가 기획, 탐색, 실험, 평가 과정에서 공동으로 설계 및 개발 수행 |
| **사용자 중심** | 실제 사용 환경에서 사용자 참여를 통한 문제 해결 |
| **반복적 개선** | 피드백을 통한 지속적인 개선 |
