# OLAP (On-Line Analytical Processing)
{: .fs-8 }

18. DS 기타
{: .label .label-purple }

---

## 핵심 키워드

`다차원 분석` `큐브` `슬라이싱` `다이스` `드릴다운` `롤업` `차원` `피벗`

---

## 정의/개념

대량의 데이터를 **다차원적으로 분석**하고, 다양한 각도에서 데이터를 조회 및 분석하여 **의사결정에 필요한 정보를 제공**하는 기술

---

## 구성요소

| 구성요소 | 설명 | 예시 |
|:-----|:-----|:-----|
| **차원 (Dimensions)** | 분석할 데이터의 관점을 제공하는 요소 | 시간, 지역, 제품 등 |
| **측정값 (Measures)** | 분석하고자 하는 데이터를 수치적으로 표현하는 요소 | 매출, 수량, 이익 등 |
| **계층 (Hierarchies)** | 차원 내에서 데이터의 집계 수준을 나타내는 구조 | 연도 → 분기 → 월 → 일 |

---

## OLAP 기술 (연산)

| 기술 | 설명 |
|:-----|:-----|
| **큐브 (Cube)** | 데이터의 다차원적 표현을 가능하게 하는 데이터 구조 |
| **슬라이싱 (Slicing)** | 큐브의 **특정 면을 선택**하여 분석하는 작업 |
| **다이스 (Dicing)** | 큐브의 **특정 부분을 잘라내어** 상세히 분석하는 작업 |
| **드릴다운 (Drill-Down)** | 데이터의 **세부 항목으로** 상세히 분석하는 작업 (상위→하위) |
| **롤업 (Roll-Up)** | 데이터를 **더 높은 수준으로 집계**하여 분석하는 작업 (하위→상위) |
| **피벗 (Pivot)** | 데이터의 **축을 회전**하여 다른 관점에서 분석 |

---

## OLAP 유형

| 유형 | 설명 |
|:-----|:-----|
| **MOLAP**<br>(Multidimensional OLAP) | 데이터를 **OLAP 큐브로 구성**, 가장 일반적인 유형<br>빠른 조회 성능, 저장 공간 필요 |
| **ROLAP**<br>(Relational OLAP) | 큐브로 재구성하지 않고 **관계형 데이터베이스**의 데이터를 직접 처리<br>유연성 높음, 조회 성능 상대적 낮음 |
| **HOLAP**<br>(Hybrid OLAP) | MOLAP과 ROLAP의 **하이브리드**<br>데이터를 OLAP 큐브로 구성하지만, 관계형 DB와 연계 |

## OLAP vs OLTP 비교

| 구분 | OLAP | OLTP |
|:-----|:-----|:-----|
| **목적** | 분석, 의사결정 지원 | 업무 처리, 트랜잭션 |
| **데이터** | 대량, 이력 데이터 | 현재 데이터 |
| **질의** | 복잡한 분석 질의 | 단순한 CRUD |
| **응답시간** | 수 초 ~ 수 분 | 수 초 이내 |
| **사용자** | 분석가, 경영진 | 업무 담당자 |
