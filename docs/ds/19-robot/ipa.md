# IPA (Intelligent Process Automation)
{: .fs-8 }

19. 로봇/자동화
{: .label .label-purple }

> 암기: 핵심기술 - **로스기자인** / 세부기술 - **프데언**

---

## 정의

명확한 규칙에 기반한 RPA와 **머신러닝, 신경망, 인지기술이 접목**되어 스스로 판단하면서 업무를 수행하는 **지능적 프로세스 자동화 기술**

---

## 특징

- 효율성 향상
- 직원 생산성 및 성능 향상
- 운영 위험 감소
- 응답 시간 개선
- 개선된 고객 여정 경험

---

## 핵심 기술 (로스기자인)

| 기술 | 설명 |
|:-----|:-----|
| **로보틱 프로세스 자동화** | RPA |
| **스마트 워크플로우** | Smart Workflow |
| **기계학습/고급분석** | Machine Learning & Advanced Analytics |
| **자연어 처리** | NLG |
| **인지 에이전트** | Cognitive Agents |

---

## 세부 기술 (프데언)

| 분류 | 기술 |
|:-----|:-----|
| **프로세싱 기술** | RPA, BPM, Process Mining |
| **텍스트/데이터 기술** | OCR, Machine Learning, Data Analytics |
| **언어 기술** | NLP, Chatbot, STT |

---

## RPA vs IPA 비교

| 구분 | RPA | IPA |
|:-----|:----|:----|
| **대상** | 정형 데이터 | 비정형 데이터 |
| **판단** | 규칙 기반 | AI 기반 자율 판단 |
| **범위** | 단순 반복 업무 | 복잡한 업무 |
