# 멀티 에이전트 협업 기술
{: .fs-8 }

19. 로봇/자동화
{: .label .label-purple }

---

## 정의

복수 에이전트 간의 상호작용 방식에 따라 집합적, 협력적, 협업적, 조정 방식으로 **서로 이득을 볼 수 있는 상호작용**을 하는 분산된 인공지능

---

## 협업 유형

### 물체 공간 인식 협업 (SLAM)

| 항목 | 설명 |
|:-----|:-----|
| **기술** | SLAM (Simultaneous Localization and Mapping) |
| **특징** | 이웃한 에이전트 간 제한된 통신으로 지도 만들기 |

### 분산 행동 지능 협업

| 항목 | 설명 |
|:-----|:-----|
| **특징** | 에이전트들의 개별 행동 결과로 최종 결과 도출 |
| **기능** | 에이전트의 행동이나 의사결정 수행 |

### 언어 기반 인간-AIbot 협업

| 항목 | 설명 |
|:-----|:-----|
| **특징** | 챗봇과 대화 맥락 유지, 에이전트 역할 조율 |

### 분산 협업 학습

| 학습 유형 | 설명 |
|:----------|:-----|
| **강화학습** | 학습 공유, 실행은 자신의 관측정보만 활용 |
| **전이학습** | 에이전트 전이 |
| **연합학습** | 분산된 데이터 활용 없이 모델 계수만 공유 |
