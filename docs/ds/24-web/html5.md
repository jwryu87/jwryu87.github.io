# HTML5
{: .fs-8 }

24. 웹 기술/검색엔진
{: .label .label-purple }

> 암기: 주요기능 - **폼소캔백워스 비오프지디**

---

## 정의

비디오, 오디오 등 다양한 부가기능과 콘텐츠 등을 **추가적인 Plugin 없이** 브라우저에서 쉽게 개발 및 표현 가능한 **차세대 HTML 표준**

---

## 주요 기능 (폼소캔백워스 비오프지디)

| 기능 | 설명 |
|:-----|:-----|
| **Webform** | 향상된 폼 요소 |
| **Websocket** | 실시간 양방향 통신 |
| **Canvas** | 2차원 그래픽 |
| **SVG** | 벡터 그래픽 |
| **Web Worker** | 백그라운드 스레드 |
| **Web Storage** | 로컬 저장소 |
| **비디오/오디오** | 멀티미디어 |
| **오프라인 웹 앱** | 오프라인 지원 |
| **지오로케이션** | 위치 정보 |
| **Web SQL DB** | 클라이언트 DB |

---

## HTML5 vs Flash 비교

| 구분 | HTML5 | Flash |
|:-----|:------|:------|
| **플러그인** | 불필요 | 필요 |
| **모바일** | 지원 | 미지원 |
| **SEO** | 양호 | 취약 |
| **표준** | W3C 표준 | Adobe 독점 |
