# Web 3.0
{: .fs-8 }

24. 웹 기술/검색엔진
{: .label .label-purple }

> Web 2.0 **공개참** (공유, 개방, 참여, 협력) → Web 3.0 **탈중앙화**

---

## 정의

웹 2.0을 넘어 차세대 웹으로 **탈중앙화, 탈중앙 자율조직, 개방성, 참여 보상, 데이터 소유권** 등을 제공하는 웹

- 플랫폼 경제 (2.0) → **토큰 경제 (3.0)**

---

## 특징

| 특징 | 설명 |
|:-----|:-----|
| **탈중앙** | 인프라, 서비스 탈중앙화 |
| **탈중앙 자율조직** | 자동화를 통한 자율적 의사결정 |
| **개방성** | 오픈소스, 기술융합 |
| **참여 보상** | 참여 기여에 따른 투명한 보상 |
| **데이터 소유권** | 자기주권 데이터 관리, 탈중앙 저장 |

---

## 상세 기술

| 분류 | 기술 |
|:-----|:-----|
| **탈중앙화** | 블록체인, DAO, 암호화폐 |
| **데이터 소유권** | NFT, Dapp |
| **보안성** | 스마트 컨트랙트, 엣지컴퓨팅 |
| **지능화** | 초거대 AI, 시맨틱웹, 상황인식, 개인화, Agent |
| **확장 미디어** | 메타버스, VR, AR, XR |

---

## Web 1.0 → 2.0 → 3.0 비교

| 구분 | Web 1.0 | Web 2.0 | Web 3.0 |
|:-----|:--------|:--------|:--------|
| **특징** | 읽기 전용 | 읽기/쓰기 | 읽기/쓰기/소유 |
| **핵심** | 정보 제공 | 참여/공유 | 탈중앙/소유권 |
| **경제** | - | 플랫폼 경제 | 토큰 경제 |
