## 정의

웹 상의 구조화된 메타데이터들 간의 표준화를 위한 **XML 및 tag 기반의 Framework**

---

## 구성 (자특서)

| 구성요소 | 설명 | 예시 |
|:---------|:-----|:-----|
| **자원 (Resource)** | 기술 대상 | HTML 문서, XML 엘레멘트, 책/사람/웹페이지 |
| **특성 (Property)** | 자원을 기술하는 특징 | 책의 저자, 주제, 크기 |
| **서술 (Statements)** | 리소스가 가질 수 있는 값 | 저자명, 주제명 |

---

## RDF Triple

```
Subject (주어) - Predicate (서술어) - Object (목적어)
```

예시: "책A - 저자 - 홍길동"
