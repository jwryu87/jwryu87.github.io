# Semantic Web (시맨틱 웹)
{: .fs-8 }

24. 웹 기술/검색엔진
{: .label .label-purple }

> 암기: 계층구조 - **UTSRU** (UI, Trust, SPARQL, RDF, URI)

---

## 정의

정보와 자원 사이의 관계 의미 정보(Semanteme)를 **기계가 처리할 수 있는 온톨로지 형태로 표현**하고, 이를 자동화된 기계가 처리하도록 하는 기술

---

## 특징

- 메타데이터 (XML, RDF)
- 온톨로지 (Ontologies)
- 논리적 추론
- Agent 활용

---

## 계층 구조 (UTSRU)

| 계층 | 설명 |
|:-----|:-----|
| **UI/Application** | 사용자 인터페이스 |
| **Trust/Proof, Logic** | 신뢰, 증명, 논리 (보안 적용) |
| **SPARQL, OWL, RDFS, RIF** | 질의 및 온톨로지 언어 |
| **RDF/XML** | 데이터 모델 |
| **URI** | 자원 식별자 |

---

## 기술 요소

| 분류 | 기술 |
|:-----|:-----|
| **수집** | Agent, Crawler |
| **분석** | Ontology, SPARQL, OWL, GPT3 |
| **정의** | RDF, XML, URI |
