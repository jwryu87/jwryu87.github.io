## 핵심 키워드

`서버 가상화` `분산 스토리지` `병렬처리` `빌링` `확장성 컴퓨팅`

---

## 정의/개념

인터넷을 통해 **컴퓨팅 자원**(서버, 스토리지 등)을 필요에 따라 사용하고, **사용한 만큼 비용을 지불하는 서비스**

- **On Demand**: 원하는 즉시 사용
- **Pay per Use**: 사용량 기반 과금 (Utility Computing)

---

## 구성요소

| 구성요소 | 설명 |
|:---------|:-----|
| 유저 인터랙션 I/F | 사용자와 인터페이스 |
| 서비스 카탈로그 | 사용할 서비스 리스트 |
| System Management | 사용 리소스 관리 |
| 프로비저닝 | 자원 할당 |
| 모니터링 툴 | 모니터링, 자원사용량, 비용 |
| 서버 | 관리되는 가상 또는 물리서버 |

---

## 6대 핵심기술 (서대가C분병)

> 암기: **서**대가**씨**/분**병** → 서대가C분병

| 기술 | 설명 |
|:-----|:-----|
| 1) 서버 가상화 기술 | 물리 서버를 가상화 |
| 2) 대용량 클라우드 컴퓨팅 인프라 기술 | 대규모 인프라 구축 |
| 3) 가상서버 관리 기술 | VM 관리 및 운영 |
| 4) CDN 기술 | 콘텐츠 분산 전송 |
| 5) 분산 스토리지 기술 | 분산 저장 시스템 |
| 6) 병렬처리 기술 | 대용량 데이터 병렬 처리 |

---

## 9대 주변기술 (오모빌네/보스피가모)

| 기술 | 설명 |
|:-----|:-----|
| 1) Open API 기술 | API 공개 및 연동 |
| 2) 모니터링 기술 | 시스템 모니터링 |
| 3) 빌링 기술 | 과금 체계 |
| 4) 네트워크 가상화 기술 | SDN, NFV |
| 5) 클라우드 서비스 보안기술 | 보안 관리 |
| 6) SLA 관리 기술 | 서비스 수준 관리 |
| 7) P2V, V2V 기술 | 물리↔가상 전환 |
| 8) 가상 클라우드 기술 | 가상화 환경 |
| 9) 모바일 클라우드 기술 | 모바일 연동 |

## 구축방법

| 단계 | 내용 |
|:-----|:-----|
| 1. BaseLine | 환경분석, 현황분석, 방향수립 |
| 2. 사업타당성조사 | 벤치마킹, 법제도 |
| 3. 구축전략수립 | 아키텍처, 컨설팅 |

| 5. 사업운용 | 서비스확대, 고도화 |
