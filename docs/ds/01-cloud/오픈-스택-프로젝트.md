## 핵심 키워드

`IaaS 오픈소스` `REST API` `Python` `Apache 2.0` `Nova` `Neutron`

---

## 정의/개념

**IaaS 형태의 클라우드 컴퓨팅 오픈소스 프로젝트**로, 데이터센터의 컴퓨팅, 스토리지, 네트워크 리소스들을 대시보드를 통해 관리하는 클라우드 운영 체제

---

## 필요성

| 항목 | 설명 |
|:-----|:-----|
| Lock-in 방지 | 특정 벤더 종속성 탈피 |
| 기능개선 공유 | 오픈소스 커뮤니티를 통한 기능 개선 및 공유 |
| 클라우드 간 이동 편리 | 표준화된 API를 통한 클라우드 간 마이그레이션 용이 |

---

## 특징

| 특징 | 설명 |
|:-----|:-----|
| REST API | RESTful API 기반 통신 |
| HW 제한 없음 | 다양한 하드웨어 지원 |
| 다양한 형태 구축 | Public, Private, Hybrid 클라우드 구축 가능 |
| 열린 설계/개발 | Python 기반, Apache 2.0 라이선스 |

---

## 구성요소 (서스사관네비)

> 📌 암기팁: **서**버 **스**토리지 **사**용자 **관**리 **네**트워크 **DB**

| 구분 | 프로젝트 | 설명 |
|:-----|:---------|:-----|
| **서버** | Nova | 가상머신 관리, 컴퓨팅 서비스 |
|  | Glance | 가상머신 이미지 관리 |
| **스토리지** | Swift | 대용량 오브젝트 스토리지, 복제 업데이트 |
|  | Cinder | 블록 스토리지 서비스 |
| **사용자** | Keystone | 중앙집중식 인증, ID 관리 |
|  | Horizon | 웹 기반 대시보드 UI |
| **관리** | Heat | 오케스트레이션, 컴포넌트 관리 |
| **네트워크** | Neutron | 가상 네트워크, SDN 지원 |
| **DB** | Trove | RDBMS 관리 서비스 |

## 주요 프로젝트 상세

### Nova (Compute)
- 가상 서버 생성, 관리, 삭제
- 다양한 하이퍼바이저 지원 (KVM, Xen, VMware 등)

### Neutron (Network)
- 가상 네트워크 관리 (VLAN, VXLAN)
- SDN 컨트롤러 연동
- 로드밸런서, 방화벽, VPN 지원

### Swift (Object Storage)
- 비정형 데이터 저장
- 높은 가용성, 자동 복제

### Cinder (Block Storage)
- 블록 스토리지 볼륨 제공
- VM에 영구 저장소 연결
