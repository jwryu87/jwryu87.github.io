# CASB (Cloud Access Security Broker)
{: .fs-8 }

1.3 클라우드 보안
{: .label .label-purple }

---

## 핵심 키워드

`클라우드 접근 보안` `DLP` `IAM` `Shadow IT` `SASE`

---

## 정의/개념

**클라우드 서비스 사용자와 클라우드 서비스 제공자 사이에 위치**하여, 기업의 보안 정책을 적용하고 클라우드 사용을 통제하는 **보안 중개 솔루션**

---

## 기능요소 (접데로이 위암D)

> 📌 암기팁: **접**근 **데**이터 **로**깅 **이**상 **위**험 **암**호화 **D**RM

| 기능 | 설명 |
|:-----|:-----|
| **접근통제** | 사용자/디바이스별 클라우드 접근 제어 |
| **데이터 손실방지 (DLP)** | 민감 데이터 유출 탐지 및 차단 |
| **로깅/감사** | 클라우드 사용 내역 기록 및 감사 |
| **이상탐지** | 비정상적인 사용 패턴 탐지 |
| **위험평가** | 클라우드 서비스 위험도 평가 |
| **암호화** | 데이터 암호화 적용 |
| **DRM** | 디지털 권한 관리 |

---

## 유형 (AAPP)

| 유형 | 구분 | 설명 |
|:-----|:-----|:-----|
| **SW형** | API형 | API 연동을 통한 클라우드 제어 |
|  | 에이전트형 | 단말기에 에이전트 설치 |
| **HW형** | 프라이빗 | Outbound 트래픽 체크 위치에 설치 |
|  | 퍼블릭 | 클라우드 내부에 배치 |

## SASE와의 관계

CASB는 **SASE(Secure Access Service Edge)**의 핵심 보안 요소로 포함됨

| SASE 구성요소 | 설명 |
|:-------------|:-----|
| CASB | 클라우드 접근 보안 중개 |
| SWG | 웹 보안 게이트웨이 |
| ZTNA | 제로 트러스트 네트워크 액세스 |
| FWaaS | 서비스형 방화벽 |

---

## 도입 효과

| 효과 | 설명 |
|:-----|:-----|
| Shadow IT 가시성 | 승인되지 않은 클라우드 사용 탐지 |
| 데이터 보호 | 민감 데이터 유출 방지 |
| 규정 준수 | 컴플라이언스 요구사항 충족 |
| 위협 방어 | 클라우드 기반 위협 탐지/차단 |
