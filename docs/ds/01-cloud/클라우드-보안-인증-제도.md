# 클라우드 보안 인증 제도
{: .fs-8 }

1.3 클라우드 보안
{: .label .label-purple }

---

## 핵심 키워드

`CSAP` `KISA` `공공부문` `보안등급`

---

## 정의/개념

**클라우드 컴퓨팅 서비스 보안인증제도(CSAP)**로, 국내 클라우드 서비스 제공자의 **정보보호 수준을 평가하여 인증**하는 제도

> 💡 상세 내용은 [CSAP](/docs/ds/01-cloud/csap) 참조

---

## 법적근거

| 법령 | 조항 | 내용 |
|:-----|:-----|:-----|
| 클라우드컴퓨팅법률 | 제23조 2항 | 클라우드 서비스 보안인증 근거 |
| 클라우드컴퓨팅법 | 제5조 | 클라우드 컴퓨팅 발전 기본계획 |

---

## 인증 유형

| 유형 | 대상 | 평가 수준 |
|:-----|:-----|:----------|
| **IaaS** | 인프라 서비스 | 전체 통제항목 |
| **SaaS 표준** | 상위 등급 SaaS | 전체 통제항목 |
| **SaaS 간편** | 중/하위 등급 SaaS | 핵심 통제항목 |
| **DaaS** | 데스크톱 서비스 | 전체 통제항목 |

---

## 인증 절차

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 인증신청  │───▶│ 서류심사  │───▶│ 현장심사  │───▶│ 인증서   │
│          │    │          │    │          │    │   발급   │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
```

---

## 평가 영역

| 영역 | 주요 항목 |
|:-----|:----------|
| **관리체계** | 정보보호 정책, 조직, 위험관리 |
| **물리적 보안** | 데이터센터 보안, 출입통제 |
| **기술적 보안** | 접근통제, 암호화, 취약점 관리 |
| **운영 보안** | 운영관리, 사고대응, 재해복구 |
| **가상화 보안** | 하이퍼바이저 보안, VM 격리 |
