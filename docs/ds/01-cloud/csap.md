# CSAP (Cloud Security Assurance Program)
{: .fs-8 }

1.3 클라우드 보안
{: .label .label-purple }

---

## 핵심 키워드

`클라우드 보안인증` `KISA` `공공부문` `SaaS` `IaaS`

---

## 정의/개념

**클라우드 컴퓨팅 서비스 보안인증제도**로, 국내 클라우드 서비스 제공자의 정보보호 수준을 평가하여 인증하는 제도

---

## 법적근거 (2325)

> 📌 암기팁: **23**조 2항, **5**조

| 법령 | 조항 | 내용 |
|:-----|:-----|:-----|
| 클라우드컴퓨팅법률 | 제23조 2항 | 클라우드 서비스 보안인증 근거 |
| 클라우드컴퓨팅법 | 제5조 | 클라우드 컴퓨팅 발전 기본계획 |

---

## 인증체계

| 주체 | 역할 |
|:-----|:-----|
| **이용자** | 공공기관 포함 클라우드 서비스 이용자 |
| **신청기관** | 클라우드 서비스 제공자 (CSP) |
| **인증위원회/평가기관** | 한국인터넷진흥원 (KISA) |
| **공공부문 기술자문기관** | 국가보안기술연구소 |
| **정책기관** | 과학기술정보통신부 |

---

## 평가/인증 절차

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 최초평가  │───▶│ 사후평가  │───▶│ 사후평가  │───▶│ 갱신평가  │
│  (1년)   │    │ (년 1회) │    │ (4회)   │    │  (1년)   │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
      │                                              │
      │              인증 유효기간 5년               │
      └──────────────────────────────────────────────┘
```

| 인증유형 | 사후평가 횟수 |
|:---------|:-------------|
| 표준등급 | 4회 (년 1회) |
| 간편등급 | 2회 |

---

## 인증 등급

| 등급 | 대상 | 평가항목 |
|:-----|:-----|:---------|
| **표준등급** | IaaS, SaaS(상) | 전체 통제항목 |
| **간편등급** | SaaS(중), SaaS(하) | 핵심 통제항목 |

---

## 개편사항 (2024년~)

| 구분 | 기존 | 개편 후 |
|:-----|:-----|:--------|
| 평가기관 | KISA 단독 | KISA 및 복수 평가기관 |
| 비용 | 무료 | 유료 |
| 평가방식 | 전 과정 평가기관 수행 | 사업자 자체수행 후 평가 |
| 평가범위 | 전 과정 신규평가 | 변경사항 중점평가 |

---

## 평가 영역

| 영역 | 주요 항목 |
|:-----|:----------|
| 관리체계 | 정보보호 정책, 조직, 자산관리 |
| 물리적 보안 | 데이터센터, 출입통제 |
| 기술적 보안 | 접근통제, 암호화, 취약점 관리 |
| 운영 보안 | 운영관리, 사고대응, 재해복구 |
