# 분산 클라우드 (Distributed Cloud)
{: .fs-8 }

1.1 클라우드 개념
{: .label .label-purple }

---

## 핵심 키워드

`Edge` `지연 최소화` `데이터 주권` `클라우드렛` `가트너`

---

## 정의/개념

**퍼블릭 클라우드 서비스를 서로 다른 물리적 위치에 배포**하는 반면, 서비스의 **운영, 거버넌스, 업데이트 및 진화는 원래 퍼블릭 클라우드 제공 업체가 책임**지는 클라우드 모델

> 💡 가트너가 정의한 2020년 10대 전략 기술 트렌드 중 하나

---

## 구성 (코리에디)

> 📌 암기팁: **코**어 **리**저널 **에**지 **디**바이스

```
┌─────────────────────────────────────────────────────────┐
│                  Core Cloud (코어 계층)                  │
│              최상위, 데이터 운영 제공                     │
├─────────────────────────────────────────────────────────┤
│                Regional Cloud (리저널 계층)              │
│              Proxy 및 배포 기능                          │
├─────────────────────────────────────────────────────────┤
│           Edge Cloud / Cloudlet (에지 계층)              │
│         신속 제공, 클라우드렛(소규모 클라우드)            │
├─────────────────────────────────────────────────────────┤
│                    Device (디바이스)                     │
│                      사용자                              │
└─────────────────────────────────────────────────────────┘
```

| 계층 | 역할 | 특징 |
|:-----|:-----|:-----|
| **Core Cloud** | 중앙 클라우드 | 최상위, 데이터 운영 제공 |
| **Regional Cloud** | 지역 클라우드 | Proxy 및 배포 기능 |
| **Edge Cloud** | 에지 클라우드 | 신속 제공, 클라우드렛(소규모 클라우드) |
| **Device** | 디바이스 | 사용자, IoT 기기 |

**기타**: 협력적 분산클라우드 운영 = 서비스 배치, 오케스트레이션

---

## 분산 클라우드 모델 (ECRC)

| 모델 | 구성 | 설명 |
|:-----|:-----|:-----|
| **EC-CC** | Device → EC → CC | 에지 → 코어 직접 연결 |
| **RC-CC** | Device → NW → RC → CC | 네트워크 → 리저널 → 코어 |
| **EC-RC-CC** | Device → EC → RC → CC | 에지 → 리저널 → 코어 (전체 계층) |

---

## 특징

| 특징 | 설명 |
|:-----|:-----|
| **지연 최소화** | 사용자와 가까운 위치에서 처리 |
| **데이터 주권** | 지역별 데이터 규제 준수 |
| **중앙 관리** | CSP가 전체 인프라 통합 관리 |
| **일관된 서비스** | 위치와 무관한 동일한 서비스 품질 |

---

## 멀티 클라우드 vs 분산 클라우드

| 구분 | 멀티 클라우드 | 분산 클라우드 |
|:-----|:-------------|:-------------|
| 운영 주체 | 여러 CSP | 단일 CSP |
| 관리 | 분리된 관리 | 통합 관리 |
| 목적 | 벤더 종속 방지 | 지연 최소화, 규제 준수 |
| 일관성 | 낮음 | 높음 |

---

## 활용 사례

| 분야 | 활용 |
|:-----|:-----|
| **IoT** | 에지에서 데이터 전처리 |
| **게임** | 저지연 게임 서버 |
| **금융** | 지역별 규제 준수 |
| **헬스케어** | 실시간 의료 데이터 처리 |
