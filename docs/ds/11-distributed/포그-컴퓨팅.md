# 포그 컴퓨팅 (Fog Computing)
{: .fs-8 }

11. 분산 컴퓨팅
{: .label .label-purple }

---

## 핵심 키워드

`시스코` `로컬네트워크-클라우드 중계` `메시형 토폴로지` `Fog Nodes`

---

## 정의/개념

**시스코**, 로컬네트워크-클라우드 중계, **메시형 토폴로지 컴퓨팅 기술**

- 클라우드와 엣지 디바이스 사이의 중간 계층에서 데이터 처리
- 시스코(Cisco)에서 제안한 분산 컴퓨팅 패러다임

---

## 아키텍처

```
          ┌─────────────────────────────────────┐
          │      CLOUD | Data Centers           │  ← Thousands
          │           (클라우드)                 │
          └─────────────────────────────────────┘
                           ↑
          ┌─────────────────────────────────────┐
          │         FOG | Nodes                 │  ← Millions
          │         (포그 노드)                  │
          └─────────────────────────────────────┘
                           ↑
          ┌─────────────────────────────────────┐
          │        EDGE | Devices               │  ← Billions
          │     (엣지 디바이스/IoT)              │
          │   📱 🏠 🚗 📷 🏭 💡 ...              │
          └─────────────────────────────────────┘
```

| 계층 | 설명 | 규모 |
|:-----|:-----|:-----|
| **Cloud** | 데이터 센터, 대용량 처리 | Thousands |
| **Fog** | 중간 계층, 로컬 처리 | Millions |
| **Edge** | IoT 디바이스, 데이터 생성 | Billions |

---

## 구성요소

| 구분 | 구성요소 | 핵심 |
|:-----|:-----|:-----|
| **인프라** | Fog Device | 포그 계층 디바이스 |
| | Fog Nodes | 포그 노드 (게이트웨이, 라우터 등) |
| | Cloud/Data Center | 클라우드 데이터 센터 |
| **기술** | 네트워크 패브릭 | 메시형 네트워크 연결 |
| | 가상화 | 리소스 가상화 |
| | 머신러닝 협력 알고리즘 | 분산 학습 및 처리 |

---

## 엣지 컴퓨팅 vs 포그 컴퓨팅 비교

| 구분 | 엣지 컴퓨팅 | 포그 컴퓨팅 |
|:-----|:-----|:-----|
| **처리 위치** | 디바이스 근접 (1 hop) | 네트워크 중간 계층 |
| **제안자** | 다양한 벤더 | 시스코(Cisco) |
| **네트워크 구조** | 분산형 | 메시형 토폴로지 |
| **주요 역할** | 실시간 처리 | 클라우드-엣지 중계 |
| **확장성** | 제한적 | 높음 |

---

## 포그 컴퓨팅 특징

| 특징 | 설명 |
|:-----|:-----|
| **저지연** | 클라우드보다 가까운 위치에서 처리 |
| **대역폭 절감** | 필요한 데이터만 클라우드로 전송 |
| **메시형 구조** | 노드 간 협력 처리 가능 |
| **위치 인식** | 지리적 위치 기반 서비스 제공 |
