# 모바일 엣지 컴퓨팅 (Mobile Edge Computing, MEC)
{: .fs-8 }

11. 분산 컴퓨팅
{: .label .label-purple }

---

## 핵심 키워드

`트래픽 오프로드` `가상화 레이어` `코드 오프로드` `엣지 컨텐츠` `MEC 플랫폼`

---

## 정의/개념

무선 기지국에 **분산 클라우드 컴퓨팅 적용**, 서비스 지연 최소화 **분산 컴퓨팅 기술**

- MEC는 단말장치의 근접한 곳에서 처리되면 레이턴시 감소
- 무선 기지국(Smart Cell)에 MEC Platform을 배치하여 서비스 제공

---

```

│  └─────────┘                 │     Management          │   │
│       ↓                      │       System            │   │
│   backhaul     Hosting Infra │                         │   │

│  │ Core N/O│                                               │
│  └─────────┘                                               │
└─────────────────────────────────────────────────────────────┘
```

## 구성요소

| 구분 | 구성요소 | 핵심 |
|:-----|:-----|:-----|
| **MEC 전송기술** | 트래픽 오프로드 | 백홀 구간 트래픽 오프로드 |
| **MEC 호스팅 인프라** | 하드웨어 자원 | 스위치, 스토리지, 서버 |
| | 가상화 레이어 | 추상화 제공 |
| **MEC 응용 플랫폼** | 가상화 관리자 | IaaS 도구 제공 |
| | 응용플랫폼 서비스 | 미들웨어 서비스, 인프라 서비스 제공 |

---

## MEC vs 네트워크 슬라이싱 비교

| 비교 | 모바일 엣지 컴퓨팅 | 네트워크 슬라이싱 |
|:-----|:-----|:-----|
| **원리** | 네트워크 단말 설치 | 논리적 망분리 |
| **특징** | 부하분산, 초저지연 | QoS 기반 품질 관리 |
| **주요기술** | 코드 오프로드, 엣지 컨텐츠 | SDN, NFV |

---

## MEC 활용 분야

| 분야 | 설명 |
|:-----|:-----|
| **V2X** | 자율주행 차량 간 통신 저지연 지원 |
| **AR/VR** | 실시간 렌더링 및 스트리밍 |
| **스마트팩토리** | 실시간 제조 공정 제어 |
| **게임** | 클라우드 게이밍 저지연 서비스 |
