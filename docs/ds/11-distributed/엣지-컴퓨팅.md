# 엣지 컴퓨팅 (Edge Computing)
{: .fs-8 }

11. 분산 컴퓨팅
{: .label .label-purple }

---

## 핵심 키워드

`Intelligence Layer` `Actionable Layer` `Data Source` `Decision Making` `Smart Device` `1 hop`

---

## 정의/개념

**1 hop**, 데이터 근접거리 **실시간 분산 처리 컴퓨팅 기술**

- 데이터가 생성되는 위치(엣지)에서 직접 처리하여 지연 시간 최소화
- 클라우드로 전송할 데이터를 사전 처리하여 네트워크 부하 감소

---

## 아키텍처

```
┌─────────────────────────────────────────────────────────────┐
│                    Intelligence Layer                       │
│              (머신러닝, 클라우드 데이터 센터)                  │
│                          ↑                                  │
│                     클라우드                                 │
│                     분산 트랜잭션                            │
│                          ↑                                  │
├─────────────────────────────────────────────────────────────┤
│                    Actionable Layer                         │
│                 (정책, Data 미가공 데이터)                    │
│                          ↑                                  │
│                   클라우드엣지(엣지)                          │
│                          ↑                                  │
│                      실시간                                  │
│                          ↑                                  │
├─────────────────────────────────────────────────────────────┤
│                      Data Source                            │
│           IoT    자동차    모바일     대응                    │
└─────────────────────────────────────────────────────────────┘
```

---

## 구성요소

| 구분 | 구성요소 | 핵심 |
|:-----|:-----|:-----|
| **Intelligence Layer** | 머신러닝 | 지도, 비지도, 강화 학습 |
| | 클라우드 데이터 센터 | 주요 데이터 전송 및 처리 |
| **Actionable Layer** | Decision Making | 데이터 이용 의사 결정 |
| | Data Application | 최적 정보 획득 |
| **Data Source** | Smart Device | 데이터 생성 디바이스 |
| | Network | 고속데이터 전송, 저지연 |

---

## 장/단점

| 구분 | 내용 |
|:-----|:-----|
| **장점** | 대기시간 감소 |
| | 비용 절감 |
| **단점** | 데이터 보전성 |
| | 데이터 해킹 |

---

## 클라우드 vs 엣지 컴퓨팅 비교

| 구분 | 클라우드 컴퓨팅 | 엣지 컴퓨팅 |
|:-----|:-----|:-----|
| **처리 위치** | 중앙 데이터센터 | 데이터 생성 위치 근처 |
| **지연 시간** | 상대적으로 높음 | 매우 낮음 (1 hop) |
| **대역폭** | 많은 데이터 전송 필요 | 필요한 데이터만 전송 |
| **실시간성** | 제한적 | 실시간 처리 가능 |
| **활용 사례** | 대용량 데이터 분석 | IoT, 자율주행, 실시간 서비스 |
