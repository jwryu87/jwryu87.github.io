# 디지털 지갑 (Digital Wallet)
{: .fs-8 }

22. 디지털 지갑/FinOps
{: .label .label-purple }

> 암기: 기능 - **인전금** (인증, 전자서명, 금융서비스)

---

## 정의

인증, 전자서명, 금융서비스 등 **디지털 신분증을 담는 공간**

---

## 기능 (인전금)

| 기능 | 설명 |
|:-----|:-----|
| **사용자 인증** | 본인 확인 |
| **전자서명** | 디지털 서명 |
| **금융서비스** | 결제, 송금 등 |

---

## 형태 (사위)

| 형태 | 설명 |
|:-----|:-----|
| **사용자 기기 설치형** | 스마트폰 등 기기에 직접 설치 |
| **위임형 (클라우드)** | 클라우드 기반 관리 |

---

## DID 연계 기술

| 기술 | 설명 |
|:-----|:-----|
| **DIDs** | 분산 식별자 |
| **DKMS** | 분산 키 관리 시스템 |
| **블록체인** | 분산원장 기술 |
| **자기주권** | 데이터 자기 관리 |
| **Credential Issue** | 자격증명 발급 |
| **Verification** | 검증 |
