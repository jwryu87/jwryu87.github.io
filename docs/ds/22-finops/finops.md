# FinOps (Finance + DevOps)
{: .fs-8 }

22. 디지털 지갑/FinOps
{: .label .label-purple }

> 암기: 원칙 - **협책단보가변** (협업, 책임의식, 단일팀, 보고, 가치, 변동비용)

---

## 정의

조직이 최대의 비즈니스 가치를 얻을 수 있게 해주는 **클라우드 재무관리에 관한 업무절차이자 문화적 관행**

- 클라우드 재무관점: CapEx → OpEx

---

## 단계

```
정보화 → 최적화 → 운용
```

---

## 원칙 (협책단보가변)

| 원칙 | 설명 |
|:-----|:-----|
| **협업** | 팀 간 협력 |
| **책임의식** | 비용 책임 |
| **중앙 단일팀** | FinOps 전담 조직 |
| **보고** | 투명한 비용 보고 |
| **가치** | 비즈니스 가치 중심 |
| **변동비용** | 클라우드 변동비용 모델 활용 |

---

## 이해관계자

| 역할 | 설명 |
|:-----|:-----|
| **경영진** | 전략적 의사결정 |
| **PO** | 제품 비용 관리 |
| **엔지니어/운영팀** | 리소스 최적화 |
| **재무/구매팀** | 예산 및 계약 관리 |
| **FinOps 실무** | 전담 실무 조직 |
