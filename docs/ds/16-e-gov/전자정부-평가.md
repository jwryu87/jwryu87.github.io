# 전자정부 평가 (UN E-Government Survey)
{: .fs-8 }

16. 전자정부
{: .label .label-purple }

> 암기: **온정인** (온라인서비스-정보통신인프라-인적자본) / **제참결** (정보제공-정책참여-정책결정)

---

## 핵심 키워드

- **전자정부 발전지수**: 온라인 서비스 지수, 정보통신 인프라 지수, 인적자본 지수 (**온정인**)
- **온라인 참여지수**: 온라인 정보제공, 온라인 정책참여, 온라인 정책결정 (**제참결**)

---

## 정의/개념

유엔 경제사회처에서 **2년마다 193개 전체 회원국 대상** 전자정부 참여 및 발전 부문 수준 **평가 제도**

---

## 평가지수 종류

### 전자정부 발전지수 (EGDI: E-Government Development Index)

| 구분 | 평가지수 | 설명 |
|:-----|:-----|:-----|
| **전자정부 발전지수** | 온라인서비스 지수<br>(Online Service Index) | 국민들에게 서비스를 제공하는 정부의 **일반적인 능력** 측정 |
| | 정보통신인프라 지수<br>(Telecommunication Infrastructure Index) | 전자정부의 선행조건인 **기본 정보통신관련 인프라** 수준 측정 |
| | 인적자본 지수<br>(Human Capital Index) | 전자정부를 이용할 수 있는 **국민의 일반적 지적능력** 측정을 통해 국민의 전자정부서비스에 대한 수용도 반영 |

### 온라인 참여지수 (EPI: E-Participation Index)

| 구분 | 평가지수 | 설명 |
|:-----|:-----|:-----|
| **온라인 참여 지수** | 온라인 정보제공<br>(E-Information) | 정부기관 웹사이트를 통해 국민들이 원하는 **정보에 쉽게 접근**할 수 있도록 하는 정부의 정보제공 수준 측정 |
| | 온라인 정책참여<br>(E-Consultation) | 정부기관 웹사이트를 통해 정부가 국민들과 **직접 의사소통**할 수 있는 창구 및 기능 평가 |
| | 온라인 정책결정<br>(E-Decision Making) | 정부가 정책결정 과정에서 **국민들의 의견을 수렴·반영**하는 수준 및 노력 측정 |

---

## 평가지수 요약

| 대분류 | 지수 | 핵심 |
|:-----|:-----|:-----|
| **전자정부 발전지수 (EGDI)** | 온라인서비스 지수 (OSI) | 서비스 제공 능력 |
| | 정보통신인프라 지수 (TII) | 인프라 수준 |
| | 인적자본 지수 (HCI) | 국민 수용도 |
| **온라인 참여지수 (EPI)** | E-Information | 정보 접근성 |
| | E-Consultation | 쌍방향 소통 |
| | E-Decision Making | 의견 반영 |
