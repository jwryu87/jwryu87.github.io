## 📖 등장배경

```
💡 과거: 내가 힘들게 만든 발명품/창작물을 남이 그냥 베껴도 막을 방법이 없었음

😢 문제: "내가 1년 걸려 만든 건데, 남이 복사해서 돈 벌면 어떡해?"

✅ 해결: 지식재산권 제도 등장! 
   → 내 아이디어/창작물에 법적 권리 부여
   → 일정 기간 독점적으로 사용할 수 있게 보호
   → 창작 의욕 ↑ 기술 발전 ↑
```

## 📝 개념 정의

> **지식재산권**이란?
> 
> 🧠 사람의 **지적 창작 활동**으로 만들어진  
> 💎 **무형의 재산**에 대해  
> 📜 법으로 인정해주는 **독점적 권리**

### 쉬운 비유

| 유형의 재산 (땅, 집) | 무형의 재산 (아이디어) |
|:------------------|:--------------------|
| 🏠 내 집에 남이 못 들어오게 함 | 💡 내 발명을 남이 못 베끼게 함 |
| 부동산 등기부등본 | 특허등록증, 저작권등록 |

---

## 🏗️ 기술요소 (3가지 그룹)

### 그룹 1: 산업재산권 (공장·회사에서 쓰는 것)

| 종류 | 뭘 보호하나? | 쉬운 예시 |
|:----|:-----------|:---------|
| **특허권** | 새로운 발명 | 💊 신약, 🔋 배터리 기술 |
| **실용신안권** | 작은 개선 발명 | 🖊️ 더 편한 볼펜 구조 |
| **디자인권** | 예쁜 모양·디자인 | 📱 스마트폰 외관 디자인 |
| **상표권** | 브랜드 로고·이름 | 🍎 애플 로고, ✓ 나이키 |

### 그룹 2: 저작권 (예술가·창작자가 쓰는 것)

| 종류 | 뭘 보호하나? | 쉬운 예시 |
|:----|:-----------|:---------|
| **저작권** | 음악·영화·소설·그림 | 🎵 BTS 노래, 📚 해리포터 |

### 그룹 3: 신지식재산권 (IT 시대 새로운 것)

| 종류 | 뭘 보호하나? | 쉬운 예시 |
|:----|:-----------|:---------|
| **반도체배치설계권** | 칩 내부 설계 | 🔲 삼성 반도체 설계도 |
| **영업비밀** | 회사 비밀 정보 | 🍗 KFC 레시피 |
| **데이터베이스권** | DB 만든 노력 | 📊 네이버 지도 데이터 |
| **컴퓨터프로그램권** | 소프트웨어 | 💻 게임, 앱 코드 |

---

## ⭐ 차별점 키워드 (답안지 가산점!)

{: .highlight }
> **TRIPs (무역관련 지식재산권 협정)**
> 
> WTO 회원국이 지켜야 하는 **지식재산권 국제 표준**
> 
> - 전 세계가 같은 기준으로 지식재산권 보호
> - 한국도 가입 → 해외에서도 내 발명 보호받음

├─────────────────┼───────────────┼───────────────────────┤
│ 📍 특허청 관할   │ 📍 저작권위원회│ 📍 각 관련 기관       │

## 📋 6하원칙 요약

| 구분 | 내용 |
|:-----|:----|
| **누가(Who)** | 발명가, 창작자, 기업 |
| **언제(When)** | 창작·발명 완성 시점부터 |
| **어디서(Where)** | 특허청, 저작권위원회 (등록) |
| **무엇을(What)** | 발명, 디자인, 창작물, 소프트웨어 등 |
| **왜(Why)** | 창작 의욕 고취, 기술 발전 촉진 |
| **어떻게(How)** | 출원·등록 → 일정 기간 독점권 부여 |
