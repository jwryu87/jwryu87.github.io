# 애플리케이션 가상화
{: .fs-8 }

7.1 가상화 기본
{: .label .label-purple }

---

## 정의

**추상 계층(Abstract Layer)**을 구성하고 이 위에 응용 프로그램이 올라가는 형태의 가상화 기술

---

## 응용 프로그램 가상화 vs 프레젠테이션 가상화

| 구분 | 응용 프로그램 가상화 | 프레젠테이션 가상화 |
|:-----|:-------------------|:-------------------|
| **동작 위치** | 클라이언트 | 서버 |
| **전송 방식** | 애플리케이션 다운로드 | 화면만 전송 |
| **자원 사용** | 클라이언트 자원 | 서버 자원 |
| **네트워크 부하** | 초기 배포 시만 | 지속적 |

---

## 응용 프로그램 가상화 동작 방식 (배포 방식)

| 단계 | 설명 |
|:-----|:-----|
| 1. 응용프로그램 설치 | 서버에 가상 환경으로 패키징 |
| 2. 사용 요청 | 클라이언트에서 사용 요청 |
| 3. 가상환경 배포 | 패키지 다운로드 및 가상환경 구성 |
| 4. 프로그램 사용 | 클라이언트에서 실행 |
| 5. 프로그램 종료 | 가상환경 정리 |

---

## 프레젠테이션 가상화 동작 방식 (원격 접속 방식)

| 단계 | 설명 |
|:-----|:-----|
| 1. 프로그램 설치 | 서버에 애플리케이션 설치 |
| 2. 원격 접속 | 클라이언트에서 서버 접속 |
| 3. 프로그램 실행 | 서버에서 실행, 화면만 전송 |

---

## 일반 프로그램 동작 방식

| 단계 | 설명 |
|:-----|:-----|
| 1. 프로그램 실행 | 실행 파일 호출 |
| 2. 프로세스 생성 | OS가 프로세스 생성 |
| 3. 메모리 적재 | 코드/데이터 메모리에 로드 |
| 4. 실행 대기 | Ready Queue 대기 |
| 5. CPU 할당 | 스케줄러에 의해 CPU 할당 |
| 6. 명령어 수행 | CPU에서 명령어 실행 |
