## 가상화 종류

| 개념도 | 가상화 종류 | 설명 |
|:-------|:-----------|:-----|
| 프레젠테이션 가상화 ↓ 응용 프로그램 | **프레젠테이션 가상화** | - 애플리케이션을 중앙 서버에 설치하고 가상의 인터페이스만 네트워크를 통해 보내는 기술<br>- 모든 작업을 서버에서 처리하고 그 화면(이미지)만 PC에 전송하는 방식 |
| 응용프로그램 가상화 ↓ 운영 체제 ↓ 하드웨어 가상화 ↓ 하드웨어 | **응용 프로그램 가상화** | - 응용 프로그램 구성 계층을 OS에서 분리시키는 기술<br>- 추상 계층(Abstract Layer)을 구성하고 이 위에 응용 프로그램이 올라가는 형태로 동작<br>- 추상의 계층에서 프로그램에 대한 고유의 환경을 구성하여 실제 운영 체제엔 영향을 주기 않음 |
|  | **하드웨어 가상화** | - 소프트웨어를 사용하여 물리적 컴퓨터를 실행하는 가상 컴퓨터 |

---

## 응용 프로그램 가상화의 장점

- **복잡한 관리 감소**: 애플리케이션 배포/관리 간소화
- **유연한 확장성**: 필요에 따라 애플리케이션 확장 용이
- **보안성 강화**: 격리된 환경으로 보안 향상

---

## 서버 가상화

| 구분 | 내용 |
|:-----|:-----|
| **정의** | CPU, 메모리, 하드디스크 등 서버 자원 가상화 제공 |
| **특징** | 온디맨드, 실시간성, 확장성, 다중 수용성 |
| **구성** | HW > Host OS > Hypervisor > Guest OS > App |
| **유형** | 하이퍼바이저(전가상화, 반가상화), OS레벨가상화 |

---

## 스토리지 가상화

| 구분 | 내용 |
|:-----|:-----|
| **정의** | 스토리지 자원을 편리한 파일/볼륨으로 보이도록 해주는 기술 |
| **발전** | DAS → NAS → SAN → 스토리지 가상화(이기종 통합) |
| **기술요소** | LUN(Logical Unit Number), RAID, NAS, SAN |

---

## 네트워크 가상화

| 기술 | 설명 |
|:-----|:-----|
| **SDN** | Software Defined Networking |
| **NFV** | Network Function Virtualization |
| **네트워크 슬라이싱** | 5G 네트워크 분리 |
| **특화망** | 기업/산업용 전용 네트워크 |

---

## 데스크톱 가상화 (VDI)

| 구분 | 내용 |
|:-----|:-----|
| **정의** | 서버기반 컴퓨팅, 가상 데스크톱 제공, 공용스토리지 |
| **유형** | 서버기반 컴퓨팅, VDI(가상 데스크톱 관리) |
| **프로토콜** | RDP(윈도우), RGS(HP개발, 고성능), X윈도우(유닉스) |
