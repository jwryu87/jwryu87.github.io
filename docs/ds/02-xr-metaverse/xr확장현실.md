2.2 XR

> 암기: **기술요소** (데오표인)

---

## XR 유형

| 유형 | 설명 | 사례 |
|:-----|:-----|:-----|
| **AR (Augmented Reality)** | 증강현실, 실세계 + 가상이미지 | 구글글래스, 포켓몬GO |
| **VR (Virtual Reality)** | 가상현실, 100% 가상 | VR게임, 건축물 재현 |
| **MR (Mixed Reality)** | 혼합현실, 현실객체 + 가상객체 | 자동차/기계 설계, 의료/수술 |

---

## 기술요소 (데오표인)

| 분류 | 기술 | 설명 |
|:-----|:-----|:-----|
| **데이터** | 교정(Calibration) | 영상 보정 |
| | 트래킹(Tracking) | 위치/동작 계산 |
| | 마커(Marker) | 좌표 획득 |
| **오감기술** | 청각, 촉각, 후각 | 몰입감 향상을 위한 감각 재현 |
| **표현기술** | 디스플레이 | 화면 출력 장치 |
| | 화면처리 3D | 3차원 화면 처리 |
| | 그래픽 구현(렌더링) | 실시간 그래픽 렌더링 |
| **인터랙션** | HCI | 인간-컴퓨터 상호작용 |
| | 동작 DOF | 자유도 기반 동작 인식 |
| | 음성인식/자연어처리 | 음성 기반 상호작용 |
