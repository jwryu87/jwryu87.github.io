> 암기: **특징** (지상몰), **기술/서비스** (콘기서)

---

## 특징 (지상몰)

| 특징 | 영문 | 설명 |
|:-----|:-----|:-----|
| **지능화** | Intelligent | 사용자 맥락 이해 및 지능적 반응 |
| **상호작용** | Interactive | 사용자와 콘텐츠 간 양방향 소통 |
| **몰입감** | Immersive | 현실감 있는 몰입 경험 제공 |

---

## 주요 기술 및 서비스 (콘기서)

### 실감형 콘텐츠

| 분류 | 설명 |
|:-----|:-----|
| 실감 미디어 | 고품질 영상/음향 콘텐츠 |
| 실감 커뮤니케이션 | 원격 회의, 아바타 소통 |
| 실감 라이프 | 일상 생활 속 실감 경험 |

### 실감형 기술

| 기술 | 설명 |
|:-----|:-----|
| 디지털 트윈 | 현실 세계의 가상 복제 |
| VR, AR, MR | 확장현실 기술 |
| 공간 3D | 3차원 공간 표현 |
| 엣지 컴퓨팅 | 지연 최소화 처리 |
| 플렌옵틱 | 광선 필드 카메라 기술 |
| 오감 기술 | 시각 외 감각 재현 |
| 홀로그램 | 3D 입체 영상 |
| 디바이스 기술 | HMD, 컨트롤러 등 |
| 5G | 초고속, 초저지연 통신 |
| AI | 지능형 처리 및 생성 |

### 실감형 서비스

| 분야 | 설명 |
|:-----|:-----|
| 공공 서비스 | 교육, 안전, 문화 등 |
| 산업 서비스 | 제조, 설계, 훈련 등 |
| 과학기술 서비스 | 연구, 시뮬레이션 등 |

## 핵심기술 (XR의 데오표인)

| 분류 | 기술요소 |
|:-----|:---------|
| 데이터 | 교정, 트래킹, 마커 |
| 오감기술 | 청각, 촉각, 후각 |

| 인터랙션 | HCI, DOF, 음성인식 |
