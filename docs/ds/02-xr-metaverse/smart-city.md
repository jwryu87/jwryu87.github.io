## 주요기술

### 인프라 기술

| 분류 | 기술 |
|:-----|:-----|
| 도시 인프라 | 스마트인프라, CITS, 지능형 재해관리 |
| ICT 인프라 | 유무선통신 인프라, 스마트팜, 가상시뮬레이션 |
| 공간정보 인프라 | 지리정보 기술, 공간정보 플랫폼, 고정밀지도 |

### 데이터 기술

| 분류 | 기술 |
|:-----|:-----|
| IoT 데이터 | 각종 센서 데이터, IoT플랫폼, 웨어러블 기기 데이터 |
| 데이터 공유 | 데이터허브, 거래구조 모델링, 플랫폼 프로모션 |

### 서비스 기술

| 분류 | 기술 |
|:-----|:-----|
| 융복합 서비스 | 스마트헬스, 지능형로봇, 무인기 ICT |
| 도시문제 해결 서비스 | 제로에너지주택, 공공안전/재해예방 ICT, 블록체인행정 |

## 스마트시티 모델 4가지 유형

| 모델 | 설명 |
|:-----|:-----|
| **OBM** (Open Business Model) | 개방형 비즈니스 모델 |
| **BOM** (Build, Operate, Manage) | 건설-운영-관리 |
| **BOT** (Build, Operate, Transfer) | 건설-운영-이전 |
| **BOO** (Build, Own, Operate) | 건설-소유-운영 |
