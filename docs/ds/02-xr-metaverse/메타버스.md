# 메타버스 (Metaverse)
{: .fs-8 }

2.1 메타버스
{: .label .label-purple }

> 암기: **5C**(세창디일연), **유형**(증라미가)

---

## 정의/개념

**가상과 현실 공진화**, 사회, 경제, 문화 활동, **초연결**, **초지능**, **초실감** 세계

---

## 메타버스 특징 5C (정보처리학회, 2021)

| 특징 | 설명 |
|:-----|:-----|
| **Canon (세계관)** | 설계자와 참여자에 의해 메타버스의 시공간이 채워지며 확장 |
| **Creator (창작자)** | 공간, 게임, 아바타 등을 누구나 만들고 확장하는 창작자로서 역할 |
| **Currency (디지털통화)** | 메타버스 내에서 생산·소비뿐만 아니라 화폐의 저장·교환 가능 |
| **Continuity (일상의 연장)** | 메타버스에서의 활동이 일상과 동시에 지속적으로 진행 |
| **Connectivity (연결)** | 시공간의 연결, 세계관의 연결, 사람과 사람(아바타)의 연결 |

---

## 메타버스 유형

### 1단계: 영역별 핵심속성을 중심으로 발전

| 유형 | 설명 |
|:-----|:-----|
| **증강현실 (Augmented Reality)** | XR 기반 |
| **라이프로깅 (Lifelogging)** | SNS 기반 |
| **거울세계 (Mirror Worlds)** | Digital Twin 기반 |
| **가상세계 (Virtual Worlds)** | 가상(Simulation) 기반 |

### 2단계: Metaverse 2.0 "초세계"

다양한 속성이 융합되어 **"초세계"**로 융합
- 초연결
- 초실감
- 초지능
