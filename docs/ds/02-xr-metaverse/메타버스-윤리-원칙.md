## 핵심 키워드

`온전한 자아` `안전한 경험` `지속가능한 경험` `진정성` `자율성` `호혜성` `사생활 존중` `공정성` `개인정보 보호` `포용성` `책임성`

---

## 정의/개념

**안전하고 신뢰할 수 있는 메타버스 구현**을 위한 **윤리 원칙**

---

## 3대 지향가치

| 지향가치 | 핵심사항 | 설명 |
|:---------|:---------|:-----|
| **온전한 자아** | 자율적 행동 | 메타버스에서 모든 개인은 선택한 삶의 가치에 충실하고 자율적으로 행동해야 한다 |
| **안전한 생태계** | 안전한 생태계 조성 | 사회는 구성원들이 메타버스를 충분히 경험할 수 있도록 안전한 생태계를 조성해야 한다 |
| **지속가능한 번영** | 미래세대 지속 | 메타버스의 편익과 혜택을 모두가 누릴 수 있도록 의도적으로 배제되지 않아야 하며 미래세대에게도 지속되어야 한다 |

---

## 8대 실천원칙

| 실천원칙 | 핵심사항 | 설명 |
|:---------|:---------|:-----|
| **진정성 (Authenticity)** | 진실된 자아 발현 | - 개발 주체는 참여자의 진정성이 발현되는 도구 지원<br>- 운영 주체는 진정성이 침해되는 경우 점검과 개선 노력<br>- 이용 주체는 내면화한 가치와 행위가 정합한가 숙고 |
| **자율성 (Autonomy)** | 능동적, 자발적, 동기와 결정 | - 개발과 운영 주체는 참여와 행동의 선택권 보장<br>- 이용 주체는 참여자의 자율성이 침해될 가능성과 참여 현실 자아의 자율성에 미칠 수 있음을 인지 |
| **호혜성 (Reciprocity)** | 우호적, 협력적, 상호작용의 발산 | - 개발과 운영 주체는 호혜적 소통을 지원하는 상호작용 강안<br>- 이용 주체는 소통에 있어 긍정적인 호혜성을 내재화 노력 |
| **사생활 존중 (Respect for Privacy)** | 가상공간 내 사적 영역의 존중 | - 개발과 운영 주체는 사적 영역이 보호될 수 있도록 노력<br>- 이용 주체는 가상공간에서도 자신과 타인의 사적 영역 인식 |
| **공정성 (Fairness)** | 창의적 역량, 범위의 기회 | - 개발과 운영 주체는 창작, 수익활동 차별없는 기회 보장<br>- 이용 주체는 창작과 수익활동 참여시 타인 권리 침해 주의 |
| **개인정보 보호 (Personal Information Protection)** | 개인정보와 정보 주체의 권리보장 | - 개발과 운영 주체는 프라이버시 중심 설계 염두<br>- 이용 주체는 자신의 부주의로 자신과 타인의 데이터가 유출될 수 있음을 인지 |
| **포용성 (Inclusiveness)** | 다양한 존중과 접근성 증진 | - 개발과 운영 주체는 특정 집단이 배제되지 않도록 접근과 활용이 용이한 환경 조성<br>- 이용 주체는 가상자아의 사회경제적 다양성을 인식하고 포용적 문화적 토대 구축 노력 |
| **책임성 (Responsibility for future)** | 지속가능한 메타버스 구현 노력 | - 개발과 운영 주체는 현 세대의 메타버스가 미래 세대의 가치 형성에 미치는 영향을 고려<br>- 이용 주체는 가상공간 내 행위의 결과가 누적되어 미래 메타버스에 영향을 미침을 고려 |
