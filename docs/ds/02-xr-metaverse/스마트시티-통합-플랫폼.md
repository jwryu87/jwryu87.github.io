# 스마트시티 통합 플랫폼
{: .fs-8 }

2.3 스마트시티
{: .label .label-purple }

---

## 구성

| 구성요소 | 기능 | 설명 |
|:---------|:-----|:-----|
| **통합연계** | 외부시스템 연계 | 타 시스템과의 데이터 연동 |
| | 내부모듈 정보전달 | 플랫폼 내 모듈 간 데이터 공유 |
| **통합관제** | 대시보드 구성 | 현황 시각화 |
| | 모니터링/제어 | 실시간 모니터링 및 원격 제어 |
| **통합플랫폼DB** | 공통DB | 공통 데이터 저장소 |
| | 메타DB | 메타데이터 관리 |
| **통합DB** | 공공서비스 DB | 공공 서비스 데이터 저장 |
| **통신규격** | OneM2M | IoT 표준 프로토콜 |
| | OpenAPI | 개방형 API 인터페이스 |

---

## 통합 플랫폼 아키텍처

```
┌─────────────────────────────────────────────────┐
│                  통합관제                        │
│        (대시보드, 모니터링/제어)                  │
├─────────────────────────────────────────────────┤
│                  통합연계                        │
│     (외부시스템 연계, 내부모듈 정보전달)          │
├────────────────────┬────────────────────────────┤
│   통합플랫폼DB      │        통합DB              │
│  (공통DB, 메타DB)   │    (공공서비스 DB)          │
├────────────────────┴────────────────────────────┤
│               통신규격                           │
│          (OneM2M, OpenAPI)                      │
└─────────────────────────────────────────────────┘
```
