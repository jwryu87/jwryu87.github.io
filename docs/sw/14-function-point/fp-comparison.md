# 기능점수 간이법과 정통법 비교
{: .fs-8 }

기능점수
{: .label .label-purple }

---

## 핵심 키워드

- **간이법**: 데이터(ILF, ELF), 트랜잭션(EO, EI, EQ)
- **정통법**: 데이터(DET, RET), 트랜잭션(DET, FTR)

---

## 정의/개념

- **간이법**: 유형별 평균 복잡도, SW 전 단계 기능 점수 산정 기법
- **정통법**: 기능별 정확한 복잡도, 분석/설계 단계, 기능 점수 산정 기법

---

## 기능점수의 간이법과 정통법 관계

```
          ┌──────────────────────────────────────────────────┐
          │                    개발비                         │
          │                    산정                          │
          └────────────────────────┬─────────────────────────┘
                                   │
         ┌─────────────────────────┼─────────────────────────┐
         ▼                         ▼                         ▼
    ┌─────────┐              ┌─────────┐              ┌─────────┐
    │  기획   │              │  예산   │              │ 분석/   │──── 개발
    │        │              │  수립   │              │ 설계    │     이후
    └─────────┘              └─────────┘              └─────────┘
         │                                                  │
         ▼                                                  ▼
    간이법 적용                                         정통법 적용
    
    - 적용시점에 따라 간이법, 정통법 분리 적용
```

---

## 간이법과 정통법 설명

### 간이법

| 항목 | 내용 |
|:-----|:-----|
| **개념** | - 유형별 평균 복잡도 계산, 기획/예산 수립단계 거략적 측정 기능 점수 산정 기법 |
| **측정** | - 데이터: ILF, ELF 측정 |
| **항목** | - 트랜잭션: EO, EI, EQ 측정 |
| **적용시점** | - 기획/예산 - 비용 산정 초기 단계 |
| **장점** | - 소요시간, 지식습득 시간 절감 및 측정 |

### 정통법

| 항목 | 내용 |
|:-----|:-----|
| **개념** | - 기능별 정확한 복잡도 계산, 분석/설계 단계 정확도 높은 측정 기능 점수 산정 기법 |
| **측정** | - 데이터: DET, RET 측정 |
| **항목** | - 트랜잭션: DET, FTR 측정 |
| **적용시점** | - 분석/설계 - 개발 시작 단계 적용 |
| **장점** | - 정확한 기능 점수 산출 위해 정통법 사용 |

---

## 간이법과 정통법 비교

| 비교 | 간이법 | 정통법 |
|:-----|:-------|:-------|
| **적용시점** | 기획/예산수립 | 분석/설계 이후 |
| **복잡도** | 유형별 편균 복잡도 | 기능별 복잡도 |
| **장점** | 짧은 소요시간 | 높은 정확도 |
| **단점** | 상대적 낮은 정확도 | 상대적 긴 소요시간 |

{: .highlight }
> 각 가능점수 단계별 목적에 맞게 적용 "끝"

---

## 측정 요소

### 데이터 기능

| 유형 | 설명 |
|:-----|:-----|
| **ILF (Internal Logical File)** | 시스템 내부에서 유지 관리되는 데이터 |
| **EIF (External Interface File)** | 외부 시스템에서 참조하는 데이터 |

### 트랜잭션 기능

| 유형 | 설명 |
|:-----|:-----|
| **EI (External Input)** | 외부 입력 |
| **EO (External Output)** | 외부 출력 |
| **EQ (External Inquiry)** | 외부 조회 |

### 복잡도 측정 요소

| 요소 | 설명 | 적용 |
|:-----|:-----|:-----|
| **DET (Data Element Type)** | 데이터 요소 유형 | 데이터/트랜잭션 |
| **RET (Record Element Type)** | 레코드 요소 유형 | 데이터 |
| **FTR (File Type Referenced)** | 참조 파일 유형 | 트랜잭션 |
