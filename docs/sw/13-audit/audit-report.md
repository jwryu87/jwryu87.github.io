# 정보시스템 감리결과보고서

감리
{: .label .label-purple }

---

## 핵심 키워드

`1)종합의견` `2)감리영역별 점검결과` `3)별첨: 감리계획서`

---

## 정의/개념

독립된 감리법인이 제3자적 관점에서 수행한 **현장감리 내용에 대해 결과를 정리하고 제출하는 최종결과보고서**

---

## 구성요소

| 구분 | 구성요소 | 설명 |
|:-----|:---------|:-----|
| | 1.전제조건 | 감리보고서를 작성할 시점에 대한 전제조건 정의 |
| | 2.총평 | 점검대상 사업에 대한 감리의견 총괄정리 |
| **1.종합의견** | 3.감리영역별 상세점검결과 요약 | 개선권고사항 및 유형, 중요도, 발주기관 협조필요여부의 표시 |

### 개선권고 유형 및 중요도

| 구분 | 유형 |
|:-----|:-----|
| **개선권고 유형** | 필수 |
| | 협의 |
| | 권고 |
| **개선시점** | 장기 |
| | 단기 |
| **중요도** | (상/중/하) |
| **발주기관** | 협조필요 |

---

## 감리영역별 점검결과

| 구분 | 구성요소 | 설명 |
|:-----|:---------|:-----|
| **2.감리영역별 점검결과** | 1.감리영역 1 | 가.점검항목별 점검결과, 나.상세점검결과<br>사업관리 및 품질보증활동 영역 점검결과 |
| | 2.감리영역 2 | 가.점검항목별 점검결과, 나.상세점검결과<br>응용시스템 영역 점검결과 |
| | 3.감리영역 3 | 가.점검항목별 점검결과, 나.상세점검결과<br>데이터베이스, 시스템아키텍처 및 보안 영역 점검결과 |
| **3.별첨** | 감리계획서 | 감리수행 전 제출된 감리계획서의 첨부 |

{: .note }
> 각 감리영역별 점검결과는 개선권고유형과 개선시점, 발주기관 협조필요여부가 작성되어 제출됨
