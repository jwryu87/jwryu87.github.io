# 테일러링 (Tailoring)
{: .fs-8 }

기타
{: .label .label-purple }

---

## 핵심 키워드

`프로젝트 특징 정의` `표준 프로세스 선정 및 검증` `상위/하위 수준 커스터마이징` `문서화`

---

## 정의/개념

조직의 표준 프로세스를 커스터마이징 하여 **프로젝트 요구에 맞게 적합한 프로세스**를 얻는 과정

---

## 목적/필요성

| 구분 | 목적 / 필요성 |
|:-----|:-------------|
| **관리적 측면** | ① How-To-Do 제시 ② 지속적 개선 |
| **기술적 측면** | ③ 최적화된 기술 및 방법론 도출 ④ 최신 기술 수용 |

---

## 고려사항

```
                    테일러링 고려사항
                          │
          ┌───────────────┴───────────────┐
          ▼                               ▼
    프로젝트 측면                      기술적 측면
          │                               │
    - 프로젝트 규모/기간               - 기술혁신(파일럿검토)
      (대규모/소규모)                  - 데이터전환(데이터전환 환경 구성)
    - 조직원(경험자/비경험자)          - 시스템연계(인터페이스 난이도)
    - 위험수준(높음/낮음)              - 분산시스템(도입/미도입)
```

## 절차

```
┌──────────┐   ┌──────────────┐   ┌─────────────┐   ┌──────────┐   ┌──────────┐
│ 프로젝트 │ → │ 표준 프로세스 │ → │ 상위 수준의 │ → │   세부   │ → │ 테일러링 │
│ 특징 정의 │   │ 선정 및 검증 │   │ 커스터마이징 │   │커스터마이징│   │  문서화  │
└──────────┘   └──────────────┘   └─────────────┘   └──────────┘   └──────────┘
      │              │                  │               │               │
      ▼              ▼                  ▼               ▼               ▼
   입출력         입출력              입출력          입출력          입출력
      │              │                  │               │               │
   - 프로젝트    - 표준프로세스 선정   - 생명주기      - 테일러링 매트릭스  - WBS 테일러링 근거
     프로파일    - 프로세스 검증        WBS           - 프로젝트 WBS      - 적용 결과서
                                                      - 산출물 구성도
                                                      - 프로젝트 스케줄
```

---

## 절차별 상세

| 단계 | 활동 | 산출물 |
|:-----|:-----|:-------|
| **1. 프로젝트 특징 정의** | 프로젝트 유형, 규모, 복잡도 분석 | 프로젝트 프로파일 |
| **2. 표준 프로세스 선정 및 검증** | 조직 표준 프로세스 선택, 적합성 검토 | 선정된 표준 프로세스 |
| **3. 상위 수준 커스터마이징** | 생명주기 모델, WBS 구조 결정 | 생명주기 WBS |
| **4. 세부 커스터마이징** | 활동, 산출물, 일정 상세화 | 테일러링 매트릭스, 프로젝트 WBS |
| **5. 테일러링 문서화** | 적용 근거 및 결과 기록 | WBS 테일러링 근거, 적용 결과서 |

---

## 테일러링 적용 예시

| 항목 | 대규모 프로젝트 | 소규모 프로젝트 |
|:-----|:---------------|:---------------|
| **생명주기** | 폭포수 + 반복적 혼합 | 애자일 |
| **문서화 수준** | 상세 문서화 | 경량 문서화 |
| **검토 횟수** | 다단계 검토 | 단순 검토 |
| **테스트 범위** | 전체 테스트 | 핵심 기능 테스트 |
