# 테스트 오라클
{: .fs-8 }

9.2 테스트 기법
{: .label .label-yellow }

> 암기: **참샘휴일**

---

## 핵심 키워드

`참 오라클` `샘플링 오라클` `휴리스틱 오라클` `일관성검사 오라클`

---

## 정의/개념

테스트를 수행한 결과가 참인지 거짓인지 판단하기 위해서 미리 정의된 참 값을 대입하여 비교하는 기법

---

## 테스트 오라클의 유형

| 유형 | 설명 및 사례 |
|:-----|:------------|
| **참 오라클 (True)** | ✓ 모든 입력 값들에 대해 예측결과들을 생성<br>✓ 오라클을 개발하는데 소요되는 비용이 큼<br>예) 0~360에 대해 모든 sine함수값 측정<br><br>활용) 항공기, 원전방사능탐지 |
| **샘플링 오라클 (Sampling)** | ✓ 특정 입력값 들에 대해서만 예측결과를 제공<br>예) 0, 90, 180, 270, 360등 경계값의 sine함수값 측정<br><br>□ Sine 함수를 테스트할 때 0, 90, 180, 270, 360도에 대해서만 올바른 결과를 제공한다.<br><br>활용) 통계 SW |
| **휴리스틱 오라클 (Heuristic)** | ✓ 특정 입력값 들에 대해서는 샘플링 오라클 적용<br>✓ 나머지 입력값들은 경험/직관으로 오라클 선정<br>예) 0, 90, 180등의 경계값은 샘플링 오라클, 나머지 구간은 sine함수의 특징 적용<br><br>응용) 게임, 업무용 SW |
| **일관성검사 오라클 (Consistent)** | ✓ 변경이 있는 경우 전후의 결과값이 같은지 확인<br>응용) 대다수 상용 test자동화 도구에서 사용 |
