## 정량적 위험분석

### 핵심 키워드

`의사결정나무` `금전적 기대값 분석` `몬테카를로 시뮬레이션` `민감도 분석`

### 정의/개념

식별 위험, 우선순위 높은 위험 중심 정량적 분석 기법

### 기법 상세

| 구분 | 설명 |
|:-----|:-----|
| **의사결정나무 분석<br>(Decision Tree Analysis)** | - 의사결정 규칙을 도표화하여 관심대상이 되는 집단을 몇 개의 소집단으로 분류하거나 예측을 수행하는 정량적 분석 기법<br>- Tree 형태 규칙으로 표현되어 쉽게 이해되고 활용이 가능<br>- 두 개 이상의 변수가 결합하여 목표변수에 어떤 영향을 주는지 확인이 가능<br>- 의사결정나무 분석을 활용한 프로젝트 투자 위험분석 예시 |
| **금전적 기대값 분석<br>(Expected Monetary Value Analysis)** | - 향후 발생할지 여부를 알 수 없는 시나리오가 수반될 때 사건의 가치와 발생확 활동을 통해 기대치를 산출하는 기법<br>- 의사결정 나무 분석을 기반으로 사용<br>- 발생확률과 영향력을 모두 고려하여 분석<br>- 여러 종류의 시나리오가 존재하는 경우 모든 가능결과를 금전적 가치분석에 포함<br>- EMV = Value × Probability<br>- EMV > 0 : 기회(긍정적 위험), EMV < 0 : 위험(부정적 위험) |
| **몬테카를로 시뮬레이션** | - 확률분포를 가정하여 모델을 만들고 난수를 발생시키 일정/원가/역량 준수확률을 계산하는 기법<br>- 난수 발생 → 현재 분석하고자는 발생 이벤트로의 변환 → 한 번의 이벤트에 대한 결과 획득 → 알의 과정을 N번 반복하여 축적 데이터 획득 → 결과(기대치) 획득<br>- A 모듈 설계시 최소 12~14일 소요됨<br>- 소요기간 12~14 일중 난수를 발생시켜 현질 선택 |
| **민감도 분석<br>(Sensitive Simulation)** | - 선택된 방법과 데이터가 결과에 어떤 영향을 미치는지 독립변수의 종속변수의 두변수에 의한 분석하는 기법<br>- 프로젝트 목표에 가장 영향을 많이 주는 위험을 식별하고 규명<br>- 사용이 쉬운 반면 불확실성이 큰 데이터는 분석 대상에서 제외될 가능성 존재<br>- 토네이도 다이어그램, 일중분산분석, 시나리오 분석 등을 이용<br>- 토네이도 다이어그램을 이용해 프로젝트에 미치는 영향 순서대로 재배열한 예시 |

## 정성적 위험분석

### 핵심 키워드

`델파이법` `시나리오법` `순위결정법`

### 정의/개념

프로젝트 위험 크기, 점수, 척도 사용 위험 분석 기법

### 기법 상세

| 구분 | 설명 |
|:-----|:-----|
| **개념** | 프로젝트 위험의 손실 크기를 화폐가치로 표현하기 어려운 정보의 평가를 위해 일반적으로 점수, 척도를 사용하는 위험분석 방법 |
| **델파이법** | 각 분야의 전문 그룹을 구성하여 위험 분석의 의사결정을 이용하는 방식 |
| **시나리오법** | 어떤 사건도 기대대로 발생하지 않는다는 사실에 근거하여 일정 조건하에서 위험에 대한 발생 가능한 결과들을 추정하는 방법 |
| **순위결정법** | 비교 우위 결정표를 통해 각각의 위험을 상호 비교하여 우선순위를 도출하는 방법 |
| **위험 확률-영향 평가** | 확률-영향 매트릭스 기반으로 위험의 등급을 산정하는 위험 평가 방법 |

- 경험 및 분류 기반하여 빠르게 분석 가능하나 수학적 수치 기반의 정량적 위험 분석으로 정확도 상승 필요
