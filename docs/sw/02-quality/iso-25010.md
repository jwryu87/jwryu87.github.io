## 품질 특성 (ISO/IEC 25010:2023)

| 주특성 | 부특성 | 설명 |
|:-------|:-------|:-----|
| **기능 (Functional Suitability)** | 기능완전성 | 기능이 지정된 작업 및 사용자 목적 전체를 다루는 정도 |
| | 적합성 (기능정확성) | 제품 또는 시스템이 필요한 전달도에 따라 정확한 결과를 제공하는 정도 |
| | 기능적절성 | 기능이 명시된 작업 달 목적의 달수를 용이하게 하는 정도 |
| **신뢰성 (Reliability)** | 성숙성 | 제품 또는 시스템이 표준작동 하에 신뢰도에 대한 요구를 충족시키는 정도 |
| | 가용성 | 사용이 필요한 경우 제품 또는 시스템이 사용 및 접근가능한 정도 |
| | 결함허용성 | H/W 또는 S/W의 결점 존재 시 제품 또는 시스템이 의도대로 작동하는지 여부 |
| | 복구성 | 중단/실패 시 제품 또는 시스템이 원하는 상태로 재설정된 데이터로 복구할 수 있는 정도 |
| | 적절 인지성 | 제품 또는 시스템이 사용자의 요구에 적절한지 여부를 식별할 수 있는 정도 |
| | 학습성 | 사용자가 제품 또는 시스템의 사용법을 배워 명시된 목적을 달성할 수 있는 정도 |
| **상호작용 능력 (Interaction Capability)** | 운용성 | 제품 또는 시스템의 작동 및 제어를 쉽게 할 수 있는 정도 |
| | 방지성 | 발생한 오류에 대해 시스템이 사용자를 보호하는 정도 |
| | 사용자 참여도 | 사용자 인터페이스가 기능과 정보를 매력적이고 동기를 부여하는 방식으로 제공하여 지속적인 상호 작용을 유도하는 정도 |
| | 포괄성 | 다양한 배경(다양한 연령, 능력, 문화, 민족, 언어, 성별, 경제적 상황 등)을 가진 사람들이 제품이나 시스템을 사용할 수 있는 정도 |
| **성능 효율성 (Performance Efficiency)** | 사용자 지원 | 다양한 특성과 능력을 가진 사람들이 특정 사용 환경에서 적응될 수 있는 정도 |
| | 시기 생명성 | 사용자가 필요로 하는 경우 자원 또는 기타 리소스(예: 사용자 설명서 및 데스크 또는 다른 사용자)와의 커뮤니케이션이 가능한 정도 사용자가 제품의 기능과 사용법을 즉시 알 수 있도록 적절한 정보를 제공하는 정도 |
| | 시간 반응성 | 기능 수행 시 사용자는 작업을 유형 및 양 및 리소스 사용시간/처리시간 및 도구사양을 준수하는 정도 |
| | 지원 요율성 | 기능 수행 시 사용하는 자원을 유형 및 양 및 요구사항을 충족하는 정도 |
| | 용량 | 최대 한도는 시스템 파라메터의 최대 허용치 요구 사용을 충족하는 정도 |
| **유지보수성 (Maintainability)** | 모듈성 | 최소 영향력 길이 - 개별 구성요소로 이루어진 정도 |
| | 재사용성 | 하나 이상의 시스템에서 사용되거나 다른 자산을 구축할 수 있는 정도 |
| | 분석성 | 시스템 문제에 대 기대 영향을 받는기 및 평가 가능한 보고서 형태로 파악하는 것도 |
| | 수정가능성 | 제품 또는 시스템이 손대면서 효과적이고 효율적으로 수정될 수 있는 정도 |
| | 시험가능성 | 제품 또는 시스템 시험 전 시험에 필요한 완전 기능 제공여부 |
| **유연성 (Flexibility)** | 적응성 | 다른 H/W, S/W 또는 기타 환경에 유효적 · 효율적으로 적용될 수 있는 정도 |
| | 확장성 | 계층적 증가 또는 감소하여 작업 부하를 처리할 수 있거나 기반시설 서비스게 위해 용량을 축정할 수 있는 정도 |
| | 설치성 | 제품 또는 시스템이 성공적으로 설치 및 제거될 수 있는 정도 |
| | 대치성 | 동일 환경에서 동일 목적을 위해 다른 지정 S/W로 대체될 수 있는 정도 |
| **보안성 (Security)** | 기밀성 | 제품 또는 시스템이 반드시 간한하는 데이터만 접근 가능한 정도 |
| | 무결성 | 프로그램 또는 데이터에 무단으로 접근 또는 편집비여 치를 방지하는 정도 |
| | 부인방지 | 시험 조항 및 부인허지 못하도록 명확 것 시건에 대해 금욕되는 정도 |
| | 책임성 | 시스템내 개인을 유일하게 식별, 연결/차단 행동을 해지시 가동여 수정할 수 있는 정도 |
| | 인증성 | 시간 및 행동에 대해 명의지용을 증명할 수 있는 능력 |
| | 대상(저항성) | 제품 또는 시스템이 의미있건 공격하의 금지할 받는 능동 자동능 유지하는 정도 |
| **안전성 (Safety)** | 운영 시범 | 작동이 가용 적절 또는 자원장의 이루에 수 있는 가치형능보증시간 는 노동력이 가능시 수 없을 수 있게 11개 적용하여 금지하고는 종의 일원에 |
| | 감시 인식 | 실행 오는 자동 속력 자원장의(의 오류 타야되기가 간단 가능에 관해하고 수 없는 종에 |
| | 위험 평가 | 사용인 자주 사용량 권장 없는 타요 밖에 억제할 수 있는 종에 |
| | 오류 방지 | 작업이 대는 어떻한 구성 측정 행동에는 등 예를 경우에 안전한 유지 및는 경도 |
