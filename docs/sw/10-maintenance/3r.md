# 3R (Reverse-Engineering, Re-Engineering, Re-Use)
{: .fs-8 }

유지보수
{: .label .label-purple }

---

## 핵심 키워드

`역공학` `재공학` `재사용` `Repository`

---

## 정의/개념

소프트웨어 생산성 극대화, **Repository 기반**, 역공학, 재공학, 재사용 공학 기법

---

## 개념

| 구분 | 설명 |
|:-----|:-----|
| **역공학 (Reverse Engineering)** | 자동화 도구 사용, 물리적 수준 SW를 논리적 SW 수준 정보 추출 |
| **재공학 (Re-Engineering)** | 자동화 도구 사용, 현재 시스템 검토 분석 통해 시스템 재설계 |
| **재사용 (Re-Use)** | 사전 개발 완료된 결과물, 신규 SW 개발 적용 신뢰성 확보 |

---

## 절차

| 구분 | 절차 |
|:-----|:-----|
| **역공학** | 코드 추출 → 코드 분석 → 코드 수정 → 문서화 |
| **재공학** | 역공학 → 재구조화 → 문서화 → 구현 |
| **재사용** | 순공학 → 저장 → 추출 → 재사용 |

---

## 유형

| 구분 | 유형 |
|:-----|:-----|
| **역공학** | 논리 역공학, 자료 역공학 |
| **재공학** | 재구조화, 재모듈화 |
| **재사용** | Library, CBD (Component Based Development) |

---

## 도구/산출물

| 구분 | 도구/산출물 |
|:-----|:------------|
| **역공학** | 원시코드 / Wire Shark |
| **재공학** | 시스템 문서 / CASE |
| **재사용** | 구현 코드 / COM |

---

## 3R 비교표

| 구분 | 역공학 | 재공학 | 재사용 |
|:-----|:-------|:-------|:-------|
| **목적** | 물리적→논리적 정보 추출 | 시스템 재설계 | 신뢰성 확보 |
| **절차** | 코드추출→분석→수정→문서화 | 역공학→재구조화→문서화→구현 | 순공학→저장→추출→재사용 |
| **유형** | 논리역공학, 자료역공학 | 재구조화, 재모듈화 | Library, CBD |
| **도구** | Wire Shark | CASE | COM |
| **산출물** | 원시코드 | 시스템 문서 | 구현 코드 |
