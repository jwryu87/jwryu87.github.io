# 사이버 킬 체인 (Cyber Kill Chain)
{: .fs-8 }

19.7 탐지/대응
{: .label .label-purple }

---

## 핵심 키워드

`정찰` `무기화` `전달` `익스플로잇` `설치` `C&C` `행동 개시`

---

## 정의/개념

사이버 공격을 **프로세스 기반으로 분석**하여, 각 단계에서 가해지는 **위협 요소를 파악하고 공격 활동을 완화**하기 위해 공격 방법을 **7단계로 정의한 사이버 공격 분석 모델**

---

## 개념도

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              공격 7단계                                      │
│                                                                             │

│  │ 정찰  │──▶│무기화 │──▶│ 전달  │──▶│익스플로잇│──▶│ 설치  │──▶│  C&C   │ │
│  │       │   │       │   │       │   │         │   │       │   │        │ │
│  └───────┘   └───────┘   └───────┘   └─────────┘   └───────┘   └────┬───┘ │
│                                                                      │     │

└─────────────────────────────────────────────────────────────────────────────┘
```

- 사이버 킬 체인은 공격을 위한 7단계 중 어떤 단계라도 탐지 및 방어를 하게 되면 실제공격으로 이어지지 않는 점에 착안된 선제적 대응 전략

---

## 7단계 상세

| 단계 | 설명 | 공격방법 | 대응방안 |
|:-----|:-----|:---------|:---------|
| **1. 정찰** | 공격 목표와 표적을 조사/식별/선정 | - 이메일, 크롤링<br>- 사회공학기법 | - Web Analytics 기반 탐지<br>- 불법 접속 차단 (ACL)<br>- 패치, 페이로드 필터링 |
| **2. 무기화** | 자동화 도구이용, 사이버 무기준비 | - 악성프로그램생성<br>- Trojan 등 결합 | - 로그분석<br>- 프로세스모니터링<br>- 오용탐지 (Misuse) |
| **3. 전달** | 표적시스템에 사이버무기를 유포 | - 이메일 첨부<br>- Drive By Download<br>- 이동식미디어/침투<br>- 패스워드니/침투 | - 시그니처, 패턴매칭<br>- 통계자료기반 비정상 행위탐지 (Anomaly)<br>- DMZ 내/외부망 분리<br>- Vigilant User 탐지 |
| **4. 익스플로잇** | 사이버무기의 작동 촉발/악용 | - 사이버무기 작동<br>- OS, App, 취약점이용<br>- Mobile 앱 공격 | - DEP 적약점 방어<br>- OS/IE/APP 패치<br>- 코드사인인증서 날짜확인 및 유효인증서 확인 |
| **5. 설치** | 표적시스템에 악성 프로그램 설치 | - Trojan 설치<br>- 원격 백도어 | - 외부망/내부망 별도 방화벽 구축을 통한 이중차단 |
| **6. C&C** | 표적시스템 원격조작 채널 구축 | - 표적시스템 권한획득<br>- 명령어 전달 제어<br>- Command 대기 | - 고의적 응답지연을 통한 공격 완화<br>- 패킷 시그니처 기반 분석 |
| **7. 행동개시** | 정보수집/시스템파괴 등 공격수행 | - 시스템 파괴<br>- 데이터 유출 | - Honey Pot을 통한 유인<br>- SIEM Log Audit 패턴매칭 |
