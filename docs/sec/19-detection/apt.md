# APT (Advanced Persistent Threat)
{: .fs-8 }

19.2 탐지/대응
{: .label .label-red }

---

## 핵심 키워드

`명확한 타겟` `목표/우회 공격` `지능화` `지속적` `침투/검색/수집/제어`

---

## 정의/개념

- **(사회 통념 측면)** 다양한 IT 기술과 방식을 이용해 조직적으로 다양한 보안위협들을 생산해 **지속적으로 특정 대상에게 가하는** 일련의 보안위협 행위
- **(보안 기술 측면)** 명확한 목표를 가지고 **Zero-day 취약점 또는 악성코드 등의 공격 기술을 우회 침투**하여 기밀데이터의 유출 및 시스템 중단의 공격 기법

---

## 라이프 사이클

| 단계 | 세부 설명 | 공격 기법 |
|:-----|:---------|:---------|
| **침투 (Incursion)** | 훔친 인증 정보, SQL 인젝션, 악성코드 등을 사용하여 오랜 시간 거쳐 공격대상 시스템에 활동 거점을 구축 | - 관찰<br>- 사회공학<br>- 제로데이취약점<br>- 수동 공격 |
| **탐색 (Discovery)** | APT 공격자는 한번 시스템에 침입하면 목표로 하는 기관의 시스템에 대한 정보를 수집하고 기밀데이터를 자동으로 검색 | - 다중 벡터<br>- 은밀한 활동 (Run silent, run deep)<br>- 연구 및 분석 |
| **수집 (Capture)** | 보호되지 않은 시스템의 데이터는 공격자에게 노출 | - 은닉(Convert)<br>- 권한 상승 |
| **제어 (Control)** | APT 공격자는 표적시스템의 제어 권한을 장악하며, 각종 기밀 데이터를 유출하며 SW 및 HW에 손상을 입힘 | - 유출, 중단 |

## 대응방안

### 조직

| 대응 방안 | 설명 |
|:----------|:-----|
| **보안 전담 조직 구성** | CISO 기반 APT 대응 TFT |

### 규정

| 대응 방안 | 설명 |
|:----------|:-----|
| **정책, 표준, 지침의 현행화** | Compliance 보안 프로세스 구축 |

### 보안의식

| 대응 방안 | 설명 |
|:----------|:-----|
| **APT 위협성 교육, 홍보** | 정기적 보안 인식 교육 |
| **감시, 내부감사** | 내부 모니터링 강화 |

### 시스템

| 대응 방안 | 설명 |
|:----------|:-----|
| **계층적 방어** | 다중 보안 레이어 적용 |
| **망 분리** | 업무망/인터넷망 분리 |

### 네트워크

| 대응 방안 | 설명 |
|:----------|:-----|
| **Inbound/Malicious Content 탐지 기술** | 악성 트래픽 탐지 |
| **Outbound Callback Traffic 기술** | C&C 통신 탐지 |
| **Port Scanning** | 비인가 포트 탐지 |
| **암호화** | 통신 암호화 |

### 데이터

| 대응 방안 | 설명 |
|:----------|:-----|
| **DLP** | 데이터 유출 방지 |
| **Log 분석/시 계열 분석** | 이상 징후 탐지 |
| **SIEM** | 통합 보안 관제 |

## APT 공격 단계별 대응

```
┌───────────┐    ┌───────────┐    ┌───────────┐    ┌───────────┐
│   침투    │───▶│   탐색    │───▶│   수집    │───▶│   제어    │
│(Incursion)│    │(Discovery)│    │ (Capture) │    │ (Control) │
└─────┬─────┘    └─────┬─────┘    └─────┬─────┘    └─────┬─────┘
      │                │                │                │
      ▼                ▼                ▼                ▼
┌───────────┐    ┌───────────┐    ┌───────────┐    ┌───────────┐
│ 경계 보안  │    │ 내부 모니터링│    │ 데이터 보호│    │ 사고 대응 │
│ 교육/훈련  │    │ 이상 탐지  │    │ DLP/암호화 │    │ 포렌식   │
└───────────┘    └───────────┘    └───────────┘    └───────────┘
```
