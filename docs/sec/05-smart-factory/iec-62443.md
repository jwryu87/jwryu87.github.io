# IEC 62443
{: .fs-8 }

5.3 스마트공장 보안
{: .label .label-yellow }

---

## 핵심 키워드

`General` `Policy & Procedure` `System` `Component` `IACS` `Security Assurance Level`

---

## 정의/개념

산업제어시스템(IACS) **보안관리 요구사항**과 **보안기술**, 제품의 **개발 요구사항 및 구성요소**에 대한 **기술적 보안 요구사항** 등이 정의되어 있는 **산업제어시스템 보안 국제 표준**

## 구성요소

### Part 1: General (일반)

| 세부 | 설명 |
|:-----|:-----|
| **용어 정의, 컨셉, 모델** | 7개 FR(Foundational requirements) 정의 |
| | - 식별 및 인증(FR1), 사용제어(FR2), 시스템 무결성(FR3), 데이터 기밀성(FR4), 데이터 제한성(FR5), 응답성(FR6), 자원가용성(FR7) |
| **용어, 약어 사전** | 사용하는 용어와 약어의 마스터 용어집 정의 |
| **시스템 보안 적합 Metric** | IACS에 대한 우선 순위가 높은 시스템 사이버 보안 적합성 측정 기준 정의 |
| **라이프사이클 및 유즈케이스** | IACS 보안 라이프사이클을 정의하고 실증 사례 설명 |

### Part 2: Policy & Procedure (정책 및 절차)

| 세부 | 설명 |
|:-----|:-----|
| **IACS 보안 프로그램 수립** | IACS 사이버 보안 관리 시스템 구축 필요 요소 정의 |
| **IACS 보안 관리 가이드 작성** | 설계 및 구현 후 보안관리시스템 운영 방안 기술 |
| **IACS 환경 패치 관리** | IT 기반과 다른 IACS 보안 환경 패치 관리 요구 사항 |
| **IACS 공급 업체 준수 요구사항** | IACS 공급 업체의 설치 및 유지관리 요구사항 |

### Part 3: System (시스템)

| 세부 | 설명 |
|:-----|:-----|
| **IACS 보안 기술** | 적용가능한 사이버보안 도구, 완화 대응책, 기술 제공 |
| **영역, 전송에 대한 보안 수준** | IACS 시스템 및 네트워크 경로 보안 수준 목표 수립 |
| **시스템 보안 요구 사항 및 수준** | 위험 평가 단계에서 필요한 서비스 및 기능을 식별 |

### Part 4: Component (컴포넌트)

| 세부 | 설명 |
|:-----|:-----|
| **제품 개발 요구 사항** | 보안개발생명주기 정의 및 그에 따른 요구사항 기술 |
| **IACS 컴포넌트 요구 사항** | 1-1에 기술된 7개 FR 관련 상세 컴포넌트 요구사항 |

> Part 3-3에 IACS 시스템의 보안 요구 사항 및 보장을 위한 **Security Assurance Level**이 정의됨

---

## Security Level (보안 수준)

| Level | 설명 |
|:------|:-----|
| **SL 1** | 우연한 또는 비의도적 위반 방지 |
| **SL 2** | 단순한 수단을 사용한 의도적 위반 방지 |
| **SL 3** | 정교한 수단을 사용한 의도적 위반 방지 |
| **SL 4** | 국가급 자원을 활용한 의도적 위반 방지 |

---

## 7개 Foundational Requirements (FR)

| FR | 요구사항 | 설명 |
|:---|:---------|:-----|
| **FR1** | 식별 및 인증 | Identification and Authentication |
| **FR2** | 사용 제어 | Use Control |
| **FR3** | 시스템 무결성 | System Integrity |
| **FR4** | 데이터 기밀성 | Data Confidentiality |
| **FR5** | 제한된 데이터 흐름 | Restricted Data Flow |
| **FR6** | 이벤트 적시 대응 | Timely Response to Events |
| **FR7** | 자원 가용성 | Resource Availability |
