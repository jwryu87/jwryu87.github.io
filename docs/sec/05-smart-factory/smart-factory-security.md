## 정의/개념

스마트공장의 **OT(Operational Technology)**, **IT**, **물리보안** 측면에서의 보안 위협과 요구사항

---

## II. 스마트 팩토리 보안 위협

### 가. OT 측면 보안 위협

| 구분 | 위협 | 설명 |
|:-----|:-----|:-----|
| **응용시스템 S/W** | MES 대상 공급망 공격 | ERP 대상 DDoS 공격 |
| **제어자동화** | PLC 대상 버퍼 오버플로우 공격 | HMI 대상 비인가자 불법 접근 |
| **현장자동화** | 센서 대상 펌웨어 취약점 공격 | 액츄에이터 포트 랜섬웨어 감염 |
| | 지능형 로봇 대상 부채널 공격 | |

### 나. IT 측면 보안 위협

| 구분 | 위협 | 설명 |
|:-----|:-----|:-----|
| **기반 기술** | CPPS 대상 Jamming 공격 | 디지털 트윈 대상 데이터 위변조 |
| **네트워크** | IIoT 대상 미라이 봇넷 공격 | 엣지 컴퓨팅 대상 MITM 공격 |

### 다. 물리보안 측면 보안 위협

| 구분 | 위협 | 설명 |
|:-----|:-----|:-----|
| **외부 환경 측면** | 불법 드론 공격 | 재해 |
| **내부 환경 측면** | 정전 | 고장/오작동 |

---

## III. 스마트 팩토리 보안 요구사항

### 가. 기술적 측면 보안 요구사항

#### 1) 접근 통제

| 항목 | 대응방안 |
|:-----|:---------|
| **사용자 식별/인증** | MAC Address, IMEI 관리 |
| **패스워드 관리** | SHA-256 |
| **인증실패** | Brute Force Attack 제한 |
| **접근제어** | MAC, DAC, RBAC |

#### 2) 데이터 보호

| 항목 | 대응방안 |
|:-----|:---------|
| **전송데이터 보호** | TLS, IPSec |
| **저장데이터 보호** | 키 스트레칭, 해시솔트 |
| **안전 암호 연산** | ARIA, SEED, AES, ECC |
| **암호키 관리** | 양자 분배키 설정 |

#### 3) 안전한 상태

| 항목 | 대응방안 |
|:-----|:---------|
| **이동식 미디어 관리** | DLP 활용, 원천 차단 |
| **SW 통제** | 펌웨어 무결성 유지, Whitelist 적용 |

### 나. 관리적 측면 보안 요구사항

#### 4) 정보보안 운영 정책

| 항목 | 대응방안 |
|:-----|:---------|
| **보안정책** | 전사 가이드 배포 |
| **보안조직** | CISO 지정 |
| **보안교육** | 주기적 훈련 |

#### 5) 자산 관리

| 항목 | 대응방안 |
|:-----|:---------|
| **중요자산관리** | 응용SW, 제어자동화, 현장자동화 기기 관리 |
| **중요정보관리** | 보안등급 부여, 로그관리 |

#### 6) 보안사고 예방/대응

| 항목 | 대응방안 |
|:-----|:---------|
| **보안사고 훈련** | 모의 훈련 |
| **시스템이상징후탐지** | SIEM, SOAR |
| **취약점 분석 및 평가** | CVE 활용 대응 |
| **감사로그관리** | 로그 활용 |
