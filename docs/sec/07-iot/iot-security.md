## 정의/개념

디스크의 가용성 및 성능 향상을 위해 **미러링 및 스트라이핑 기술** 이용하는 디스크 고가용성 기술과 IoT 보안

---

## I. IoT 보안 위협의 중요성

### IoT 기술의 특성

- IoT 기술이 5G, AI, 빅데이터, 클라우드 등 여러 기술과 융합되어 경제·사회 전반에 내재됨에 따라, 초연결 사회에서의 IoT 보안위협은 더욱 광범위하고 파괴적인 영향이 생성됨

### IoT 아키텍처

## II. IoT 보안 위협

### 가. 네트워크 측면 보안 위협

| 구분 | 위협 | 설명 |
|:-----|:-----|:-----|
| **송/수신** | 데이터 위변조 | 전송 데이터 조작 |
| **트래픽** | DDoS | 분산 서비스 거부 공격 |
| **엣지컴퓨팅** | 인증 키 탈취 | 엣지 디바이스 인증 키 탈취 |

### 나. 디바이스 측면 보안 위협

| 구분 | 위협 | 설명 |
|:-----|:-----|:-----|
| **물리적 공격** | 디캐핑, 마이크로 프로빙 | 하드웨어 분석 공격 |
| **SW 공격** | 버퍼 오버플로우, 트로이 목마 | 소프트웨어 취약점 공격 |
| **부채널 공격** | 비침입, 준침입 | 전력, 전자파 분석 공격 |

---

## IoT 보안 대응 방안

| 측면 | 대응방안 |
|:-----|:---------|
| **네트워크** | TLS/DTLS, 암호화 통신 |
| **디바이스** | 시큐어 부팅, 펌웨어 무결성 검증 |
| **플랫폼** | 접근 통제, 인증/인가 |
| **데이터** | 암호화, DLP |
