# XDR (Extended Detection and Response)
{: .fs-8 }

12.4 보안솔루션
{: .label .label-purple }

---

## 핵심 키워드

`EDR/EPP` `SIEM` `SOAR` `엔드포인트` `위협탐지` `위협대응` `통합`

---

## 정의/개념

**EDR/EPP, SIEM, SOAR 등의 솔루션을 단일 플랫폼에 데이터를 통합하여 분석, 고수준 위협탐지, 보안 사고를 대응하는 보안솔루션**

- 기업이 기존의 SIEM과 SOAR의 한계를 극복하고, 보다 포괄적이고 정밀한 위협 탐지 및 대응을 통해 보안 사고를 예방하고자 하는 경우, XDR이 적합

## 정보수집 대상과 대응을 위한 보안 솔루션

### Front End (사용자)

| 구분 | 설명 |
|:-----|:-----|
| **로그인 및 사용자 관리** | XDR 시스템에 로그인하고, 사용자를 관리 |
| **위협 모니터링** | 위협을 모니터링 및 위협 발생 시 알림 |
| **대응 조치** | 위협에 대한 대응 조치 |

### Back End (탐지/대응)

| 구분 | 설명 |
|:-----|:-----|
| **데이터 수집** | NW, APP, EPP 등에서 생성되는 보안 데이터를 수집 |
| **데이터 통합** | 수집된 보안 데이터를 통합 분석 |
| **위협 탐지** | 통합 분석된 데이터를 기반으로 위협을 탐지 |
| **위협 대응** | 탐지된 위협에 대해 대응 조치 |

- 탐지와 대응에 중점을 두고 정보수집 방법과 분석을 위한 클라우드, 인공지능(ML), 통합과 자동화 기술 등으로 구성
- Front End 솔루션을 위한 데이터 통합, 연동 API 제공, 통합 정책관리 등의 기능 제공

## SIEM, SOAR, XDR 비교

| 비교 항목 | SIEM | SOAR | XDR |
|:---------|:-----|:-----|:----|
| **보안 데이터 수집** | 네트워크, 엔드포인트, 애플리케이션 등 | 네트워크, 엔드포인트, 애플리케이션 등 | 네트워크, 엔드포인트, 애플리케이션 등 |
| **보안 데이터 분석** | 위협 탐지 | 위협 탐지 및 대응 | 위협 탐지 및 대응 |
| **위협 탐지** | 위협 이벤트 기반 | 위협 이벤트 기반 | 위협 이벤트 기반, 시계열 분석, AI/ML |
| **위협 대응** | 보안 전문가 주도 | 자동화 | 통합 자동화 |
| **장점** | 다양한 보안 데이터 수집, 분석 가능 | 보안 운영 효율성 향상, 보안 전문가 업무 부담 감소 | 보다 포괄적이고 정밀한 위협 탐지 및 대응 가능 |
| **단점** | 위협 탐지에만 집중 | 보안 데이터 수집 및 분석에 의존 | 도입 비용 고가 |

## XDR 도입 효과

| 효과 | 설명 |
|:-----|:-----|
| **통합 가시성** | 엔드포인트, 네트워크, 클라우드 전체 가시성 확보 |
| **빠른 탐지/대응** | AI/ML 기반 자동화로 MTTD/MTTR 단축 |
| **운영 효율화** | 개별 솔루션 통합으로 복잡성 감소 |
| **비용 절감** | 중복 솔루션 제거 및 인력 효율화 |
