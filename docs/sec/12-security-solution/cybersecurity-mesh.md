## 핵심 키워드

`이메일 보안` `EFW` `EPP` `DLP` `IDPS` `SWG` `CASB` `CWPP` `CSPM`

## 정의/개념

보안 경계, 위치 상관없이, **디지털 자산 보안 접근 통합 아키텍처** 기술

## 개념도

### 사이버보안 메시(Cybersecurity Mesh) Architecture

```
┌─────────────────────────────────────────────────────────────────────┐
│                   Cybersecurity Mesh Architecture                    │
│                                                                      │
│                     ┌──────────────────┐                            │
│                     │   <End Point>    │                            │
│                     │   MTD, EPP, EDR  │                            │
│                     └──────────────────┘                            │
│                              ↓                                       │
│  ┌──────────┐      ┌──────────────────┐      ┌──────────────────┐   │
│  │ <클라우드> │      │  사이버보안 메시   │      │    <네트워크>     │   │
│  │CASB, CWPP│──────│                  │──────│ IDPS, SWG, EFW,  │   │
│  │  CDPM    │      │      대시보드      │      │      ZTNA       │   │
│  └──────────┘      │   중앙 정책 향상    │      └──────────────────┘   │
│                    │     인증기관       │                            │
│  ┌──────────┐      │   지능적 보안 분석  │      ┌──────────────────┐   │
│  │  <응용>   │      │                  │      │     <데이터>      │   │
│  │ SEG, WAF │──────│                  │──────│  DLP,            │   │
│  │          │      └──────────────────┘      │ Data Classification│  │
│  └──────────┘                                │      EDRM         │   │
│                     ┌──────────────────┐      └──────────────────┘   │
│                     │     <인증>       │                            │
│                     │ CIEM, MFA, PAM,  │                            │
│                     │       IGA        │                            │
│                     └──────────────────┘                            │
└─────────────────────────────────────────────────────────────────────┘
```

- 사이버보안 메시는 사람 또는 사물을 중심으로 보안 경계를 정의

## 보안방안

### 사이버보안 메시(Cybersecurity Mesh)의 보안 방안 설명

| 영역 | 보안 방안 | 상세 기술 설명 |
|:-----|:---------|:-------------|
| **지역 보안 영역** | **Email Security** | - 개인 메일 보안 기술<br>- 메일 Sniping 방지 |
| | **EFW** | - End Point 방화벽<br>- Cloud Application Defender |
| | **EPP** | - 다양한 보안 기능의 유기적인 연동 및 통합 관리<br>- 개인정보 유출방지, OS/보안패치관리 |
| **네트워크 보안 영역** | **DLP** | - 데이터 이동경로를 감시<br>- 기업 내부의 중요 정보와 데이터 유출 방지 |
| | **IDPS** | - 네트워크 기반 침입 탐지와 대응을 복합적으로 수행<br>- 지능 주행 보안 기술로 활용 |
| | **SWG** | - 외부의 위협으로부터 내부 시스템과 사용자를 보호<br>- WAF 체계를 구축하여 내부의 안전 보장 |
| **분산 환경 보안 영역** | **CASB** | - 클라우드 환경의 데이터 보안 기술<br>- 사용자와 애플리케이션 간의 데이터 모니터링 및 로깅 |
| | **CWPP** | - 하이브리드 환경 지원<br>- 클라우드 서버 워크로드 보안 플랫폼 |
| | **CSPM** | - 클라우드 내부 공격방지를 위한 관리 프로세스<br>- 실시간 조직 내 클라우드 사용 현황 파악 |

## 영역별 주요 보안 솔루션

### 지역 보안 영역 (Local Security)

| 솔루션 | 풀네임 | 역할 |
|:-------|:-------|:-----|
| **EPP** | Endpoint Protection Platform | 엔드포인트 통합 보안 |
| **EDR** | Endpoint Detection & Response | 엔드포인트 위협 탐지/대응 |
| **MTD** | Mobile Threat Defense | 모바일 위협 방어 |

### 네트워크 보안 영역 (Network Security)

| 솔루션 | 풀네임 | 역할 |
|:-------|:-------|:-----|
| **IDPS** | Intrusion Detection & Prevention System | 침입 탐지/방지 |
| **SWG** | Secure Web Gateway | 웹 보안 게이트웨이 |
| **ZTNA** | Zero Trust Network Access | 제로 트러스트 네트워크 접근 |

### 분산 환경 보안 영역 (Distributed Security)

| 솔루션 | 풀네임 | 역할 |
|:-------|:-------|:-----|
| **CASB** | Cloud Access Security Broker | 클라우드 접근 보안 중개 |
| **CWPP** | Cloud Workload Protection Platform | 클라우드 워크로드 보호 |
| **CSPM** | Cloud Security Posture Management | 클라우드 보안 태세 관리 |

## 사이버보안 메시 vs 기존 경계 보안

| 구분 | 기존 경계 보안 | 사이버보안 메시 |
|:-----|:-------------|:---------------|
| **보안 경계** | 네트워크 경계 | 사람/자산 중심 경계 |
| **접근 방식** | Castle & Moat | Anywhere, Anytime 보안 |
| **확장성** | 제한적 | 분산 환경 지원 |
| **통합성** | 개별 솔루션 | 통합 아키텍처 |
| **가시성** | 제한적 | 전체 가시성 확보 |

## Gartner의 사이버보안 메시 4대 계층

| 계층 | 설명 |
|:-----|:-----|
| **보안 분석/인텔리전스** | 위협 정보 수집 및 분석 |
