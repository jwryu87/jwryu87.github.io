# 디지털 포렌식 (Digital Forensic)
{: .fs-8 }

14.1 포렌식
{: .label .label-purple }

> 암기: **정신무연재** (정당성, 신속성, 무결성, 연계보관성, 재현성)

---

## 핵심 키워드

`법적 증거자료 확보` `네트워크 증거수집` `시스템 증거수집` `응용프로그램 증거수집`

---

## 정의/개념

컴퓨터를 이용하거나 활용해 이루어지는 **범죄 행위 법적 증거자료 확보** 위해, 컴퓨터 시스템, 네트워크 등 **가상 공간의 자료가 법적 증거물로 법원에 제출**될 수 있도록 확보하는 일련의 절차와 방법

---

## 절차/원칙

### 디지털 포렌식의 개요

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│   사전 준비 ───▶ 증거 수집 ───▶ 포장 및 이송 ───▶ 조사 분석 ───▶ 보고서 작성│
│                                                                         │
│   - 장지          - 증거물 획득   - 감사용 활용    - 이미지 복사   - 분석결과│
│   - 대상선별     - 대상선정(위    - 대상서버,      - 포렌터 조사   - 의미 기재│
│   - 디지털       모니의 확보      클라우드 포팅    및 포팅         - 제출     │
│   - 포렌식       식별 확보        - 디지털 증거    - 클라우드                │
│   - 인력관리 연성 - 현장 자료 및   확보 포팅       및 검색                   │
│                   증거를 포팅     - 증거를 포팅                              │
└─────────────────────────────────────────────────────────────────────────┘
```

### 디지털 포렌식 원칙

| 원칙 | 설명 |
|:-----|:-----|
| **정당성(正當性)의 원칙** | - 입수 증거는 적법절차를 거쳐 획득 - 위법수집증거 배제 법칙 |
| **재현(再現)** | - 위법 절차를 통해 수집된 증거는 증거 능력을 부정 - 독수의 과실 이론 |
| | - 위법하게 얻어진 증거는 2차 증거도 부정 |

## 수집방법

### I. 디지털 포렌식 개념 및 5원칙

### II. 네트워크 증거수집

#### 가. Passive
- **물리적 감청**: 케이블, 허브, 포트 미러링
- **트래픽 수집**: libpcap, TCP Dump, WireShark

#### 나. Active
- **일반 인터페이스**: Console, SNMP
- **간접조사**: 포트 스캐닝, 취약점 스캐닝

### III. 시스템 증거 수집

#### 가. 시스템 운용 측면
- **비운용**: 디스크 복제, 이동식 저장 매체
- **운용**: 탐침 HW, 내장 SW

#### 나. 시스템 정보 유형 측면
- **기본**: OS, 디스크 정보
- **시간**: MAC, 로컬
- **사용자**: 계정, 원격 IP

### IV. 응용 프로그램 증거 수집

#### 가. 범용 프로그램 측면
- **프로세스**: Windows 작업관리자, eProcess
- **프로그램**: Log File

#### 나. 임베디드 프로그램 측면
- **파일단위**: USB 통신 프로토콜
- **제조사 전용 통신 프로그램**: PC Sync SW

## 주요 기법

| 기법 | 설명 |
|:-----|:-----|
| **디스크 이미징** | 원본 보존을 위한 비트 단위 복제 |
| **해시 검증** | MD5, SHA256 등으로 무결성 검증 |
| **타임라인 분석** | MAC Time 기반 행위 시간 분석 |
| **메모리 분석** | RAM 내 휘발성 데이터 수집 |
| **로그 분석** | 시스템/애플리케이션 로그 분석 |

---

## 포렌식 5원칙 (정신무연재)

| 원칙 | 영문 | 설명 |
|:-----|:-----|:-----|
| **정당성** | Legitimacy | 적법한 절차로 수집 |
| **신속성** | Promptness | 신속한 증거 수집 |
| **무결성** | Integrity | 위/변조 방지 |
| **연계보관성** | Chain of Custody | 보관 이력 관리 |
| **재현성** | Reproducibility | 동일 결과 재현 가능 |

---

## 안티포렌식

| 기법 | 설명 |
|:-----|:-----|
| **데이터 파괴** | 완전 삭제, 와이핑 |
| **데이터 은닉** | 스테가노그래피, 암호화 |
| **데이터 위조** | 타임스탬프 조작 |
| **흔적 제거** | 로그 삭제, 히스토리 삭제 |
