## 핵심 키워드

---

## 정의/개념

- **가명 처리**: 개인정보 일부를 삭제하거나 일부 또는 전부를 대체하는 등의 방법으로 추가 정보 없이 **특정 개인을 알아볼 수 없도록** 하는 기술
- **익명 처리**: 개인정보를 영구적 삭제하거나, 개인식별정보 전부 또는 일부를 **해당 기관 고유식별기호로 대체**하는 기술

---

## 주요 기법

### 삭제기술

| 항목 | 설명 |
|:-----|:-----|
| **삭제** | 원본정보에서 개인정보를 단순 삭제 |
| **부분삭제, 로컬삭제** | 일부 속성 삭제 등 다양한 기법 존재 |
| **마스킹** | 특정 항목의 일부 또는 전부를 공백 또는 문자로 대체 |

### 통계도구

| 항목 | 설명 |
|:-----|:-----|
| **총계처리** | 평균값, 최댓값, 최솟값, 최빈값, 중간값 등으로 치환 |
| **부분 총계** | 다른 정보에 비해 오차가 큰 범위가 큰 항목을 평균값 등으로 대체 |

### 일반화

| 항목 | 설명 |
|:-----|:-----|
| **일반 라운딩** | 올림, 내림, 반올림 등의 기준을 적용하여 수치 처리하는 방법 |
| **랜덤 라운딩 (Random rounding)** | 수치 데이터를 임의의 수인 자리 수, 실제 수 기준으로 올림(round up) 또는 내림(round down)하는 기법 |
| **제어 라운딩 (Controlled rounding)** | 라운딩 적용 시 값의 변경에 따라 행이나 열의 합이 원본의 행이나 열 합과 일치하지 않는 단점을 해결하기 위해 원본과 결과가 동일하도록 특 라운딩을 적용하는 기법 |
| **상하단코딩 (Top and bottom coding)** | 정규분포의 특성을 가진 데이터에서 양 극 끝에 몰려진 정보는 적은 수의 분포를 가지게 되어 식별성을 가질 수 있음<br>이를 해결하기 위해 적은 분포를 가진 양 끝단의 정보를 범주화 등의 기법을 적용하여 식별성을 낮추는 기법 |
| **로컬 일반화** | 전체 정보집합을 특정 열 항목(들)에서 특이한 값을 가지거나 보호성의 특이성으로 인해 식별성이 높아지는 경우 해당 부분만 일반화 등의 식별성을 낮추는 기법 |

### 범위화

| 항목 | 설명 |
|:-----|:-----|
| **범위 방법 (Data range)** | 수치 데이터를 임의의 수 기준의 범위(range)로 설정하는 기법으로, 해당 값 범위 또는 구간(interval)으로 표현 |
| **문자데이터 범주화** | 문자로 저장된 정보에 대해 보다 상위의 개념으로 범주화하는 기법 |

### 암호화

| 항목 | 설명 |
|:-----|:-----|
| **양방향 암호화** | 특정 정보에 대해 암호화와 암호화된 정보에 대한 복호화가 가능한 암호화 기법 (대칭키, 비대칭키 암호화 구분) |
| **암호학적 해시함수** | 원문에 대한 암호화의 적용만 가능하고 암호문에 대한 복호화 특성이 없는 암호화 방식 |
| **불가능한 암호화 기법** | MDC, MAC로 구분 |
| **순서보존 암호화** | 원문정보의 순서와 암호값의 순서가 동일하게 유지되는 암호화 방식 |
| **형태보존 암호화 방식** | 원본 정보의 형태와 암호화된 원의 형태가 동일하게 유지되는 암호화 방식 |
| **동형암호화** | 암호화된 상태 연산값을 복호화 하면 원래의 값을 연산한 것과 동일 |

## 무작위 기술

| 기술 | 설명 |
|:-----|:-----|
| **다형성 암호화 기법** | 한 결과를 얻을 수 있는 4세대 암호화 기법<br>- 가명정보의 부정한 결합을 차단하기 위해 각 도메인별로 서로 다른 암처리 방법을 사용하여 정보를 제공하는 방법<br>- 정보 제공 시 서로 다른 암식의 암호화된 가명정보를 적용하여 마도 암호별로 다른 가명정보를 갖지게 |
| **잡음 추가** | 개인정보 일부의 숫자 또는 문자을 추가하는 방법<br>- 분석 시 가치가 적고 식별성이 높은 열 앞쪽에 대상 열의 으 E 값을 범위 내에서 무작위로 순서를 변환하여 식별성을 낮추는 기법 |
| **치환** | 가명을 식별할 수 있는 정보를 토큰으로 변경 후 대체값으로써 개인정보를 적절히 사용하여 발생하는 식별 위험을 제거하여 개인정보를 보호하는 기술 |
| **부분집합 (Sampling)** | 데이터 주체로 전체 모집단이 아닌 표본에 대해 무작위 데이터를 통해 의미를 통해 모집단의 일부를 분석하여 전체에 대한 분석을 하는 기법 |
| **해부화 (Anatomization)** | 기준 하나의 데이터셋(테이블)을 식별성이 없는 정보집합과 없는 정보집합으로 구성한 2개의 테이터셋으로 분리하는 기법 |
| **재현데이터 (Synthetic data)** | 평균과 최대한 유사한 통계적 성질을 갖는 가상의 데이터를 생성하위해 개인정보의 통계적 성질을 분석하여 새로운 데이터를 생성하는 기법 |
| **동형비밀분산 (Homomorphic secret sharing)** | 식별정보를 기반 또는 키와 함께 암호화하여 비밀값을 메시지를 보유 알고리즘에 의한 생성된 두 개 이상의 쉐어(share)로 대체 |
| **차분 프라이버시 (Differential privacy)** | 특정 개인에 대한 사전정보이 있는 상태에서 데이터베이스 질의(Query)에 대한 튜의 간으로 개인을 알 수 있도록 왜곡 확인 값의 외의 수 잡음 (Noise)를 추가하여 특정 개인의 존재 여부를 알 수 있도록 하는 기법<br>- 1개 행위의 차이나는 두 데이터베이스간의 차이(확률분포) 기준으로 하는 프라이버시 보호 모델 |
