### 🔑 핵심 키워드 3개
- **채널용량(C)**: 통신 채널로 보낼 수 있는 **최대 데이터 속도**
- **대역폭(B)**: 도로의 폭처럼 **얼마나 넓은 주파수 범위**를 쓸 수 있는지
- **SNR(S/N)**: 신호가 잡음보다 **얼마나 강한지** 비율

### 📖 등장배경

> 전화선이나 인터넷 케이블로 데이터를 보낼 때, **"최대 얼마나 빠르게 보낼 수 있을까?"**라는 궁금증이 있었어요. 1948년 **클로드 샤논(Claude Shannon)**이라는 수학자가 이 질문에 답했어요. 잡음이 있는 채널에서도 **오류 없이 전송 가능한 최대 속도**를 수학 공식으로 증명한 거예요!

### 📝 개념 정의

> 채널용량은 **"잡음이 있는 통신 채널에서 오류 없이 보낼 수 있는 최대 데이터 속도"**예요.
>
> 비유하자면, **고속도로의 최대 차량 통과량**과 같아요!
> - 🛣️ **대역폭(B)**: 도로의 차선 수 (넓을수록 많이 보낼 수 있음)
> - 📶 **신호 강도(S)**: 자동차의 속도 (빠를수록 좋음)
> - 🌧️ **잡음(N)**: 도로 위의 장애물 (적을수록 좋음)

### 🏗️ 기술요소 (2그룹)

**그룹 1: 샤논 공식의 변수**

| 변수 | 의미 | 쉬운 설명 |
|:-----|:-----|:---------|
| C | 채널용량 (bps) | 최대로 보낼 수 있는 데이터 속도 |
| B | 대역폭 (Hz) | 사용할 수 있는 주파수 범위 |
| S | 신호 전력 | 보내는 신호의 세기 |
| N | 잡음 전력 | 방해하는 잡음의 세기 |
| S/N | 신호 대 잡음비 (SNR) | 신호가 잡음보다 얼마나 강한지 |

**그룹 2: 채널용량을 늘리는 방법**

| 방법 | 쉬운 설명 |
|:-----|:---------|
| 대역폭 증가 | 더 넓은 주파수 범위 사용 (차선 늘리기) |
| 신호 강도 증가 | 더 강한 신호 전송 (더 빠른 차 사용) |
| 잡음 감소 | 간섭 제거 (장애물 치우기) |

### ⭐ 차별점 키워드

> **C = B log₂(1 + S/N)**: 채널용량은 **대역폭에 비례**하고, **SNR의 로그값에 비례**한다!

### 📋 6하원칙 요약

| 구분 | 내용 |
|:-----|:-----|
| **누가** | 클로드 샤논(Claude Shannon), 1948년 |
| **무엇을** | 채널용량의 이론적 한계 |
| **언제** | 데이터 통신 시스템 설계 시 |
| **어디서** | 유선/무선 통신 채널 |
| **왜** | 잡음 환경에서 오류 없는 최대 전송률 계산 |
| **어떻게** | 대역폭(B)과 SNR(S/N)을 이용한 수학 공식 |

## 🎯 기술사 수준 설명
{: #-기술사-수준-설명}

{: .highlight }
> **채널용량(샤논 제3정리)**: C = B log₂(1 + S/N)
> - (공식) `C = B log₂(1 + S/N)`
> - (변수) `CBSN` (채널용량, 대역폭, 신호전력, 잡음전력)
> - (의미) `무잡→무한`, `유잡→한계`
> - (정리) `1정리`, `2정리`, `3정리`

---

### 📝 개념 정의

**정보이론**에서 채널모델 및 전송 제약조건이 주어진 상태에서 **신뢰성 있게 전달할 수 있는 최대 전송률**을 의미하는 이론

| 구분 | 설명 |
|:-----|:-----|
| **정의** | 채널모델 및 전송제약조건이 주어진 상태에서 신뢰성 있는 통신이 가능한 **최고 전송률** (22자) |
| **제안자** | 클로드 샤논(Claude Shannon), 1948년 |
| **목적** | 잡음(Noise)이 있는 환경에서도 오류 없이 데이터를 전송할 수 있는 최대 속도를 계산 |

---

### 🔄 샤논 제3정리의 의미

| 조건 | 의미 |
|:-----|:-----|
| **잡음이 없는 상황** (N → 0, S/N → ∞) | 임의의 대역폭에서도 채널 용량을 거의 **무한으로 할 수 있음** |
| **잡음이 큰 상황** (N → ∞, S/N → 0) | 대역폭을 아무리 증가시켜도 채널 용량을 **무한정 증가시킬 수 없음** |

> 샤논의 제 3 정리를 통해 채널용량을 **이론적으로 계산**할 수 있는 공식 제시

---

### 📐 채널용량 수식

```
┌─────────────────────────────────────────────────────────────────┐
│                     샤논 채널용량 공식                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│                              S                                   │
│         C  =  B  log₂ ( 1 + ─── )                               │
│                              N                                   │
│                                                                  │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  C = 채널용량 (Channel Capacity), bps                    │   │
│   │  B = 대역폭 (Bandwidth), Hz                              │   │
│   │  S = 신호 전력 (Signal Power)                            │   │
│   │  N = 잡음 전력 (Noise Power)                             │   │
│   │  S/N = 신호 대 잡음비 (SNR, Signal-to-Noise Ratio)       │   │
│   └─────────────────────────────────────────────────────────┘   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

> 채널 용량은 **대역폭에 비례**하고, **신호 대 잡음비(SNR)의 로그값에 비례**

---

### 🏗️ 채널용량 수식 상세 설명 `CBSN`

| 수식 표기 | 항목 | 설명 |
|:---------|:-----|:-----|
| **C** | 채널용량 (Channel Capacity) | 채널을 통해 **오류 없이 전송 가능한 최대 데이터 속도**. 단위는 비트/초(bps) |
| **B** | 대역폭 (Bandwidth) | 채널의 **주파수 범위**로, 처리 가능한 데이터 양에 비례. 대역폭이 클수록 전송 속도 증가 |
| **S** | 신호 전력 (Signal Power) | 데이터 전송 **신호의 강도**. 신호가 강할수록 잡음의 영향을 덜 받으며, 데이터 전송에 유리 |
| **N** | 잡음 전력 (Noise Power) | 채널에서 발생하는 **잡음의 강도**. 잡음이 높아지면 데이터 전송 속도가 제한됨 |
| **S/N** | 신호 대 잡음비 (SNR) | 신호 강도와 잡음 강도의 비율. **SNR이 높을수록 채널 용량 증가** |
| **log₂** | 로그 함수 (Base 2) | 정보 용량 계산 시, **이진 비트(binary bits)** 단위로 나타냄 |

#### 공식의 의미

| 구분 | 역할 | 설명 |
|:-----|:-----|:-----|
| **대역폭의 역할** | B ↑ → C ↑ | 대역폭이 넓을수록 데이터 전송 속도 증가. 채널의 주파수 범위가 넓으면 **더 많은 데이터를 동시에 전송** 가능 |
| **SNR의 역할** | S/N ↑ → C ↑ | SNR이 높을수록 데이터 전송 속도 증가. 신호가 잡음보다 강할수록 **안정적인 데이터 전송**이 가능 |

> 채널 용량을 증가시키기 위해서는 **대역폭을 최대한 확보**하고, **잡음의 영향을 줄일 수 있는 H/W, S/W 기술** 적용 필요

---

### 📊 샤논의 제 1, 2, 3 정리

| 구분 | 수식 | 설명 |
|:-----|:-----|:-----|
| **샤논의 제 1 정리** | L̄ ≥ H(X) | 채널에서 **최대로 전송할 수 있는 전송률** (소스 코딩 정리) |
| **샤논의 제 2 정리** | R < C (가능) / R > C (불가능) | R < C: 적절한 부호화 기술 사용시 **오류 최소화 가능** / R > C: 어떤 부호화 기술이라도 **에러가 존재** (채널 코딩 정리) |
| **샤논의 제 3 정리** | C = B log₂(1 + S/N) | **채널용량 공식** (정보 용량 정리) |

> 샤논의 1, 2, 3 정리를 통해 정보 전송에 대한 **이론적 기반** 마련

| 개념 | 설명 |
|:-----|:-----|
| **Cross-Entropy** | 두 확률 분포 p와 q를 구분하기 위해 필요한 **평균 비트 수**를 의미 |
| **KL-divergence** | Kullback-Leibler Divergence, **두 분포의 차이**를 측정하는데 사용 |

> 샤논의 정리를 통해 발표된 정보이론은 기계학습 분야에서 해당 확률분포의 특성을 알아내거나 확률분포 간 유사성을 정량화에 사용

### ⭐ 차별점 키워드

> **C = B log₂(1 + S/N)**: 채널용량은 **대역폭(B)**과 **신호 대 잡음비(S/N)**에 의해 결정되며, 이론적으로 **오류 없이 전송 가능한 최대 속도의 한계**를 제시하는 핵심 공식!
