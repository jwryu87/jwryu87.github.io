## ISO/IEC 27018

### 퍼블릭 클라우드 개인정보보호 표준

> 키워드: **개+동합수/최이정/공개책/정프**

| 번호 | 도메인 |
|:-----|:-------|
| 1 | **개**요 |
| 2 | **동**의와 선택 |

| 4 | **수**집 제한의 원칙 |
| 5 | 데이터 **최**소화 |
| 6 | **이**용 및 공개 제한 |
| 7 | **정**확성과 품질 |
| 8 | **공**개성, 투명성 |
| 9 | **개**별 참여와 접근 |
| 10 | **책**임성 |
| 11 | **정**보보호 |
| 12 | **프**라이버시 보호 |

## 비교

### ISO/IEC 27017 vs ISO/IEC 27018

| 항목 | ISO/IEC 27017 | ISO/IEC 27018 |
|:-----|:--------------|:--------------|
| **목적** | 클라우드 서비스 정보보호 | 개인식별정보(PII) 보호 |
| **기반** | ISO/IEC 27002 | OECD 개인정보보호 원칙 |
| **대상** | 클라우드 서비스 전반 | 퍼블릭 클라우드 |
| **도메인** | 14개 도메인, 37개 항목 | 11개 도메인, 28개 항목 |
| **핵심** | 보안 통제 | 프라이버시 통제 |
