# 양자 암호 키 분배 (QKD: Quantum Key Distribution)
{: .fs-8 }

11.4 양자 보안
{: .label .label-purple }

---

## 핵심 키워드

- QKD 핵심기술: BB84, E91, QTTH, One way
- QKD 하드웨어: 변조기(PM), 검출기(APD), 전자 제어부, 양자중계기

---

## 정의/개념

**양자채널, 공용채널, QRNG**, 비밀키 실시간 분배 **양자암호키 분배 기술**

- 보안이 필요한 송수신자 사이에 양자 암호 키 분배(QKD: Quantum Key Distribution) 기술을 사용하여 암호화에 필요한 비밀키를 통신상에서 실시간으로 분배하기 위한 양자암호키 분배 기술

---

## 개념도

### 양자암호통신의 암호키 분배방식

```
┌──────────────────────────────────────────────────────────────────────┐
│                                                                      │
│   Hello    키분배장비(QKD)      양자채널       키분배장비(QKD)    Hello │
│   ALICE ──────QRNG────────(Quantum Channel)──────QRNG──────── BOB    │
│              │                                    │                  │
│           양자암호키      양자암호채널         양자암호키              │
│              │                                    │                  │
│           암복호장비 ────────────────────────── 암복호장비            │
│              │                                    │                  │
│            송신측                               수신측                │
└──────────────────────────────────────────────────────────────────────┘
```

### 구성요소

| 구분 | 설명 |
|:-----|:-----|
| **양자채널** | 양자암호키 분배 위해 편광신호를 전송하기 위한 양자전용채널 |
| **공용채널** | 기존의 통신망을 이용해 송수신자간 편광필터를 송신, TCP/IP 프로토콜을 활용 |
| **QRNG** | 반복되는 주기를 예측하기 어려운 진정 난수를 생성하기 위한 장치 |

---

## 주요기술

### 양자키분배 주요기술

| 구분 | 핵심 기술 | 설명 |
|:-----|:---------|:-----|
| **QKD 핵심기술** | 프로토콜 | - 대표 프로토콜 BB84<br>- B92(초기), E91, SARG04, DPS, TF, MDI, CV 등 다수 있음 |
| | 네트워킹 | - QTTH (Quantum To The Home) |
| | 분배구현 방식 | - One way방식(고속 동작), Two Way 방식(안정적 동작) |
| **QKD 하드웨어** | 양자 광학부 | - 양자광원과 간섭계, 변조기(PM), 검출기(APD) 구성 |
| | 전자 제어부 | - 구동회로부, 신호처리부, 외부시스템과의 연결을 위한 인터페이스, 제어회로 구성 |
| | 양자중계기 | - 장거리 전송 한계 극복을 위한 장거리 전송 기술 |

- BB84(Bennett & Brassard가 1984년 발표)프로토콜이 대표적인 양자키 분배 방식으로 사용됨

---

## BB84 프로토콜

### BB84 동작 과정

| 단계 | 설명 |
|:-----|:-----|
| 1 | Alice가 랜덤 비트열과 랜덤 기저 선택 |
| 2 | 편광된 광자를 Bob에게 전송 |
| 3 | Bob이 랜덤 기저로 측정 |
| 4 | 공용 채널로 기저 비교 |
| 5 | 일치하는 기저의 비트만 키로 사용 |
| 6 | 오류 검출 및 키 확정 |
