# 양자내성암호화 (Post-Quantum Cryptography)
{: .fs-8 }

11.2 양자 보안
{: .label .label-purple }

---

## 핵심 키워드

`다변수` `코드` `격자` `아이소제니` `해시` `NIST` `CRYSTALS`

---

## 정의/개념

**양자 컴퓨팅 보안 위협 대응**, 수학적 난제 활용 **암호화 기술**

- 양자 컴퓨터의 공격에도 안전한 암호 알고리즘
- 양자컴퓨터 활용 수준(상용화 아님) 대비 양자 내성 암호화 상용화는 높아지고 있다

---

## 유형

### 양자내성 암호화 유형별 특징

| 양자내성 암호화 | 개념도 | 설명 | 알고리즘 |
|:--------------|:-------|:-----|:---------|
| **다변수 기반 암호화** | 다변수 방정식 | - 유한체 위에서 계산하는 다변수함수 문제의 어려움에 기반하는 암호 시스템 | - HFE<br>- Rainbow<br>- Gui |
| **코드 기반 암호화** | 오류 정정 코드 | - 일반적인 선형 코드를 디코딩하는 어려움에 기반하는 암호 시스템 | - QC-MDPC<br>- Wild<br>- McEliece |
| **격자 기반 암호화** | 격자 구조 | - 격자 위에서 계산하는 문제의 어려움에 기반하는 암호 시스템 | - SS-NTRU<br>- NTRU<br>- Prime<br>- LWE-Frodo |
| **아이소제니 기반 암호화** | 타원 곡선 | - 순서가 같은 두 타원 곡선 사이에 존재하는 아이소제니(Isogeny)를 구하는 문제의 어려움에 기반하는 암호 시스템 | - SIDH |
| **해시 기반 전자 서명** | 해시 트리 | - 해시 함수의 안전성을 기반으로 한 전자 서명 시스템 | - XMSS<br>- SPHINCS |

## NIST PQC 표준화

### NIST 선정 알고리즘 (2022년)

| 구분 | 알고리즘 | 기반 | 용도 |
|:-----|:---------|:-----|:-----|
| **암호화/키캡슐화** | CRYSTALS-Kyber | 격자 | 키 교환 |
| **전자서명** | CRYSTALS-Dilithium | 격자 | 범용 서명 |
| | FALCON | 격자 | 짧은 서명 |
| | SPHINCS+ | 해시 | 보수적 서명 |

---

## 기존 암호 vs 양자내성암호

| 구분 | 기존 암호 | 양자내성암호 |
|:-----|:---------|:------------|
| **RSA** | 소인수분해 기반 | 쇼어 알고리즘에 취약 |
| **ECC** | 이산대수 기반 | 쇼어 알고리즘에 취약 |
| **PQC** | - | 양자 컴퓨터에 안전 |
| **대칭키** | AES 등 | 그로버 알고리즘 영향 (키 길이 2배) |

## 양자 위협 타임라인

| 시기 | 예상 위협 |
|:-----|:---------|
| **현재** | HNDL (Harvest Now, Decrypt Later) 공격 |
| **2030년대** | 암호학적으로 유의미한 양자 컴퓨터 등장 예상 |
| **대응** | 암호 민첩성(Crypto Agility) 확보 필요 |
