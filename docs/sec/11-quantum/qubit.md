# 큐비트 (Qubit)
{: .fs-8 }

11.3 양자 보안
{: .label .label-purple }

---

## 핵심 키워드

- 설계방식: 양자어닐링, 레이저 네트워크, 초전도 큐비트형, 스핀큐비트형
- 표기법: 바닥 상태, 들뜬 상태

---

## 정의/개념

**양자 중첩, 양자 얽힘, 양자 불확정성**, 0과 1 동시상태 **양자 연산 기본단위**

- 고전 컴퓨터의 비트(0 또는 1)와 달리 0과 1의 중첩 상태를 가질 수 있음

---

## 설계 방식

### 큐비트의 설계방식

| 구분 | 구조도 | 설명 |
|:-----|:-------|:-----|
| **양자 어닐링** | 고리형 구조 | - 니오븀을 재료로 하는 초전도회로에 양자비트를 512개 탑재(D-wave2)<br>- 전류가 왼쪽에서 회전하면 상향, 오른쪽으로 회전하면 하향의 자속전자가 생김 |
| **레이저 네트워크** | 레이저 격자 | - 양자 뉴럴 네트워크에 활용되는 레이저로 구현한 아날로그 양자 기술<br>- 변동성이 낮고 상온에서 작동, 비용에 강점 |
| **초전도 큐비트형** | 초전도 회로 | - 한쪽에서 전기저항을 없앤 극저온 초전도체 환경에서 전자 흐름의 양자현상을 신호로 사용하는 초전도 회로 개발 |
| **스핀 큐비트형** | 반도체 | - 반도체 안에 전자를 고립시켜 스핀을 이용하는 방식, 양자점 활용 |
| **이온트랩형** | 이온 가둠 | - 초전도체에 비해 훨씬 더 긴 양자 결맞음 시간과 큐비트간 임의의 연결이 가능함을 이용해 구현<br>- 수소나 칼슘 등 원자에서 전자를 제거한 양이온의 에너지 상태 |

- 큐비트를 이용해 양자컴퓨팅, 양자정보통신, 양자암호화 등 적용 범위가 점차 확대됨

---

## 상태 표현

### 큐비트의 상태 표현

| 구분 | 상태 표현 | 설명 |
|:-----|:---------|:-----|
| **표기법** | \|ψ⟩ = α\|0⟩ + β\|1⟩ | - 디랙(Paul Adrien Maurice Dirac)의 브라-켓(bra-ket) 표기법<br>- α와 β는 복소수로 언제나 \|α\|²+\|β\|²=1를 만족 |
| **값 상태** | 바닥 상태(ground state)<br>들뜬 상태(excited state) | - 바닥 상태는 전자의 가장 안전한 상태<br>- 들뜬 상태는 전자의 불안정한 상태 (바닥 상태로 회귀)<br>- 구분이 가능한 두 가지 상태를 갖는 모든 양자 상태는 수학적으로 큐비트로 정의<br>- 큐비트 2개가 있을 경우 중첩 가능한 기저 상태는 \|00⟩, \|01⟩, \|10⟩, \|11⟩ 의 총 4가지 |

- 큐비트의 특성인 양자 병렬성을 이용한 양자컴퓨터 연구 진행중

---

## 양자컴퓨터와 전통컴퓨터의 비교

| 비교 항목 | 양자컴퓨터 | 전통컴퓨터 |
|:---------|:----------|:----------|
| **연산단위** | 큐비트(Qubit) | 비트(bit) |
| **값 상태** | 동시존재('0', '1') | 단독존재('0', '1') |
| **연산방식** | 모든 조합 동시연산 | 순차연산 |
| **상태변화** | 불확정(측정시 확정) | 고정 |
| **구현방법** | 초전도체 | 전기회로 |

- IBM, 구글 등 큐비트 이용한 양자컴퓨터 활발 연구중이나 극저온, 무진동, 무소음 기반 환경 문제 해결 필요

---

## 양자 특성

| 특성 | 설명 |
|:-----|:-----|
| **양자 중첩** | 0과 1의 상태를 동시에 가짐 |
| **양자 얽힘** | 두 큐비트 간 상관관계 유지 |
| **양자 불확정성** | 측정 전까지 상태 미확정 |
