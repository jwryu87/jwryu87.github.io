# 암호화 분류
{: .fs-8 }

10.1 암호화
{: .label .label-purple }

---

## 핵심 키워드

`대치` `블록화` `치환` `압축` `혼돈` `확장` `블록 암호화` `스트림 암호화` `SPN` `Feistel`

---

## 정의/개념

**평문을 암호문으로 변경**, 기밀성, 무결성 유지 기술

---

## 원리

| # | 원리 | 핵심 |
|:--|:-----|:-----|
| 1 | **대치** | 글자 변경 |
| 2 | **블록화** | 열과 행 변경 |
| 3 | **치환** | 위치 변경 |
| 4 | **압축** | 문자 삭제 |
| 5 | **혼돈/확산** | 치환/전치 활용 |
| 6 | **확장** | 문자 삽입 |

---

## 분류

### 1. 정보단위

| 분류 | 암호화 방식 | 핵심 |
|:-----|:-----------|:-----|
| **블록 암호화** | 블록 암호화 | 평문 블록단위 암호화, 고정 크기 |
| **스트림 암호화** | 스트림 암호화 | 키 스트림 생성, XOR 이진연산 |

### 2. 키 적용 방식

| 분류 | 암호화 방식 | 핵심 |
|:-----|:-----------|:-----|
| **대칭키** | 대칭키 | 비밀키(암)-비밀키(복) |
| **비대칭키** | 비대칭키 | 공개키(암)-개인키(복) |

### 3. 암호화 기술

| 암호화 방식 | 핵심 |
|:-----------|:-----|
| **SPN** | Substitution, Permutation 이용, 128bit, 4*4 행렬, S-BOX, P-BOX |
| **Feistel** | N비트 블록, N/2 분할, R번 라운드, S-BOX, XOR |
| **인수분해** | RSA |
| **타원곡선** | RSA 속도 안정성 해결, ECC |
| **이산대수** | Diffie-Hellman |
| **해시 알고리즘** | 제 1역상 저항성, 제 2역상 저항성, 충돌저항성 |

---

## 대칭키 vs 비대칭키

| 구분 | 대칭키 | 비대칭키 |
|:-----|:-------|:---------|
| **키** | 동일 | 공개키/개인키 |
| **속도** | 빠름 | 느림 |
| **키 분배** | 어려움 | 용이 |
| **알고리즘** | AES, DES, ARIA | RSA, ECC |
| **용도** | 데이터 암호화 | 키 교환, 전자서명 |
