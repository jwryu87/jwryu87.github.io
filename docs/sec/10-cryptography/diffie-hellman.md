# 디피-헬만 알고리즘 (Diffie-Hellman Algorithm)
{: .fs-8 }

10.8 암호화
{: .label .label-purple }

---

## 핵심 키워드

`비밀키 공유` `이산대수 어려움` `키 교환` `MITM 취약점`

---

## 정의/개념

**비밀키 공유, 이산대수 어려움** 사용 암호화 **키 교환 알고리즘**

- 안전하지 않은 채널에서 두 당사자가 공유 비밀키를 생성하는 방식

---

## 메커니즘

### Diffie-Hellman 키 교환의 동작 원리

```
          Alice                              Bob
            │                                  │
     ①소수 p, 정수 g (1~p-1 값) 선택 후 사전에 공유
            │                                  │
     ②정수 x 선택                        ④정수 y 선택
            │                                  │
     ③연산결과 전송 (A=gˣ mod p)              │
            │ ─────────────────────────────────>│
            │                                  │
            │          ⑤연산결과 전송 (B=gʸ mod p)
            │<─────────────────────────────────│
            │                                  │
     ⑥K=Bˣ mod p                       ⑥K=Aʸ mod p
        연산                              연산
            │                                  │
            └──────── ⑦키 공유 (K=gˣʸ mod p) ─────┘
```

- 소수 p, 정수 g, 연산값 A, B를 알더라도, Alice와 Bob이 선택한 x와 y값을 모르면 비밀키를 알 수 없음
- **이산대수 문제의 어려움**에 기반

---

## 문제점 및 해결방안

### 디피헬만 키 교환 문제점 해결 기법 발전

| 공격기법 | 취약점 | 해결방안 |
|:---------|:-------|:---------|
| **MITM** | 중간자 공격 | RSA 변경, 다중인증 활용 |
| **Logjam** | TLS 통신 추출 공격 | TLS 버전 업그레이드, ECDHE 사용 |

### 발전 방향

```
디피헬만          공유키 문제 해결                1일 만에
키교환 알고리즘 ──> MITM 해결 ───────> RSA ──────> 소인수 해독 ──> 양자암호화
                                                              통신기술
    │                  │                              │
    ↓                  ↓                              ↓
비밀키(공유키) 기반   공개키/개인키 기반            양자 특성 이용
이산 대수 원리 이용   소인수분해의 해독 어려움      BB84 프로토콜 이용
                                                  현재 개발 중
                     5G, WAN, IoT 까지 적용

                클라우드 기반 보안 서비스 새시 적용(SASE)
```

## 알고리즘 절차

| # | 단계 | 설명 |
|:--|:-----|:-----|
| 1 | 공개 값 설정 | 소수 p, 원시근 g 선택 및 공유 |
| 2 | 개인키 선택 | Alice: x, Bob: y 비밀 선택 |
| 3 | 공개키 계산 | A = gˣ mod p, B = gʸ mod p |
| 4 | 공개키 교환 | A, B 상호 전송 |
| 5 | 공유키 계산 | K = Bˣ mod p = Aʸ mod p = gˣʸ mod p |

---

## ECDH (Elliptic Curve Diffie-Hellman)

| 항목 | DH | ECDH |
|:-----|:---|:-----|
| **기반** | 이산대수 | 타원곡선 이산대수 |
| **키 길이** | 2048비트 | 256비트 |
| **속도** | 상대적 느림 | 빠름 |
| **안전성** | 동등 수준 | 동등 수준 (짧은 키) |
