# MDC (Modification Detection Code)
{: .fs-8 }

10.4 암호화
{: .label .label-purple }

---

## 핵심 키워드

`MD5` `SHA-1` `HAS-160` `메시지 무결성` `해시 함수`

---

## 정의/개념

**메시지 변경 감지 코드** 및 메시지 무결성 보장 인증 기술

- 키를 사용하지 않는 해시 함수 기반 무결성 검증 방식

---

## 메커니즘

### MDC 동작 과정

```
┌───────┐                                       ┌───────┐
│ Alice │                                       │  Bob  │
│       │                                       │       │
│ ┌─M─┐ │        Insecure channel              │ ┌─M─┐ │
│ │   │ │ ─────────────────────────────────────>│ │   │ │
│ └───┘ │                                       │ └───┘ │
│   │   │                                       │   │   │
│ ┌Hash┐│                                       │ ┌Hash┐│
│ │    ││                                       │ │    ││
│ └────┘│                                       │ └────┘│
│   │   │                                       │   │   │
│ ┌MDC┐ │     Channel immune to change         │ ┌MDC┐ │──>Same?─>Accept
│ └───┘ │ ─────────────────────────────────────>│ └───┘ │     │
└───────┘                                       └───────┘     N
                                                              │
M: Message                                                 Reject
Hash: Cryptographic hash function
MDC: Modification detection code
```

## 알고리즘

| 알고리즘 | 핵심 |
|:---------|:-----|
| **MD5** | 임의 길이(평문) → **128비트**(암호문) |
| **SHA-1** | 임의 길이(평문) → **160비트**(암호문) |
| **HAS-160** | 국내 개발, SHA-1 기반, **160비트**(암호문) |

---

## MDC vs MAC 비교

| 구분 | MDC | MAC |
|:-----|:----|:----|
| **키 사용** | 미사용 | 사용 |
| **인증** | 무결성만 | 무결성 + 인증 |
| **알고리즘** | MD5, SHA | HMAC, CMAC |
| **용도** | 변조 감지 | 변조 감지 + 출처 인증 |

---

## 특징

| 특징 | 설명 |
|:-----|:-----|
| **키 불필요** | 공유 비밀키 없이 동작 |
| **무결성 보장** | 메시지 변조 여부 확인 |
| **인증 미제공** | 송신자 인증 불가 |
| **단방향** | 복호화 불가능 |

---

## 한계점

| 한계 | 설명 |
|:-----|:-----|
| **인증 부재** | 누가 메시지를 보냈는지 확인 불가 |
| **중간자 공격** | 메시지와 MDC 모두 변조 가능 |
| **키 부재** | 송신자 식별 불가 |
