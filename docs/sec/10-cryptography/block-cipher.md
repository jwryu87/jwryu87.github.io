# 블록 암호화
{: .fs-8 }

10.2 암호화
{: .label .label-purple }

---

## 핵심 키워드

`고정 길이 블록단위` `SPN` `Feistel 구조` `S-Box` `P-Box` `ECB` `CBC` `CTR`

---

## 정의/개념

평문을 고정 길이의 **블록단위로 나누어** 각 블록마다 암호화를 수행하여, 고정된 크기의 블록 단위 암호문을 생성하는 **암호화 기술**

---

## 알고리즘

### SPN 구조

| 분류 | 알고리즘 | 핵심 |
|:-----|:---------|:-----|
| **SPN** | **AES** | DES 단점 개선, 128비트 블록 크기 사용, 4단계 알고리즘 절차 |
| | **ARIA** | ISPN 구조, 고정길이(128비트) 입/출력, 가변길이 지원 |

### Feistel 구조

| 분류 | 알고리즘 | 핵심 |
|:-----|:---------|:-----|
| **Feistel** | **DES** | 56비트(키), 64비트(평문)→64비트(암호문) |
| | **LEA** | ARX 구조, AES 대비 2배 속도, 경량환경 기밀성 제공, 128비트 |
| | **HIGHT** | RFID, USN, 경량환경 기밀성 제공, 64비트(평문), 128비트(키) |
| | **SEED** | 128비트(평문), 128비트(키), 128비트(암호문), 16라운드, 전자상거래 |
| | **IDEA** | DES 대체, 128비트(키), 64비트(암호문), 8라운드 |
| | **RC5** | 다양한 크기 키, 블록, 라운드 |

---

## 운영모드

### 암호화 강도 향상을 위한 블록 암호화 운영방식

| 분류 | 운영모드 | 핵심 |
|:-----|:---------|:-----|
| **블록암호화만 사용** | **ECB** | 평문-암호문 1:1, 고속처리, 기밀성 낮음 |
| | **CBC** | IV 사용 |
| | **PCBC** | IV 사용, XOR 사용 |
| **자기 동기식 스트림 암호 결합** | **CFB** | IV 암호화, XOR 선처리 |
| | **OFB** | IV 암호화, XOR 후처리 |
| | **CTR** | Nonce 사용, Count 사용, Padding 불필요 |

### 운영모드 구조

```
                    병렬처리    병렬처리
                    불가능     불가능
운영모드  ┌───────────────────────────────────────┐
(특징)    │  PCBC          OFB                    │
          │    ↓             ↓                    │
          │   XOR           XOR                   │
          │  전처리         전처리                 │
┌──────┬──────┬──────┬──────┬──────┐
│ ECB  │  ↔  │ CBC  │  ↔  │ CFB  │  ↔  │ CTR  │
└──────┴──────┴──────┴──────┴──────┘
         IV           사용자       사용자         암복호화
        사용          암호화       암호화          가능
```

- 차분공격, 선형공격, 연관키 공격, 대수적 공격, Square 공격 등 다양한 공격에 대한 안전성 향상
- 평문과 암호문이 노출된 경우와 평문이 반복되는 패턴이 있는 경우 블록조합에 따른 복호의 저항성 향상

## AES vs DES 비교

| 구분 | DES | AES |
|:-----|:----|:----|
| **블록 크기** | 64비트 | 128비트 |
| **키 길이** | 56비트 | 128/192/256비트 |
| **라운드** | 16 | 10/12/14 |
| **구조** | Feistel | SPN |
| **안전성** | 취약 | 안전 |
