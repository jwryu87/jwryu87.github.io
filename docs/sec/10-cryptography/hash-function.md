# 암호학적 해쉬(Hash) 함수
{: .fs-8 }

10.3 암호화
{: .label .label-purple }

---

## 핵심 키워드

`역상 저항성` `제2 역상 저항성` `충돌 저항성` `일방향 함수` `SHA` `MD`

---

## 정의/개념

가변길이의 Data를 입력으로 받아 **고정길이의 Message Digest를 출력**으로 하는 **단방향 암호 알고리즘**

---

## 유형

### 해시함수 구분

```
              해시함수 구분
                   │
       ┌───────────┴───────────┐
       │                       │
  Index 해시함수         무결성 해시함수
  (Hashing Function)          │
                    ┌─────────┴─────────┐
                    │                   │
               암호학적              비 암호학적
               해시함수              해시함수
             SHA, MD, MAC 등      CRC, Checksum 등
```

---

## 특징 (3가지 저항성)

### 해쉬함수의 보안 특성

| 특징 | 개념도 | 설명 |
|:-----|:-------|:-----|
| **역상 저항성** | X → H → Y | - 주어진 Y 값에 대해 H(X) = Y를 만족하는 X 값을 찾는 것이 힘든 특성<br>- **일방향(One Way Function) 특성** |
| **제 2 역상 저항성** | X → H → Y<br>? → H → Y | - 주어진 X에 대해 H(X) = H(X')를 만족하는 X(X ≠ X')를 찾기 힘든 특성<br>- 동일한 해시값에 대해 초기 입력값과는 다른 입력값을 찾기 어려운 특성 |
| **충돌 저항성** | ? → H → Y<br>? → H → Y | - H(X) = H(X')를 만족하는 임의의 두 입력값 X, X' (X≠X')를 찾기 힘든 특성<br>- 동일한 해시값을 생성하는 두개의 서로 다른 입력값을 찾기 어려운 특성 |

---

## 주요 해시 알고리즘

| 알고리즘 | 출력 길이 | 특징 |
|:---------|:----------|:-----|
| **MD5** | 128비트 | 빠른 속도, 충돌 취약점 발견 |
| **SHA-1** | 160비트 | MD5 대체, 현재 취약 |
| **SHA-256** | 256비트 | SHA-2 계열, 현재 표준 |
| **SHA-512** | 512비트 | 높은 보안성 |
| **SHA-3** | 가변 | Keccak 기반, 최신 표준 |
| **HAS-160** | 160비트 | 국내 개발, SHA-1 기반 |

---

## 해시 함수 vs 암호화

| 구분 | 해시 함수 | 암호화 |
|:-----|:---------|:-------|
| **방향** | 단방향 | 양방향 |
| **복호화** | 불가능 | 가능 |
| **출력** | 고정 길이 | 가변 길이 |
| **용도** | 무결성 검증, 전자서명 | 기밀성 보장 |
| **키** | 불필요 | 필요 |

---

## 활용 분야

| 분야 | 설명 |
|:-----|:-----|
| **비밀번호 저장** | 평문 저장 대신 해시값 저장 |
| **전자서명** | 메시지 다이제스트 생성 |
| **무결성 검증** | 파일/데이터 변조 여부 확인 |
| **블록체인** | 블록 해시, 트랜잭션 해시 |
