# RSA (Rivest, Shamir, Adleman)
{: .fs-8 }

10.9 암호화
{: .label .label-purple }

---

## 핵심 키워드

`미러링` `스트라이핑` `해밍` `패리티` `분산 패리티` `소인수 분해` `0+1` `1+0`

---

## 정의/개념

**소인수 분해, 전자서명** 알고리즘, 소수 p,q 의존성 **암호화 알고리즘**

- 큰 수의 소인수 분해가 어렵다는 수학적 원리에 기반
- 1977년 발표된 대표적인 비대칭키 암호화 알고리즘

---

## 메커니즘

### RSA 암호화/복호화 과정

```
         암호화                          복호화
┌─────────────────────────┐    ┌─────────────────────────┐
│                         │    │                         │
│   M : 평문    C : 암호문 │    │  C : 암호문   M : 평문  │
│       │           │     │    │       │           │     │
│       ↓           ↓     │    │       ↓           ↓     │
│   (n, e): 공개키        │    │   (n, d): 개인키        │
│       │           │     │    │       │           │     │
│       ↓           ↓     │    │       ↓           ↓     │
│   E : 암호화  D : 복호화│    │   D : 복호화  E : 암호화│
│       │           │     │    │       │           │     │
│       ↓           ↓     │    │       ↓           ↓     │
│   C = Mᵉ mod n          │    │   M = Cᵈ mod n          │
└─────────────────────────┘    └─────────────────────────┘
```

- RSA를 이용한 전자서명 시 PRNG(Pseudo Random Number Generator)를 이용하여 동일 문서에 대해 다른 전자서명 값을 생성함

---

## 키 생성 및 암/복호화 절차

| 구분 | # | 프로세스 | 공식 |
|:-----|:--|:---------|:-----|
| **키 생성** | 1 | 소수 p,q 선택 | n = p * q |
| | 2 | 오일러 파이함수 계산 | Ø(n) = (p - 1) * (q - 1) |
| | 3 | e 값 계산 | gcd(e, Ø(n)) = 1 |
| | 4 | d 값 계산 | e * d mod Ø(n) = 1 |
| | 5 | 공개키, 개인키 생성 | 공개키(PK) = (n, e), 개인키(SK) = (n, d) |
| **암호화** | 6 | 공개키 암호문 생성 | C = Mᵉ mod n |
| **복호화** | 7 | 비밀키 평문 복호화 | M = Cᵈ mod n |

---

## RSA 보안 원리

| 원리 | 설명 |
|:-----|:-----|
| **소인수 분해 어려움** | n = p × q에서 p, q를 찾기 어려움 |
| **대수 문제** | Cᵈ mod n에서 d를 찾기 어려움 |
| **키 길이** | 2048비트 이상 권장 (현재 표준) |

---

## RSA vs 기타 비대칭키 알고리즘

| 구분 | RSA | ECC | DSA |
|:-----|:----|:----|:----|
| **기반** | 소인수 분해 | 타원곡선 | 이산대수 |
| **키 길이** | 2048비트 | 256비트 | 2048비트 |
| **속도** | 느림 | 빠름 | 중간 |
| **용도** | 암호화, 서명 | 암호화, 서명 | 서명 전용 |

---

## RSA 취약점 및 대응

| 취약점 | 설명 | 대응 |
|:-------|:-----|:-----|
| **쇼어 알고리즘** | 양자 컴퓨터로 소인수 분해 가능 | 양자내성암호(PQC) 전환 |
| **키 길이 부족** | 1024비트 이하 취약 | 2048비트 이상 사용 |
| **PKCS#1 v1.5** | 패딩 오라클 공격 | OAEP 패딩 사용 |
