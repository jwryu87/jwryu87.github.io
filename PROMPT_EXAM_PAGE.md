# 기출문제 학습 페이지 생성 프롬프트

## 📋 사용 방법

아래 **[입력 템플릿]** 에 해설지 내용을 복사해서 붙여넣고, 이 프롬프트 전체를 AI에게 전달하세요.

---

## 🎯 요청 사항

아래 해설지 내용을 바탕으로 **기출문제 학습 페이지**를 만들어줘.

### 파일 생성 규칙:
1. **파일 위치**: `docs/{카테고리}/exam/{회차}-{교시}-{번호}-{영문주제}.md`
2. **기출문제 목록 링크 추가**: `docs/{카테고리}/exam/index.md` 에 해당 문제 링크 연결

## 📝 문제 유형별 페이지 구성

### 🅰️ 1교시형 (단답형)

> 단일 주제에 대한 설명 문제

#### 페이지 구성:

**🎓 고딩 수준 설명**
- 키워드 3개 (쉬운 설명)
- 등장배경 (이야기 형식, 문제→해결 구조)
- 개념 정의 (비유 포함)
- 기술요소 2그룹 (표 형식)
- 차별점 키워드 1개 (답안지 가산점)
- 6하원칙 요약

**🎯 기술사 수준 설명**
- 키워드 3개 (기술 용어)
- 등장배경 2개 (핵심만)
- 개념 정의 (22자 이내)
- 구성요소 2그룹 × 4항목 (표 형식)
- **🧠 암기법**: 영어 약자 기반으로 (한글 X)
- 상세 비교표
- 차별점 키워드 1개 (심화 개념)
- 프로토콜/기술 간 관계도 (ASCII 다이어그램)
- 6하원칙 요약
- 상위 토픽 계층도
- 향후 전망 (기술/서비스/표준화)

---

### 🅱️ 2교시형 (서술형)

> 가, 나, 다 등 세부 질문이 있는 문제

#### 페이지 구성:

**🎓 Quick 개념잡기** (간략하게)
- 한 줄 정의
- 쉬운 비유
- 왜 필요한가? (3줄 요약)

**가. 첫 번째 질문**
- 1. 개념 (정의 22자 + 상세 개념 표)
- 2. 특징/구성요소 (표 형식)
- 3. 암기법

**나. 두 번째 질문**
- 1. 개념도 (ASCII 다이어그램)
- 2. 구성요소 상세 (표 형식)
- 3. 암기법

**다. 세 번째 질문**
- 1. 개념/정의
- 2. 비교표 (있다면)
- 3. 동작 원리/절차
- 4. 암기법

**⭐ 차별점 키워드**

**📈 상위 토픽 계층도**

**✅ 학습 체크리스트**

---

## 🧠 암기법 규칙

- **영어 약자 기반**으로 작성 (한글 번역 암기법 X)
- 풀네임(Full Name)과 약자의 관계가 명확하도록
- 진화/발전 순서가 있으면 순서대로 암기
- 예시: 
  - "RT-P → RT-CP → RT-SP → S-DP"
  - "A-S-C-E" (API-Scheduler-Controller-Etcd)
  - "H-D-C-LL" (HLS-DASH-CMAF-LL)
